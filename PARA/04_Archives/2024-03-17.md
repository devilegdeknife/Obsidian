# 玻璃杯挑选指南
#生活技巧/购物指南

## 指标

### 品牌
| 品牌名称 | 特点描述                                                                                                     |
|-----------|------------------------------------------------------------------------------------------------------------|
| 富光     | 耐热玻璃杯专家，产品抗摔耐用，适合日常使用。设计简约，适合家庭和办公室。                                           |
| 耐克玻璃 | 轻便耐高温的玻璃杯，适合户外活动使用。极简风格，外观清新，易于携带。                                             |
| 乐美雅（Luminarc） | 耐热玻璃杯品牌，产品透明度高，现代简约设计。适合家庭和餐厅使用，耐用且易于清洁。                               |
| 康宁（Corningware） | 耐热玻璃杯品牌，采用特殊玻璃材质，耐磨损。结合经典与创新的设计，产品多样，适合不同场合使用。                   |
| libbey   | 专业玻璃杯品牌，产品厚度适中，耐摔。独特设计，适合酒吧和餐厅使用，也适合收藏。                                     |
| Spiegelau | 德国玻璃杯品牌，以精致工艺和多样选择著称。产品优雅经典，适合品酒和各种场合。                                     |
| Riedel   | 奥地利玻璃杯品牌，专业品酒用杯。独特造型，提高品酒体验，艺术感强，适合葡萄酒爱好者。                             |
| 玛歌（Margaux） | 耐热玻璃杯品牌，产品多样，花纹丰富。浪漫风情，适合家庭和餐厅使用，增添生活乐趣。                               |

### 材质

| 材质类型  | 优点                             | 缺点                            |
| ----- | ------------------------------ | ----------------------------- |
| 高硼硅玻璃 | 耐热性极好，可承受大温差而不炸裂；高透性、强度高、可做得很薄 | 价格相对较高                        |
| 水晶玻璃  | 安全性高，不含铅，适合长期使用                | 水晶玻璃硬度较低，容易刮花；价格较高            |
| 普通玻璃  | 价格低廉，透明度好，耐热性较好                | 硬度较低，容易破裂，对温差的适应性较差，冷热交替时容易爆裂 |

### 设计

- 茶水分离；
- 握把；
- 刻度；
- 美观；
- 容量。

## 决策

1. 无铅高硼硅玻璃
2. 带隔热设计，如握把，双层
3. 美观设计，刻度
