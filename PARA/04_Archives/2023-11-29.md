# [[政治光谱-意识形态]]
#知识

| 政治意识形态             | 描述                                                                                                                                                                      |
|------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **蜂群思维集体主义**         | 个人被集体意识或群体思维所取代，强调一致性和缺乏个人自主性。                                                                                                                   |
| **英社主义**                | 描绘了《1984》中的虚构政治意识形态，代表专制政权通过监视和宣传控制公民生活。                                                                                                  |
| **社会达尔文主义**           | 将达尔文的自然选择理论应用于人类社会，认为适者生存，不适者将被淘汰。                                                                                                         |
| **斯大林主义**               | 与苏联领导人斯大林的统治有关，包括极权主义、国有化、集中计划和政治压制。                                                                                                     |
| **民族布尔什维克主义**         | 将极左的布尔什维克主义与极端的民族主义相结合，主张建立一个民族主义社会主义国家。                                                                                           |
| **复兴主义**                | 在阿拉伯国家突出，强调阿拉伯民族主义、社会主义和阿拉伯国家之间的团结。                                                                                                      |
| **斯特拉瑟主义**              | 法西斯主义的变种，强调社会主义、反资本主义和反犹太主义，与纳粹德国的斯特拉瑟兄弟有关。                                                                                   |
| **国家工团主义**              | 结合了民族主义和工团主义，主张以民族身份为基础组建工会，并建立按照企业主义路线组织的社会。                                                                                |
| **纳粹主义**                | 与德国纳粹党有关的意识形态，结合了极端的民族主义、种族优越感、反犹太主义和独裁主义。                                                                                       |
| **法西斯主义**               | 一种专制和民族主义的政治意识形态，包括独裁权力、集中控制、压制反对派，通常伴随着极端的民族主义或种族主义观点。                                                           |
| **白人至上主义**              | 宣称白人文化优越于其他种族，提倡歧视和种族等级制度。                                                                                                                     |
| **皮诺切特主义**              | 以智利前独裁者奥古斯托-皮诺切特的名字命名，涉及自由市场资本主义、独裁统治和对异议的压制。                                                                                   |
| **民族资本主义**              | 强调国家经济利益、保护主义和政府干预经济以促进国内产业的经济制度。                                                                                                        |
| **毛泽东思想**               | 源于中华人民共和国开国元勋毛泽东的政治思想，强调阶级斗争、革命的群众动员和农民在实现社会主义中的重要性。                                                                     |
| **穆加贝主义**               | 以津巴布韦前总统罗伯特-穆加贝的名字命名，涉及独裁主义、土地再分配和对非洲民族主义的强调。                                                                                    |
| **君主制社会主义**             | 结合了君主制和社会主义元素，主张建立社会主义经济体系，同时保留君主作为国家元首。                                                                                           |
| **布尔什维克民族主义**          | 俄国革命期间布尔什维克运动中的一个民族主义分支，提倡建立以工人阶级和俄罗斯民族利益为中心的社会主义国家。                                                                     |
| **开明的君主制**               | 将君主制与启蒙和理性治理的原则相结合，主张君主利用权力促进社会进步、智力发展和人民福祉。                                                                                 |
| **社团主义**                | 强调将社会组织成代表不同利益的企业集团，如工会和专业协会，以创造社会和谐与合作的政治意识形态和经济制度。                                                                 |
| **整合主义**                | 起源于20世纪初巴西，提倡民族主义、传统价值观和公司化的社会结构，往往包含专制主义和反自由主义的元素。                                                                         |
| **君主制**                 | 主张建立由君主领导的政府体系，强调维护传统、稳定和等级制度。                                                                                                               |
| **反动主义**                | 旨在维护或恢复传统或过时的制度、规范或价值观，以应对进步或改革带来的感知威胁或变化的一种政治或社会立场。                                                                  |
| **威权资本主义**              | 经济权力集中在少数人手中，通常与强大的中央政权或政府合作的一种经济制度。                                                                                                    |
| **正统马克思主义**             | 强调工人阶级和资产阶级之间的斗争，主张社会主义革命和最终建立一个无阶级社会的马克思主义理论。                                                                             |
| **铁托主义**                | 以南斯拉夫前领导人铁托的名字命名，主张自我管理、不结盟和分权的社会主义经济，与国际关系中的苏联保持距离。                                                                   |
| **卡斯特罗主义**              | 以古巴前领导人卡斯特罗的名字命名，强调社会主义原则、反帝国主义和革命斗争。                                                                                                 |
| **左翼民族主义**              | 将民族主义情绪与左派或社会主义原则相结合，主张在国家框架内赋予工人阶级或被压迫群体权力和自决权。                                                                          |
| **休伊-朗主义**               | 与20世纪初路易斯安那州的政治家休伊-朗有关，旨在解决收入不平等问题，促进社会福利计划，并挑战企业权力的政治意识形态和民粹主义运动。                                      |
| **俾斯麦主义**               | 以19世纪德国政治家俾斯麦的名字命名，注重现实政治、社会福利改革和保持权力平衡。                                                                                                |
| **LGBT保守主义**            | 描述持有保守政治观点或与保守意识形态一致的LGBTQ+个体的术语，反映了性取向或性别认同与保守主义原则的交集。                                                                 |
| **新保守主义**               | 20世纪末在美国出现的政治意识形态，主张强有力的外交政策、坚定推进民主，并相信美国的力量和价值观的有效性。                                                                  |
| **里根主义**                | 与美国第40任总统里根有关的政治意识形态和政策，提倡自由市场资本主义、减少政府干预，并强调强大的国防。                                                                     |
| **旧保守主义**               | 强调传统价值观、有限政府和对社会和文化变革的谨慎态度的保守政治意识形态。通常拒绝全球主义，优先考虑国家主权。                                                             |
| **托洛茨基主义**              | 与俄国革命的关键人物托洛茨基的思想有关，强调永久革命和国际主义，主张推翻资本主义制度，建立世界性的社会主义。                                                                |
| **左翼共产主义**              | 拒绝主流社会主义政党的战略和战术，认为它们是妥协的或不够革命的一系列共产主义意识形态和运动。                                                                              |
| **欧洲共产主义**              | 20世纪70年代和80年代在西欧共产党内出现的政治趋势，试图与苏联保持距离，接受民主价值观，主张走 "欧洲共产主义 "的社会主义道路。                                            |
| **社会主义超人类主义**          | 将社会主义原则与探索新兴技术相结合，以提高人类能力和克服社会不平等的政治和哲学运动。                                                                                       |
| **分配主义**                | 经济和社会理论，主张广泛的财产所有权和财富分配，反对资本主义和社会主义，提倡小规模的地方经济、合作社和社会正义。                                                        |
| **基督教民主主义**            | 将基督教原则与民主治理相结合的政治意识形态，强调社会保守主义、辅助性、社会福利，以及由道德原则规范的市场经济。                                                         |
| **保守主义**                | 维护传统的制度、习俗和价值观，强调稳定、秩序和渐进的变化的政治意识形态。原则在不同文化中可能有所不同，但通常重视个人自由、有限政府干预和传统制度。                    |
| **特朗普主义/MAGA**         | 与唐纳德-特朗普在总统任期内倡导的政治运动和意识形态有关，强调民粹主义的言论、民族主义、保护主义、移民限制和对国内经济利益的关注。                                      |
| **自由保守主义**             | 将保守的价值观与个人自由、有限政府和自由市场的信念相结合的政治意识形态，支持社会自由，并主张维护传统社会制度。                                                        |
| **国家自由主义**             | 将自由主义原则与国家在促进社会福利和经济监管方面的重要作用相结合。国家自由主义者优先考虑个人权利和自由，但也支持政府干预以减轻社会和经济不平等。|
| **社群主义** | 主张分散决策和社区对资源的控制，强调直接民主和合作治理的政治哲学。 | 
| **工团主义** | 强调将工人组织成工会或辛迪加以实现社会和经济目标，通常主张推翻资本主义，建立基于工人自我管理的社会。 |
| **雅各宾派** | 最初与法国大革命期间的雅各宾俱乐部有关，指的是一种左翼的政治意识形态，提倡激进的社会和政治变革，通常主张建立强大的中央政府和平等。 | 
| **超集体主义** | 描述极端形式的集体主义，即集体或社区优先于个人权利和自由，通常涉及重大的国家控制和有限的个人自主权。 | | 左翼民粹主义 | 将民粹主义言论和对工人阶级利益的呼吁与左翼政策相结合的政治方法，旨在挑战精英的影响，促进社会和经济正义。
| **左翼民粹主义** | 将民粹主义言论和对工人阶级利益的呼吁与左翼政策相结合的政治方法，旨在挑战精英的影响，促进社会和经济正义。|
| **第三条道路**            | 寻求调和传统左翼和右翼意识形态，主张在市场导向的政策框架内实现混合经济、社会正义和个人自由。                     |
| **自由主义**               | 强调个人权利、自由和政府对经济的有限干预，通常与平等、个人自由和法治等原则相关。                               |
| **重商主义**               | 16至18世纪盛行的一种经济理论和实践，特点是国家对贸易的控制、保护主义和通过出口积累财富。                      |
| **秩序自由主义**           | 德国的一种经济和政治哲学，提倡社会市场经济，将自由市场与强有力的监管框架相结合，确保社会稳定，防止垄断。   |
| **君主立宪制**             | 政治制度，君主在宪法框架内担任礼仪性国家元首，限制其权力，提供民主治理。                                    |
| **经济自由主义**           | 强调自由市场、有限政府干预和保护私有产权的经济意识形态，通常与个人自由和竞争的原则相关。                   |
| **侵略主义**               | 主张采取自信或侵略性的外交政策，通常涉及使用军事力量来保护或推进国家利益的意识形态。                         |
| **议会共产主义**           | 马克思主义政治理论，主张用工人委员会或类似的民主结构来控制和组织生产和社会，建立无国家、无阶级的社会。    |
| **民主社会主义**           | 将民主原则与生产资料的社会所有权相结合，通过财富再分配、社会安全网和民主决策改革，争取建立更公平的社会。 |
| **环境保护主义**           | 倡导保护和保存自然环境的社会和政治运动，解决环境问题、促进可持续性，并减轻人类活动对生态系统的影响。         |
| **进步主义**               | 倡导社会、经济和政治改革，解决社会不平等，促进进步，支持财富再分配、社会公正和政府干预。                 |
| **社会民主主义**           | 结合了社会主义和民主元素的政治意识形态，通过混合经济、社会福利计划和累进税制促进社会公正和减少不平等。   |
| **新自由主义**             | 强调自由市场、有限政府干预、私有化和放松管制，通过市场方法促进个人自由和经济效率。                         |
| **新自由保守主义**        | 将保守主义价值观与新自由主义经济政策相结合，支持有限政府、自由市场和传统社会价值观。                       |
| **资本主义的超人类主义**   | 将资本主义原则与使用新兴技术提高人类能力、超越生物限制的信念结合的哲学和技术运动。                           |
|**自由意志保守主义**       | 结合了保守的价值观和政府在经济和社会领域的有限干预。自由主义保守派优先考虑个人自由、自由市场和最小国家。|
| **自由意志社会主义**       | 结合了自由主义和社会主义原则，主张废除等级制度和压迫性制度，以支持分散的决策和集体所有。               |
| **哈耶克主义**             | 弗里德里希-哈耶克的思想，提倡古典自由主义原则，包括有限政府干预、自由市场、个人自由和自发秩序的重要性。 |
| **社会无政府主义**         | 废除所有形式的等级制度，包括国家和资本主义，建立基于自愿合作、互助和分散决策的社会的政治哲学。       |
| **安那其和平主义**         | 无政府主义与非暴力承诺相结合的政治哲学，以和平方式实现社会变革和解决冲突，反对使用暴力。                 |
| **民主邦联主义**           | 民主联邦制提倡分权自治、性别平等和社会生态，强调直接民主和地方自治。                                      |
| **布克钦社群主义**        | 强调基于社区的决策和建立面对面的民主机构的分散的、生态的、直接民主的社会组织形式。                       |
| **红色自由意志主义**       | 将左翼或社会主义原则与自由主义价值观相结合，主张经济和社会自由，以及平等和社会正义。                   |
| **利己主义**               | 强调自我利益和个人自主的重要性，通常与马克斯-施蒂尔纳的作品 "自我和它自己 "有关。                           |
| **自愿主义**               | 主张建立在自愿的、非强制性的关系上，反对使用武力或侵略，强调同意的重要性的政治哲学。                     |
| **安那其资本主义**         | 无政府主义的一种形式，支持废除国家，建立基于私有产权、自由市场和自愿交换的社会，强调自愿的互动。         |
