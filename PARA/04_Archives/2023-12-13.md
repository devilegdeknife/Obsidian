# [[DIY主机误区]]

| 焦虑点          | 内容                                                                                   |
|----------------|----------------------------------------------------------------------------------------|
| 电源焦虑        | 实际需求不需要过大电源，推荐按峰值功耗计算，不必盲目追求高瓦特电源。                 |
| 风道焦虑        | 家用电脑风扇相对较小，机箱侧板拆除可能比增加风扇更有效。                               |
| 风冷焦虑        | 高端风冷在某些情况下优于水冷，360水冷不一定适合所有情况，需根据实际情况选择。        |
| 温度焦虑        | 不必过于焦虑温度，合理范围内运行温度是正常的，不同硬件可能有不同的适宜温度范围。     |
| 帧率焦虑        | 游戏帧率需根据实际需求，不必盲目追求高帧率，对于大多数人，60fps 或 144fps 已足够。 |
| 机箱放右手边    | 根据桌面宽度选择机箱位置，左右均可，避免碍事。                                        |
| 强行走线        | 美观的走线可以提高整体外观，但不必强行走线，避免过度束缚。                             |
| 迷信高端        | 主板、显卡等高端型号并非性能提升的唯一途径，需根据实际性能需求选择。                   |
| 合理升级        | 对于中高端显卡，每4年升级一次足够，不必过度频繁升级。                                 |
| 不信任AMD新技术 | AMD的新技术可能是对NVIDIA技术的仿制，需谨慎采纳。                                     |
| 架构追新        | 对CPU、显卡等硬件追求最新架构，而对主板、内存等规范不必追新。                         |
| 不要超频        | 现代CPU显卡性能已经很强大，超频带来的收益相对较小，需权衡风险和收益。                 |
| 战未来          | 实际需求和当下情况更重要，谨慎对待所谓的未来趋势。                                    |
| 等等党          | 避免盲目相信不明确的市场信息，理性对待产品降价和上市。                                |
| 鸡头不买凤尾    | 对于同一性能水平，选择性价比更高的产品，不必盲目追求高端。                           |

# [[缺爱哲学探讨]]

可以假设诸位现在身处精神病院中。

我们面前有一位患有严重妄想症的缺爱少女，她的特质是会相信我们接下来所描绘的宗教。

而妄想肯定是治不好的，所以我们把目光投射到缺爱。
那么为了解决缺爱这个问题，我们可以向她发出两种宣言。

1. **这个世界上存在着一位爱世人的神。
2. **这个世界上存在着一位只爱你的神。

前者我们可以称为基督教，
后者可以称为朱慈打个飞机嗷嗷教。

*很明显，她几乎必然选择投身第二个宗教，也就是我们的，朱慈打个飞机嗷嗷教。

是的，虽然这个宗教的名字受到了创始人精神状态的影响，而变得极其猥琐，但她依然会选择这个宗教，然后沉浸在被爱的幻觉里。

原因很简单，
**对于真正缺爱的人来说，博爱等于没有爱。

这句话可能有那么点武断，但倘若你了解缺爱者的两个特质，
**高敏感以及占有欲**，
就不会觉得这是开玩笑。

我也在此告知诸位，缺爱的概念只会在对比中产生。

就好比你和父母住在荒山野岭，方圆百里寸草不生，鸡犬不留。你们家平时的日常就是互殴，你父亲打你母亲，你母亲打你父亲，他俩再一起揍你。
寒冬腊月里你得出去洗衣服，酷暑难耐下你得上山去挖土。
但你就是不会意识到你缺爱。

而如果把居住地方从荒山野岭改成幸福小区，每天你楼上爸妈带孩子旅游，隔壁父母教孩子唱歌，门对面俩老年人天天带着孙子出门上网吧。

你看，你马上就会在目睹他人幸福的恍然中，感受到爱的残缺。

**缺爱导致的占有欲是极端的，

**这种极端意味着爱必然是一种具备特殊性的事物**，

**而它特殊在只属于你自己**。

**这样的爱不会与他人分享**，**不会泄露**，**不会溢出**。

**它完完全全被你所拥有**。

**并充斥着在你躯壳里的每个角落**，**像这具皮囊装满了海水一样充盈**，**时而让你沉溺又漂浮**。

**于是你可以安下心**，**像最渺小的芥子那般藏匿**，**然后蜷缩在安全感的嗡鸣声慢慢感受融化**，**那些嘈杂的吵闹会渐渐离你远去**，**而你通过极度私密的管道后会变得轻盈**。

**那种轻盈伴随着摇晃**，**像是热汤又或者染料**。

**你的幸福**，**快乐**，**圆满**，**都将以这样的姿态肆意涂抹在原野上**。

**那些树应该是红色的**，**河流应该是黄色的**，**大象应该是彩色的**，**人流的鼻涕都得是蓝色的**。

**乱七八糟的颜色**，**还有微醺的风在山坡上流淌**。

**视线随关门而合上**，**这个充斥着幸福**，**兴奋**，**释放的专属画室**。

**就是对于你**，**爱的到来**。

它看上去不像是耶稣能够赐予的，在我们的世界里它往往被称为精神病患的狂想。

而这种狂想似乎并不现实，可是触碰不到，不代表不想要，

于是哪怕缺爱的人躺在一大片芦苇中间，还是会喊着要根救命稻草。

这的确应该被治愈。

但我想那拯救肯定不会是虚幻的博爱，这并不受缺爱者们的期待，那大概是某种暴烈的，炙热的，能够把骨骼压到发出响声，能够癫狂到手拉手一起跃下高楼的爱。

所以，你看上去真的不了解缺爱。

> [!note]
> 天底下没有渴望被爱的僧人，
也没有精神错乱的神父，
你可以端起金刚经诵念两句揭语，
一切有为法，如梦幻泡影，如露亦如电，应作如是观，
并以此来安抚对方的精神空虚，又
或者拿起新约念着生命在他里头，
这生命就是人的光，
祈祷对方从更高的地方得到力量。

但这些都没有意义，因为缺爱不是精神的空虚，毕竟这些要是有意义，西班牙的主教也不会跟着写重口味小说的女作家跑了。

它更像一种饥饿感。

它需要被实实在在地填补，需要被真真切切地感受，需要占有与被占有，拯救与被拯救。

哲学可以用来填补空虚，但缺陷必承受不住渴望。

当然，或许神的伟岸超越我的想象，他的福音可以给予人战胜缺陷的力量，但那不关精神病患事。

精神病患理应有自己的狂想，敲敲木鱼不如加入朱慈打个飞机嗷嗷教，起码我还会骗诸位我们教每周都发一个只爱你的忠犬。

虽然他始终处于叠加态，好像很难被人看到。

但不重要。

在痛苦和幻觉的精神世界里，妄想的爱虽然飘渺，但已然来到。

愿诸位不必死于真实的残酷，而是坠入虚幻的黄昏，因为通往幸福的道路上没有聪慧和愚蠢。

啊。

朱门！
