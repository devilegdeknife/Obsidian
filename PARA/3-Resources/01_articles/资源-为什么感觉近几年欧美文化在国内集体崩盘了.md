# 为什么感觉近几年欧美文化在国内集体崩盘了？
[内容链接](https://www.zhihu.com/question/648255803/answer/3432433479)

**因为美国玩后现代上瘾，解构拆台，终于自己给玩挂了。**

拜美国90年代蓬勃发展心理科学和认知科学进步，美国人发现“人类社会是靠想象共识凝聚在一起”

大部分东西都是想象“民族、历史、认知、宗教…………都是想象的。这就是后现代解构的理论基础。

然后苏联的解体。更加加剧了美国的这种认知，即：可以通过操作媒体宣传，重构认知，从而重构现实，从而创造现实。

问题是美国人猜中了前半部分，没猜中后半部分。

**确实可以靠操作媒体宣传，解构原来的认知，但是解构完了，就重构不起来了！**

**这就好比熊孩子拆钟表。拆开了，就装不回去了。**

想象的共识并不是没有现实基础，更不是可以枉顾现状，瞎JB乱搞，而是要经过“实践的检验，”能够解决或者解释现实问题，才会成为想象的共识。越复杂问题，越广阔的国家，越众多的人口，这个“实践检验”的过程就越漫长和复杂。

美国人历史太短，自身都没有完成整合，所以根本没有意识到，重构是何等艰难，重构的代价有多大。历史上所有能完成重构的民族和国家，都是熬过血与火，然后机缘巧合，运气爆棚，才迎来一线生机。

**有周期律，能够系统崩溃后重启可是高难度操作**




那现在通俗例子来理解：代码只要能跑，就不要动。

任何历史悠久的民族和文明的制度、文化都是屎山，但是只要还能跑，就不要去动，哪怕要动，也得小心在小心。

而美国历史太短，年轻气盛，得国太易，一来就碾压本地土著，独立战争是法国帮忙打赢。然后国土广阔自然物产丰富，两次世界大战都孤悬海外未受冲击，可以讲美国在建立过程中，就没有啥严酷的环境打磨。

结果以为重构是很简单的，一上来就重新造轮子，然后连测试都不测试一下，就直接更新…………这下，连回滚都没有了！

**结果美国文化就剩下人性中最本能最兽性最不可名状部分——————萝莉岛只是冰山一角。**

当然如果没有中国，美国还可以多坚持下，毕竟可以危机外移。就美国这种搞法，如果不存在中国，最后只会把全世界拖入深渊，上限就是赛博朋克2077里面那种世界！当然我对此很怀疑，现代社会的生产力非常强大，但是也非常的复杂和精密，很难想象萝莉岛那种玩意，能够建构起那么庞大复杂精密的组织。都靠反人类投名状互相拿捏把柄，聚啸山林的土匪？这就是犹太人的组织能力？

并不是说你保留点先进技术，会从全世界筛选人才，就能维持住科技发展的，这是上亿人精密协作的结果。美国之所以今天还可以有创新，是因为全球化还没有彻底崩溃。产业链还在。但是一旦萝莉岛那种玩意成主流，如何维持上亿人的协调一致呢？

**我们在未来的十年，是可以看见，美国自己亲手拆掉产业链，结果会是什么！**

事实上从美国建筑行业，造船工业，汽车工业，到现在波音开始崩了，已经可以看见是啥样了，最后蹦到IT，医药，基本也就完蛋了。

不保留基础和整个体系，尖端将不复存在！

评论里有人说中国的文化也崩盘了！

![图片描述](https://picx.zhimg.com/v2-5f0ec531766d7a5b3d6690bd7b4cf4b4_r.jpg?source=2c26e567)




中国传统文化从1860年就崩溃了呀，而且不只是文化崩盘，制度，认知，价值观，世界观，人生观…………凡是基于想象共识东西，全部都崩了呀。不但想象共识这些虚东西崩了，国土沦丧，差点解体。这都164年前的事情了，这段近代史应该每个中国人都熟悉呀！

都2024年了，你才知道吗？历史课、政治课，可是着重讲了这段历史，以及发生了什么，为什么会发生呀！怎么现在才一副恍然大悟的样子？

从1860~2008这段历史，中国一直是以西方为蓝本，不断重建自己“想象的共识”，不断常识了很多次。

直到2008年，西方自我爆炸，中国才走出自己的路。先一带一路，然后人类命运共同体，然后中国式现代化，直到这一刻，中国才在全面学习和继承西方文明的基础上，开始建构自己想象的共识！这也是新闻天天讲的呀！
