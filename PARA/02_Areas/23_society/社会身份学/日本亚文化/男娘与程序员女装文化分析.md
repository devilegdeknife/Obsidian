# 什么是男娘，女装文化代表什么？

下面是基于已收集文献与资料，对“男娘 / Otokonoko”在日本ACG及东亚语境中的**系统研究性总结**。内容围绕：**定义与概念谱系—历史演变—审美逻辑—理论框架—东亚本土化—文化意义**展开。

---

## 一、概念与术语谱系：从“男の娘”到“男娘 / 伪娘”

### 1.1 日文原词 “男の娘 / おとこのこ（Otokonoko）”

- **字面结构**：
    - 传统“男の子”（男孩）写作「男の子」，读作 _otokonoko_，意为“男性小孩”。
    - 亚文化用语“男の娘”将“子”替换为“娘”（少女），仍读作 _otokonoko_，字面变为“男少女 / 男女儿”，形成语义悖论与视觉反差[1][2]。
- **定义要点（英/日学术与百科）​**：
    - 指**具有女性化性别表达的男性**：包括外表、服饰、举止高度女性化的男性[3]。
    - 常通过**女装 / crossdressing**、化妆、体态训练等实现“看上去完全是可爱的少女”，但生理为男[3][4]。
    - 在ACG场景中既是一类**角色原型**，也是一条**题材类型 / genre**：围绕“伪百合”“误认性别”“性别暧昧恋爱”等展开叙事[5]。

> 关键区别：
> 
> - **Otokonoko** 既指“人”（角色 / 现实者），也是“题材 / 流派”；
> - 与一般“女装”（女装 / 女装男子、josou）相比，Otokonoko更强调“**美少女级**的女性外观”和“**可爱性 / 萌属性**”，而不只是单纯变装行为[3][4]。

### 1.2 中文语境中的“伪娘 / 男娘”

- 中文百科与问答指出：“伪娘（日语：男の娘／おとこのこ，Otokonoko），也称伪女、男娘，是指外表像年轻女性的男性”[5][6]。
- 中文互联网中，“伪娘”“男娘”常出现以下区分：
    - **伪娘**：源自ACG，强调“ACG角色 / Cosplay / 宅文化”中的女性化男性形象，多数在生理性别与性别认同上仍自认男性[5][6]；
    - **男娘**：在很多中文社交平台上被当作“伪娘”的同义词，但语气较口语化、带有“既可爱又有点调侃”的色彩[7]；
    - 区分于“跨性别女性”“变性人”“双性人”等：这些在中文学术与社群语境中多被视为**另一个范畴**，不简单归为“伪娘 / 男娘”[6][8]。

### 1.3 与相关术语的关系

- **Josou / 女装**：泛指男扮女装，行为层面更宽泛；
- **Bishounen / 美少年**：中性、精致的少男性外貌，但不必女装；
- **Genderless Kei / 无性别系**：2010年代日本街头时尚中，一批以“性别中性 / 打破男女装界限”为目标的年轻男性，强调的是**中性时尚**，而非特指女性化男孩[9]。
- **Transgender / 跨性别**：医学与社运语境中的性别认同转换，与Otokonoko（多被视为表演性、部分时间性的性别越界）只在部分实践上有交集，不可简单等同[10][11]。

## 二、历史演变：从前现代越界传统到ACG“男の娘”Genre

### 2.1 前史：日本的男扮女装与“女形”传统（简要背景）

- 江户时代歌舞伎中**女形（onnagata）​**由男性扮演女性角色，已构成日本文化中“男演女”的长期传统[12]。
- 近代以前男色文化、若众道等，也为“男性阴柔化”“男扮女装”提供了审美与社会记忆基础[13]。

虽然Otokonoko是1990年代后才在ACG领域形成的术语，但其可追溯的**文化惯性**来自这些历史实践：

> “男性扮演女性”在日本一直被视作**艺术 / 娱乐可接受**的越界形态，这种“可接受性”为后来的ACG“伪娘”角色提供了社会土壤。

### 2.2 1980年代：前驱文本 Stop!! Hibari-kun!

- **《Stop!! Hibari-Kun!》**（1981–1983，江口寿史）被多篇研究视为Otokonoko Genre的**前史 / Prefiguration**[1][14]：
    - 故事核心：男主Kosaku寄住在黑道头目家中，与外表完全是“美少女”的Hibari共同生活，随后逐渐对Hibari产生感情；
    - 关键设定：Hibari被设定为“生理男性，但以女孩身份生活、上学”，在叙事中被视为“女儿”和恋爱对象[14][15]；
    - 从酷儿研究视角看，该作**预演了**后来Otokonoko作品中经典的情节结构：
        
        > “男主对一位‘被大多数人视为女生，但其实是AMAB（出生指定为男性）’角色坠入爱河，并在‘自认异性恋’与‘对方的出生性别’之间产生冲突”[1]。
        

学术评价指出，Hibari 形象在1980年代已呈现出**自信、受欢迎、具性魅力的跨性别 / 性别越界角色**，在当时语境下相当前卫，同时作品整体仍夹杂时代局限（如物化、刻板描写等）[15]。

### 2.3 2000年代：No Bra 与 Otonoko Genre 的成形

- **《No Bra》**（2002–2004，川津健二朗）是另一部被集中讨论的前驱作品[1][16]：
    
    - 主角 Masato 与童年玩伴 Yuki 重逢，发现对方现在是“全校最可爱的女孩”，却又得知其生理为男；
    - 作品中Yuki逐步被呈现为**跨性别女性（trans female）​**，而非仅是“好玩的伪娘”，围绕“被当作女孩子生活”的愿望展开[16]；
    - 研究指出，Hibari 与 Yuki 两部作品——前者连载于少年搞笑漫杂志（1980s），后者连载于少年漫画杂志（2000s）——共同构成了后续Otokonoko Genre的**叙事模板和角色谱系**[1]。
- 学界一般认为：
    
    - **Otokonoko这一ACG Genre在2000年代早期“定型”**；
    - 2004年前后常被视为Genre的“成形时间点”[1][17]。

### 2.4 2000s–2010s：从二次元题材到现实亚文化

- 2000年代中后期起，带有Otokonoko角色的动画、漫画、游戏、同人作品迅速增多，形成明显的**“伪娘热 / 男の娘ブーム”**，并出现专门的选集、画集与指南书。
    
- 约在2009年前后，日本出现以Otokonoko为卖点的：
    
    - **女仆咖啡厅**（由男生女装服务）、
    - 专门化妆 / 变装摄影工作室、
    - 男性向化妆品牌与服饰店[3]。
- 2010年代：Otokonoko从二次元逐步**渗透到三次元实践**，特别是在：
    
    - **秋叶原、池袋等宅文化街区**的商业空间；
    - 线上社群：Twitter、二次元论坛、同人展，与Cosplay圈高度重合。

研究指出，Otokonoko从“纸上角色”向“现实身份”迁移时，仍保持了强烈的**ACG美学与“可爱性”逻辑**，与西方Drag文化相比，更强调“萌、少女感”，而非夸张、讽刺风格[18][19]。

### 2.5 2020年代：Senpai wa Otokonoko 等新一代作品

- **《先辈是男娘 / Senpai Is an Otokonoko》**（漫画2020s，TV动画2024起）代表了：
    - 以校园爱情为框架的**Queer Love Triangle**：  
        Saki（双性恋女孩）爱上外表为女生的学长 Makoto，后知其为Otokonoko；童年好友 Ryuji 也暗恋 Makoto，引出多方性别与取向的交错[20]。
    - Makoto被明确设定为**男性身份+女性化表达**，故事探讨的是“爱是否超越性别”的议题。
- 在当代评论与学术讨论中，此类作品被视为：
    - 连接Otokonoko传统审美与当代LGBTQ话语的一座桥梁；
    - 也是ACG产业在全球化语境下对“性别多样性”的一种“温和、可消费化”的表现。

## 三、审美逻辑：可爱、性别模糊与“安全越界”

### 3.1 “萌文化 / Moe” 与 Kawaii 审美

Otokonoko的核心审美逻辑可以概括为：

> **“女孩子的一切优点+男孩身份的反差感”**。

- **可爱（Kawaii）​**：白皙皮肤、大眼、小脸、细眉、柔和声音等“萌属性”，对观者构成强烈的“保护欲 / 心动感”；
- **反差萌**：在揭露“其实是男孩”之前，观众被引导以“女主角”观看；在真相暴露后，“错位感”本身成为新的萌点；
- **去性别化审美**：在很多作品与粉丝语境中，性别被“暂时挂起”，转而以“可爱 / 萌”作为评判标准——这与“Genderless Kei”（无性别时尚）中“衣服先于性别”的逻辑有内在亲缘性[9]。

### 3.2 叙事结构：异性恋框架中的性别张力

以 Stop!! Hibari-Kun! 与 No Bra 的学术分析为例[1][14]，Otokonoko作品中常见的情节范式是：

1. 男性主角自认是“直男”（heterosexual male）；
2. 他爱上的对象 **表面上被所有人视为女生**；
3. 情节发展中，观众及主角逐渐意识到对方“生理为男 / AMAB”；
4. 男主陷入“**自我异性恋认同 vs 对方被指定性别**”的冲突。

这种叙事模式在酷儿研究中被视为：

- 利用了主流读者对“异性恋恋爱故事”的熟悉度作为入口；
- 又在关键节点通过“性别反转”制造张力，引导读者在情感上体验**性别模糊与欲望流动**，而不必立即使用“同性恋 / 跨性别”标签来框定。

### 3.3 “安全越界”（Safe Transgression）

相对现实生活中承担高社会风险的完全性别转变（如出柜为跨性别女性），Otokonoko提供了一种**“低风险 / 部分时间”的越界路径**：

- **部分时间性**：
    - 白天在学校 /职场以“普通男性”身份生活；
    - 夜晚或特定线上空间以“女装身份”出现。
    - 台湾田野研究使用“**part-time gender transgression / 部分时间性别越界**”描述这种实践[11]。
- **可控场域**：ACG圈、女装咖啡厅、伪娘基地等都有明显“边界”，可以在需要时“撤出”（卸妆、换装），从而保证日常生活中男性身份的稳定。
- **话语缓冲**：
    - 在日本与台湾语境中，“伪娘 / Otokonoko”这一亚文化标签**刻意与“transgender / 跨性别”一词保持距离**，既避免触碰法律、医疗与制度性的尖锐议题，也减轻了社会对其的政治警惕[10][11]。

因此，从理论上讲，Otokonoko 提供的是一种 **“安全的性别实验场”**：  
观众和实践者可以在不彻底破坏现有性别秩序的前提下，探索性别流动、身份解构与审美越界。

## 四、理论视角：性别表演、酷儿时间性与亚文化消费

### 4.1 性别表演（Gender Performativity）

沿用Butler的性别表演理论，Otokonoko可被看作：

- **对“女性性别表演”的再表演**：  
    男性角色/实践者通过学习、模仿、放大“传统女性姿态”（温柔、可爱、顺从）来完成自身的性别伪装；
- 同时也在显露出：
    
    > “所谓女性气质本身就是可学习、可模仿、可买卖的符号组合”，并非与生理女性本质绑定。
    

某些研究将此视为对性别本质主义的潜在反讽：

> 如果一个男性通过服饰与妆容就能“看起来比真实女性更可爱”，那么“女性气质”就不再能被视为**自然 / 本质**，而是**可被任何人习得的文化技术**[2][3]。

### 4.2 酷儿时间性与“平行时空”

台湾关于“偽娘”的民族志指出[11]：

- 实践者往往强调“白天是普通男性，夜晚是伪娘”的时间切分；
- 他们会用“平行时空”“游戏世界”等隐喻来讲述自己的扮装经验：
    - 将女装状态视为“进入另一个宇宙 / 时间线”；
    - 这种“另一个时间”与现实生活并存而互不完全重叠；
- 这种时间经验被纳入**酷儿时间性（Queer temporality）​**框架：  
    超越主流异性恋生命周期（上学–结婚–生子）的线性模式，而在多条时间线之间切换。

这与ACG文化本身对“虚拟世界 / 二次元”的迷恋高度契合，使得“男娘”成为**介于现实与虚拟之间的性别存在**。

### 4.3 亚文化消费与性别符号商品化

Otokonoko 同时也是典型的**亚文化消费品类**：

- **符号商品化**：
    - 女装男孩形象被制作为手办、抱枕、周边、写真集；
    - 真实伪娘通过写真、付费内容、直播打赏等实现经济收益[18][19]。
- **消费者结构**：
    - 日本研究指出，Otokonoko题材既面向“对伪娘有性幻想的男性读者”，也在欧美跨性别女性读者群中具有高人气[1]；
    - 在中国，男娘内容常出现在直播、陪玩、短视频等平台，被包装为“高颜值陪伴型角色”。
- **性别符号消费**：  
    观众并不一定真正认同跨性别政治，却通过消费“女装男孩”这一符号来体验：
    - 暧昧的欲望位置（既非传统异性恋，也非明确同性恋）；
    - 对“传统男性气质”的暂时性逃离。

## 五、东亚本土化：台湾“偽娘”与中文互联网“男娘”

### 5.1 台湾：从日本次文化到在地社群

罗盘针对北台湾“偽娘”社群的研究显示[11]：

- “偽娘”一词20世纪末自日本ACG次文化输入，约2010年前后被台北扮装社群主动用作自称；
- 都市内陆续出现“偽娘主题工作室”“偽娘基地”等实体空间，为扮装者提供化妆、更衣、寄物、交友等服务，是一种“**共享空间+缓冲地带**”；
- 偽娘实践被理解为：
    - **追求美学与娱乐的自我改造计划**，而非彻底性别转换；
    - 多数实践者仍在社会与法制层面维持“异性恋男性”身份，仅在部分时间进入女装状态；
- 研究者强调：
    - 偽娘应与“跨性别者”区分看待；
    - 它提供了一种不同于西式跨性别运动、但同样具有性别越界意涵的在地形式。

### 5.2 中国大陆与华语互联网：“伪娘 / 男娘”的大众化与争议化

在中文互联网中，“伪娘 / 男娘”现象大致经历：

1. **2000年代末–2010年代初**：
    - ACG同好圈中出现大批“伪娘Cos”“女装正太”等内容；
    - 选秀节目与综艺出现“伪娘型选手”，引发“审美多元 / 毁三观”等争议；
2. **2010年代中后**：
    - B站、微博、小红书等平台上，“男娘”“女装大佬”成为流量标签，孕育出一批以女装为主要人设的UP主/主播；
    - 同时，主流媒体与部分学界把这一现象与“男性娘化”“阳刚危机”联系，常以批判视角讨论[7][21]。
3. **2020年代**：
    - 伴随官方文件中出现“防止男性青少年女性化”“杜绝娘炮等畸形审美”等表述，“男娘”在主流话语中被高度单一化为“负面审美符号”[22]；
    - 但在性别与酷儿研究、跨性别社群档案中，“男娘”又被视作观察**当代年轻男性性别气质变迁、数字时代性别实践**的重要个案[23][24]。

整体来看，华语语境中的“男娘”处在**高度撕裂**的框架中：

> 一方面是ACG与年轻人中带有“可爱”“好玩”意味的次文化身份；  
> 另一方面则被主流媒体当作“男性危机”的症状与“需要纠正的审美问题”。

## 六、综合评估：男娘 / Otokonoko 的文化意义

从1980年代的Hibari到2020年代的Senpai wa Otokonoko，再到台湾“偽娘基地”和中文互联网“男娘”社群，可以看到：

1. **在ACG叙事层面**，男娘/Otokonoko：
    
    - 提供了一种在异性恋框架内植入酷儿张力的叙事机制；
    - 让主流读者在“安全的故事距离”中体验性别模糊与欲望流动。
2. **在审美与消费层面**：
    
    - 男娘是“可爱文化+男性身份”的合成体，推动了**去性别化 / 性别中性的审美扩散**；
    - 同时被产业高度商品化，成为宅文化、时尚、美妆等多重消费场域的关键符号。
3. **在性别理论层面**：
    
    - 它具现化了Butler的性别表演理论：“女性气质可被学习与复制”；
    - 通过“部分时间性别越界”，拓展了酷儿时间性在东亚的在地版本。
4. **在东亚社会政治层面**：
    
    - 日本、台湾较多将其视为“亚文化实践”和“性别多元的一部分”；
    - 中国大陆则处在国家阳刚话语与青年亚文化抵抗交织的张力场中，“男娘”成为围绕男性气质、代际冲突与文化主权辩论的焦点符号。

**因此，要理解“男娘”这一ACG亚文化概念，必须同时把它看作：​**

- 一种 **ACG性别叙事技巧**（制造性别张力与情感冲突的工具）；
- 一种 **审美元件与消费符号**（被商品化的性别越界形象）；
- 一种 **部分时间、可撤回的性别实践**（安全越界的路径）；
- 以及在东亚不同社会中，被用来谈论 **男性气质危机、性别多样性与文化政治** 的关键话语节点。

## 一、概念与术语谱系：男の娘 / 伪娘 / 女装文化

1. **男の娘（otokonoko）**
    
    - 日语本来“男の子”（男孩子），改写为“男の娘”（娘=女孩），形成谐音双关，特指**外表完全按“可爱少女”绘制，但设定为“男性”的二次元角色类型**。[[ja.wikipedia](https://ja.wikipedia.org/wiki/%E7%94%B7%E3%81%AE%E5%A8%98)]​
        
    - 与“扶她”“性转换”等设定不同，男の娘通常**生理上保持男性（尤其被强调有“男性身体”），但视觉上几乎与美少女无异**。[[tdr.lib.ntu.edu](https://tdr.lib.ntu.edu.tw/bitstream/123456789/79715/1/U0001-2308202102225100.pdf)]​
        
2. **伪娘 / 男娘（中文）**
    
    - 中文“伪娘”一词借译自“男の娘”，最早在ACG 圈内使用，指“天生相貌、体型接近女性，女装后具有强烈萌属性的男性”，与“人妖”“变性人”区分开来。[wikipedia+1](https://zh.wikipedia.org/zh-sg/%E5%81%BD%E5%A8%98)
        
    - 《伪娘白皮书》给出了一套圈内标准：生理为男、条件天生非手术获得、外貌身材可被视作女性、美感上“比女更女”、心理性别与性取向仍自认男性等，强调的是一种**“双性魅力”的男性**。[[educ-res](https://www.educ-res.com/index.php/er/article/view/3053/2917)]​
        
    - 在中文网络上，“男娘”通常是较中性的自称；“伪娘”带有“伪、假”的贬义色彩，学界和圈内都有是否使用该词的争议。[cssn+1](https://cssn.cn/zx/mx/202210/t20221027_5556022.shtml)
        
3. **与跨性别、drag、现实女装的区分**
    
    - **跨性别（transgender）**：指性别认同持续地与出生指定性别不符，往往追求在法律、身体、社会角色上“全职”转换。许多跨性别女性并不接受被叫做“伪娘”，认为那是否认自身“真实女性身份”。[[tdr.lib.ntu.edu](https://tdr.lib.ntu.edu.tw/bitstream/123456789/79715/1/U0001-2308202102225100.pdf)]​
        
    - **drag queen / 反串（舞台女装）**：以夸张女性形象为表演的艺术形式，观众一般清楚其“反串”属性。
        
    - **男娘 / 伪娘**更多指：
        
        - 部分时间（part-time）女装；
            
        - 目的以**审美、自我表达、娱乐与社群归属**为主，而非必然追求生活中永久性别转换。[[tdr.lib.ntu.edu](https://tdr.lib.ntu.edu.tw/bitstream/123456789/79715/1/U0001-2308202102225100.pdf)]​
            
    - 这也是你排除“跨性别医疗”“易装癖诊断”的原因：这里讨论的是**文化现象与性别表演，而非医学或病理化视角**。

## 二、历史脉络：从传统“阴柔男性”到ACG 萌属性（1980s–至今）

## 1. 先行传统：歌舞伎女形、美少年与“男色”

- 日本自江户时代歌舞伎“女形”（onnagata）起，就有**男性扮演女性角色、并被视为“更胜于女”的女性性别美学**。[[cssn](https://cssn.cn/zx/mx/202210/t20221027_5556022.shtml)]​
    
- 古典文献中，阴柔、美少年与“男色之好”构成了早期“女性化男性”的审美基础，这为后来的bishōnen、美型少年、男の娘奠定了文化土壤。[[cssn](https://cssn.cn/zx/mx/202210/t20221027_5556022.shtml)]​
    

## 2. 1980s–1990s：Bishōnen、美型少年与性别反串桥段

- 80–90年代的少年漫画、动画里，已经大量出现：
    
    - 美型、阴柔的少年（Bishōnen）；
        
    - 反串情节（男主被迫女装、性别误认的喜剧桥段）。
        
- 这时还没有“男の娘”这一固定术语，但**“美少年 + 性别模糊”的萌点已形成套路**，为2000年代标签化铺路。[[tdr.lib.ntu.edu](https://tdr.lib.ntu.edu.tw/bitstream/123456789/79715/1/U0001-2308202102225100.pdf)]​
    

## 3. 2000年代：男の娘作为“萌属性”被命名与系统化

- 2000年前后，“男の娘”一词在日本匿名论坛2ch等ACG 社群中稳定使用，并在2010年前后进入流行语榜单。[[tdr.lib.ntu.edu](https://tdr.lib.ntu.edu.tw/bitstream/123456789/79715/1/U0001-2308202102225100.pdf)]​
    
- 在“萌文化”中，角色被拆解为“属性”（眼镜娘、傲娇、病娇等），男の娘被视为**一种可被消费的性别属性**：
    
    - 视觉上几乎完全是“可爱少女”（kawaii）；
        
    - 剧情中以“其实是男的”的反转制造震惊感与额外萌感（“这么可爱不可能是女孩子”一类台词）。[[tdr.lib.ntu.edu](https://tdr.lib.ntu.edu.tw/bitstream/123456789/79715/1/U0001-2308202102225100.pdf)]​
        
- 与此同时，伪娘咖啡厅、男の娘酒吧、女装沙龙等现实商业空间在东京（秋叶原、新宿）出现，形成**虚构角色—现实服务业—同人产业**的三角结构。[[tdr.lib.ntu.edu](https://tdr.lib.ntu.edu.tw/bitstream/123456789/79715/1/U0001-2308202102225100.pdf)]​
    

## 4. 2010年代以后：东亚扩散与在地化

- **台湾**：
    
    - 约2010年前后，本地扮装者开始采用“伪娘 / weiniang”自称；2015年前后台北出现以“伪娘”为主题的变装空间（如“伪娘基地”），提供妆发、衣橱与社交场所。[[tdr.lib.ntu.edu](https://tdr.lib.ntu.edu.tw/bitstream/123456789/79715/1/U0001-2308202102225100.pdf)]​
        
    - 2019年成立“中华民国伪娘交流协会”，获得内政部正式登记，显示伪娘社群在台湾**从隐匿实践走向公开的社团与产业**。[[tdr.lib.ntu.edu](https://tdr.lib.ntu.edu.tw/bitstream/123456789/79715/1/U0001-2308202102225100.pdf)]​
        
- **中国大陆**：
    
    - “伪娘”一词经由日本ACG 翻译与台湾传播进入内地，2010年前后在选秀节目、网络红人（如某歌唱比赛选手）事件中出圈，引发媒体对“伪娘现象”“娘炮之争”的集中报道。[language.chinadaily+1](https://language.chinadaily.com.cn/a/201809/13/WS5b9a2e3ca31033b4f4655ebc.html)
        
    - 2010s B站、贴吧、“十万粉就女装”等网络梗，使“男娘”成为ACG 宅圈与短视频平台中的高流量标签。[[funinxxoo](https://funinxxoo.com/blog/-nanniangwenhuadaodishizenmyihuishi)]​
        
- **整体看**，1980s–2000s 是**审美与叙事套路形成期**，2000s–2010s 是“男の娘/伪娘”作为**命名概念—商品—社群身份**的确立与扩散期。

## 三、审美逻辑：萌文化、性别模糊与去性别化审美

## 1. “画女硬说男”：性别记号与身体的错位

- 男娘的关键不在“穿了女装”，而在于：
    
    - **身体被绘制成少女模板（细腰、无胡须、光滑皮肤、少儿感）**；
        
    - 然后在设定里“揭示”这是男性，从而制造一种**性别符号与身体的错位快感**。[wikipedia+1](https://ja.wikipedia.org/wiki/%E7%94%B7%E3%81%AE%E5%A8%98)
        
- 这里的快感是典型“萌”：观者对角色的脆弱、可爱、羞耻感产生保护欲与情绪卷入，而**不一定指向现实性行为**。
    

## 2. 去性别化审美与“中性化可爱”

- 在ACG 语境中，“可爱（kawaii）”本身逐渐**从“女性美”中抽离，变成一种可贴在任何身体上的审美范畴**：
    
    - 男性角色只要满足“白、细、软、小、干净”等视觉代码，就能被视作可爱；
        
    - 服装（制服、洛丽塔、JK、病娇服饰）等性别符号变成**可自由组合的“皮肤”**。[moegirl+1](https://zh.moegirl.org.cn/%E5%A5%B3%E8%A3%85)
        
- 这对应了一种“去性别化审美”：**性别不再是决定美的首要维度，“可爱/萌”凌驾于男女之上**，甚至有“我身为男，但比你更女”的话语。[funinxxoo+1](https://funinxxoo.com/blog/-nanniangwenhuadaodishizenmyihuishi)
    

## 3. 安全越界（safe transgression）

- 对观众而言：
    
    - 通过追随男娘角色，可以**在“明知其为男”的前提下享受“女体美感”与性别混淆的刺激**，而不必在现实中承担性取向、社会标签的压力。
        
- 对创作者与产业而言：
    
    - 男の娘提供了一种**“既卖女体又卖反转”的加倍收益”，在保守审查之下仍能声称“这是男性”**。
        
- 对部分扮装者而言：
    
    - 现实中的女装多发生在特定场所（cos、基地、派对）与部分时间，形成“平行时空”，在这个时空内可以越界，在日常生活中仍维持“普通男性”角色。[[tdr.lib.ntu.edu](https://tdr.lib.ntu.edu.tw/bitstream/123456789/79715/1/U0001-2308202102225100.pdf)]​
        
- 这种在虚构文本或限定空间里的越界，就是典型的“安全越界”：**在规则之内被管理好的叛逆**。

## 四、男性气质危机与性别模糊：男娘如何重写“做男人”

## 1. 日本与东亚的“阳刚焦虑”

- 日本长期存在“草食系男子”“王子系”“美形男子”等讨论，反映：
    
    - 传统以事业、权力、家庭支配为中心的“昭和男性气质”与
        
    - 新一代温柔、内向、对竞争冷感的男性之间的张力。[pmc.ncbi.nlm.nih+1](https://pmc.ncbi.nlm.nih.gov/articles/PMC11024095/)
        
- 中国大陆语境里，“娘炮之争”“防止男性青少年女性化”的官方表述，正是对这种**阴柔男性可见度上升的焦虑回应**。[language.chinadaily+1](https://language.chinadaily.com.cn/a/201809/13/WS5b9a2e3ca31033b4f4655ebc.html)
    

## 2. 男娘作为“去阳刚”的男性形象

- 学界分析指出，伪娘现象在性别美学上**削弱“强壮、粗犷、支配”这些传统阳刚指标，转而强调瘦弱、敏感、细腻、美感劳动**。[educ-res+1](https://www.educ-res.com/index.php/er/article/view/3053/2917)
    
- 伪娘被粉丝喜爱，部分原因是：
    
    - 同时拥有“温柔、体贴”的女性化特质与“责任感、保护欲”的男性化特质，被建构为“理想对象”。[[educ-res](https://www.educ-res.com/index.php/er/article/view/3053/2917)]​
        
- 对青年男性而言，“男娘/女装”也提供了一种**逃脱“必须强硬”的角色要求的方式**：可以承认脆弱、追求美感，把对抗社会压力转化为对身体与形象的精细经营。[educ-res+1](https://www.educ-res.com/index.php/er/article/view/3053/2917)

## 五、性别表演（Gender Performativity）视角：部分时间的越界

## 1. Butler 的性别表演简述到男娘实践

- 性别表演理论认为：所谓“男人/女人”，不是内在本质，而是**通过重复的姿态、服装、语气、行为不断“演出来”的效果**。
    
- 男娘实践很好地体现了这一点：
    
    - 通过化妆、脱毛、选裙、训练姿态与说话方式，去“排除男性痕迹”，营造女性表象；
        
    - 同时，在合适时机“翻牌”（揭示自己是男）又让这一表演显露其“伪装性”，从而对“真女人/假女人”的二元做出戏仿。[[tdr.lib.ntu.edu](https://tdr.lib.ntu.edu.tw/bitstream/123456789/79715/1/U0001-2308202102225100.pdf)]​
        

## 2. “部分时间”的性别越界

台湾田野研究对北台伪娘社群有非常清楚的描述：[[tdr.lib.ntu.edu](https://tdr.lib.ntu.edu.tw/bitstream/123456789/79715/1/U0001-2308202102225100.pdf)]​

- 多数扮装者：
    
    - 证件性别为男，日常工作中以男性身份生活；
        
    - 自认“男性、异性恋”，却在夜晚/周末进入女装时空。
