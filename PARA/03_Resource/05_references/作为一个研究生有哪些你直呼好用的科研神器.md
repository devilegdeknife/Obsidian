# 作为一个研究生，有哪些你直呼好用的科研神器？
[内容链接](https://www.zhihu.com/question/484596211/answer/2160222107)

介绍几个算的上科研神器的网址吧。

## 1.Connected Papers

网址：[Connected Papers | Find and explore academic papers](https://link.zhihu.com/?target=https%3A//www.connectedpapers.com/)

![图片描述](https://picx.zhimg.com/v2-bca70eb2b3f0cf47a91e0c90e7dca9e3_r.jpg?source=2c26e567)

下面是这个网站的介绍：

- 获得一个新的学术领域的视觉概述

搜索一个论文，它会形成一个图表，通过球连接来建立这个论文在所在领域的位置

![图片描述](https://pic1.zhimg.com/v2-67cda618a37672bcc663840b14291f3f_r.jpg?source=2c26e567)

1) 每个小球代表一篇文献，最中间黑色边框的小球就是搜索的文章

2) 右下角有年份分布图标，小球的颜色越深代表时间越近

3) 球的大小与被引用次数有关，小球越大表示被引次数越多

4) 小球之间的连线表示引用关系，相似的文章会距离较近或者中间有很强的连线

5) 点击小球右侧会出现该文献的相关信息。

- 确保没有漏掉一篇重要的论文

在一些领域，比如机器学习，许多新的论文已经发表，很难跟踪。通过连接文件，你可以搜索和直观地发现最近的重要文件。不需要列清单。

- 发现最相关的优先级和衍生作品

使用先前的工作视图，找到感兴趣领域的重要文献。使用衍生作品视图来查找该领域的文献评论，以及最近发表的文章之后的最新状态。

**总的来说，这个网站真的很有用，许多功能还有待发掘，我现在还没有搞太懂，只是简单用来搜索论文来源的逻辑链。**

5.[X-MOL科学知识平台](https://link.zhihu.com/?target=https%3A//www.x-mol.com/)

再更新一个网站，这个网站我比较常用的是文献直达和导师库

![图片描述](https://picx.zhimg.com/v2-5b1d667adf9928bb843a6f3151a5778b_r.jpg?source=2c26e567)

文献直达就复制doi或者期刊名、年、卷、页码就可以查询，我一般都是当国外知网用。

然后那个导师库，你可以进行大学搜素，然后找相关导师，它会出来导师的简短介绍和发表的论文

![图片描述](https://picx.zhimg.com/v2-501b186c675787dd2b135204a8e2c693_r.jpg?source=2c26e567)

当然还有联系方式，只不过很少用

![图片描述](https://picx.zhimg.com/v2-50ab3819c6940de18ba9e7d352b417ec_r.jpg?source=2c26e567)
