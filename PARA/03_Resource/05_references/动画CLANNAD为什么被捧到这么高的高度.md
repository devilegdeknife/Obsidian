# 动画《CLANNAD》为什么被捧到这么高的高度?
[内容链接](https://www.zhihu.com/question/2555194699/answer/115586246659)

说CLANNAD「只不过」是一个司空见惯的河南故事，其实就是一个很经典的发问，为什么我们的苦难没有人提及，你们的苦难却能受到如此关注？

平心而论，这个发问还是挺发人深省的，是有很深刻的批评潜力的。但是当批判的矛头从「探究我们的苦难无人提及的原因」，以及「如何提出我们的苦难」转向了一种，「既然我们的苦难无人提及，那么你们的苦难也必须不值一提」的嫉妒之后，整个发问就变味了。

这仿佛是在说，**司空见惯的苦难不配提及，每个人必须默默接受**。

我不是很喜欢「认知配得上苦难」这句话，但是图片里的人却把这句话提现得淋漓尽致。他声称自己扒掉了CLANNAD的皮露出了河南故事的内核，却在自己对河南故事的叙述中引入的社达价值观毫无自觉。在他看来，一切的苦难都是因为“混”，渚只不过是一个有病嫁不出去的女人，朋也和她结婚只不过是因为“混”的不行而没有选择。只要“混”得好了，自然可以抛弃这个鬼地方和糟糠妻女走上人生巅峰。所有的苦难都必须归因于「自我责任」，用现在流行的话来说就是「你怎么不多找找自己的问题？」

而CLANNAD恰恰是对这种「自我责任」的价值观进行抵抗的作品。

一个很显然的事实是，对朋也以及和对CLANNAD产生共鸣的玩家来说，小镇绝不是「混」出名堂之后就可以抛弃的人生过客，渚也不是因为混的不好而不得不将就的嫁不出去的生病女人，而是正如整部作品在最后借着「光玉」之力所发出的扪心自问一般，即便有着诸多苦难，但是即便世界重来一次朋也和玩家依然还会选择与渚相遇相识。这很显然是与图片中对河南故事的社达叙述大相径庭的。

一个容易被忽视的事情是，CLANNAD原作游戏到东映电影版、京都动画TV版的2004~2008年，恰好是所谓「失去的一代人」、「就职冰河期世代」的时代。小泉新自由主义改革的恶果带来年轻人（15~24岁）高达10%的失业率，以及劳动市场上30%的非正规雇佣率。这是一个即使年轻人再怎么努力，也很难避免失业或者非正规雇佣的命运的时代。而更为地狱的事情在于，与新自由主义经济改革一同到来的社会价值观的剧变——「自我责任论」的流行，让思考「苦难的来源」成为了一种禁忌。

「你不干有得是人干」、「你的贫穷都是源自自身的不努力」、「怎么不多找找自己的问题」、「每个人是自己的第一责任人，不要给政府添麻烦」等等话术，在这个时代的日本屡见不鲜。这样一种思想上的禁忌，导致这个时代文艺作品往往给人一种极度内敛的感受，将诸多源自社会的苦难和情绪，压缩并封闭在个人的自我空间中去处理，而这种倾向特别是动漫作品尤为突出。

麻枝准作品最大的问题，同时也是其最大的特点，也就是在这里。例如中日观众吐槽多年的渚的自宅出产问题，比起渚不理智的「自我决定」，其实更让人迷惑的是这一「决定」能够被众人轻易的接受，以及这一接受过程中「社会」的隐身。放在今天少子高龄化已经深入骨髓的日本，自宅出产死亡这种新闻肯定会引起诸多「定体问」，以批判日本政府在生育·出产补贴政策上的抠抠搜搜。但在2000年代这个充斥着「自我责任」的语言空间中，思想的禁忌让这一问题不得不在自我空间之中进行处理。

换而言之，麻枝准作品中「廉价」的悲剧其实并非是源自其剧作能力的低下，其笔下角色的一种似是而非的「不理智」，本质上是时代将本不该由个人承受的苦难与问题强加与个人的肩膀之上时所迸发的一种失措与迷茫的象征性体现。其感动叙事的「空洞」也并非是源自其思想的匮乏，而是面对「自我责任」大行其道所带来的思想禁忌的一种创伤性处理。

也正是因为意识到了麻枝准的「廉价」背后有着诸多社会环境和历史包袱的存在，著名Key厨东浩纪才会在他的AIR论中指出，麻枝准笔下总是缺少父亲的家庭结构，以及将男主人公=玩家变成乌鸦剥夺其干预剧情能力的处理，本质上是一种对战后日本社会中父性的崩解和不能的反映，而以和动漫美少女做爱为目的的Galgame的形式，表达这一不能性主题，更是体现了御宅族对自身欲望的一种集内省和快感为一体的复杂心境。甚至我们完全可以沿着东的问题意识进一步说，这种动漫化表达同时还揭示了不得不用私人空间处理战后日本问题时的禁忌性。

如果说AIR是「我无法成为父亲」的一种不能性体验，那么CLANNAD就是「即便是这样的我也能成为父亲吗？」的一种自问与恢复。在一定程度上来说，渚以及其家庭还有学校的同学们都成为了朋也以及玩家重建自身「父性」信心时的血包这个说法是有道理的。但仅仅把角色的「非理性」视作一种满足剧情设计需要的功利性并不假思索地加以批判的行为，无疑是忽视了「非理性」的剧情与行为背后的思想性问题。其次，所谓「血包」一说也不能仅从经济的角度来把握，在这一点上题主图片中的说法就犯了经济还原论的错误。

当然，用经济去把握重建「父性」，这个角度本身并不算一个很严重的错误。实际上，上面提到的小泉改革所带来的「失去的一代」的失业与非正规雇佣，无疑在2000年代经济性地加速家庭和父性的崩解起到了至关重要的作用。一个没有经济实力的男性，在劳保价值观内无疑是无法获得作为父性的信心的。而进行经济再分配，提高劳动报酬和社会保障确实也是重建家庭和父性时经常可以看到的一种解决方案。

但问题在于2000年代日本不仅有经济再分配的问题，同时还有尊严的再分配，即承认的问题存在。「再分配和承认」的对立可以说是西马中一个比较经典的问题，有兴趣的可以去参考南希·弗雷泽的著作。大体上来说其内涵在于，人要活得像一个人，不仅有其经济维度，即公正的再分配，同时还要有保障尊严的维度，即消除歧视。例如，仅仅是严格执行同工同酬，对少数族裔、性少数、女性的歧视并不会因其经济上的再分配变得公正而消失，因而尊严需要另外一套系统来保障。从结果上来说，承认理论成为了现代身份政治的重要组成部分。

虽然在今天，对承认即身份政治的过度关注会导致对新自由主义带来的再分配不公的轻视，已经在诸多领域得以认识并加以批判，但大体上来说，同时关注二者的二元论态度还是主流的。不过在2000年代初的日本，还没有今天这样的视角。现在的人们或许会难想象，在小泉新自由主义改革的前夕，非正规雇佣的自由职业者会被视作是一种逃离了千篇一律的工资劳动，充分发挥了个人特质和能力的人生规划，成为诸多年轻人实现「承认」的梦寐以求。但他们投票选出的首相无疑把自己带向了南辕北辙的道路。

言归正传，从「再分配和承认」的角度来说，麻枝准笔下的诸多故事，本质上都是一种在排除了经济再分配改革的前提下——或者说在面临「自我责任」的思想禁忌的前提时——依托地域共同体进行承认的再分配的模式。即便是经济贫困、举目无亲的我们，也可以互相抱团取暖形成一个小共同体，互相为对方提供并满足承认的需求。即便是其笔下「父性」的恢复，也并非是从经济再分配的角度上的恢复——更加不会是飞黄腾达之后就抛弃小镇和糟糠妻女的劳保式的恢复——而是基于世界系的二人关系，拟似家庭的组建等私空间的重建所带来的一种另类的恢复。

因而，就对飞黄腾达的劳保以及对「自我责任」思想禁忌的抵抗来说，麻枝准的作品虽然有着和动漫美少女做爱的形式，却通过在父性的不能和恢复上的特殊处理，将御宅族和战后日本勾连起来，在一定程度上获得了与女性主义的共鸣，麻枝准对「我无法成为父亲」、「即便是这样的我也能成为父亲吗？」的问题的症候性描绘，也成为了当时御宅族的身份政治的一个典型代表。这与今天更多被视作与女性主义不共戴天的御宅族形象有着显著的差异。

需要额外补充的是，日本并非只存在这种将「自我责任」禁忌进行动漫化处理的思想。例如08年经济危机以后在日本兴起的失去的一代批评（ロスジェネ批評），就更加偏向传统的政治经济学批判，从正面抵抗新自由主义并主张公正的再分配，但是失去的一代批评在处理「阶级政治」和「身份政治」的平衡上做得不好，很容易陷入低收入男性vs中产阶级女性的经典争论里，因此失去的一代批评的旗手们都在重新思考御宅文化批评的价值。

总的来说，包括CLANNAD在内麻枝准的诸多作品之所以能够受到广泛的欢迎，主要还是其中对社会的一种症候式的书写，无意中切中了时代和观众的要害。在书写苦难来源于社会成为一种思想禁忌的语言空间中，开辟了一种隐微书写的、独特的政治·思想话语。
