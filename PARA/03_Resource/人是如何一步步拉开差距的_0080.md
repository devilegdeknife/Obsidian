# 人是如何一步步拉开差距的？
[内容链接](https://www.zhihu.com/question/1907699737605439644/answer/1907708464857346908)

我跟我老婆都是中科大本科，我数学专业，她英语专业。她因为读研究生所以晚一年毕业，因为数学专业当时科大是五年制。

大学

[https://www.zhihu.com/video/1912673342961460761](https://link.zhihu.com/?target=https%3A//www.zhihu.com/video/1912673342961460761)

毕业前一年我没有好好找工作，她去上海实习了一个暑假。

大学毕业后我去了北京创业，其实就是跟着兄弟去瞎混了一年，因为啥也不懂，啥也没做成。老婆已经开始在四大实习了。

毕业两年，我回到南京陪她，陪了半年我社招进了一个大专做辅导员。她在四大两年了。

毕业六年，我也读了个硕士，我讲师没有评上，评上了校内正科，工资五万一年。她跳槽了三家公司最后进了某大国企做部门财务副主任。期间她考取了注会，收入是我的三倍多。

毕业九年到十一年，因为觉得工作无聊，前途没劲，我打算申请出国，我老婆也很支持。我们一起准备GMAT。我考了550，我老婆考了770。我们出国了，我老婆读MBA，我继续准备。她读了两年MBA，找到某大厂开始工作。我总算考进所在州的金融硕士。

毕业十二年到十四年，我读金融硕士并在某小fund实习，直到口罩开始失业。我老婆工作稳定，promote到总监。

毕业十五年到现在，我自己在家做家教，教孩子们数学，从小学到线性代数，多元微积分，学费$100每小时；我老婆又promote了一次，离CFO还有三层，收入是我的四倍。我还做点兼职替原来的的fund做点数据分析的零活。今年我还作死去了一家全职的高中，结果每天平均要工作十一个小时，看上去又老了十岁。

我很努力，但是差距越来越大。我命很好，能遇到这么优秀的老婆，话说刚开始恋爱的时候我还有点瞧不上她，后来发现人家智商，情商都很好，还很努力。我也很辛苦，因为总也翻不了身，说实话要是不是老婆比我混的好太多，也许我就在那个大专混吃等死了，没办法。差距是一点点积累的，并且选择也非常总要。我毕业后浪费了一年，后来进了高校可能也不是一个很好的选择，读硕士的时候我不应该选择小公司，如果有个大公司给我机会，应该不会那么快失业。

也不是完全没有收获，我发现我是真的是一个好老师，因为我耐心，善于沟通，又数学知识扎实。所以我打算辞掉学校的工作，安心做辅导，照顾好家庭，等老婆继续升职。她再升两级的话我们就可以可以自由了。现在这样的话，在混个十年就退休了，回来养老，希望到时候又养老签证。
