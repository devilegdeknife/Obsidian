# 实验室师兄不愿意带刚进组的师弟/师妹，最好的解决方法是什么？
[内容链接](https://www.zhihu.com/question/30849269/answer/2958553740)

今天疯狂星期四心情好，哥们给你整点实在的。

我当时本科课题双选结束时只有我一个本科生成功进入这个课题组，其他的有的保研的都没进来，有个我们班的哥们甚至找了班主任去求情都没通过。

有些回答里说，让你积极上进谦虚一点的，没用。

拜托，你积极上进跟我有什么关系？

你拿了项目奖金会分给我？

还是你发期刊了会把我写成二作三作？

那种觉得自己表现好学生姿态别人就会热衷带你的，多半是没有出过象牙塔，觉得还是在初中小学，只要自己勤奋好学，老师就会多点拨自己。

还有的说平时勤快一点，帮师兄师姐点小忙和打杂....

大二我在上海一家互联网公司实习了快半年，部门里的大哥大姐都跟我很熟，平时都会互相请喝奶茶吃水果，时至今日也依然关系很好，我到上海他们会约饭的程度。

但问有没有转正hc(head count,指招聘名额)的时候都是抱歉无能为力，你学历211，我们只要双9(指本硕学历均为985)。

玩伴就是玩伴，不要打感情牌哈，谁在乎你帮这点小恩小惠？

也别提什么奶茶水果吃饭的，谁缺你这几十一百的？

所以问题精简成：自己屁用没有啥也不会，但是要麻烦别人教自己，应该怎么办？

第一种方法就是刚才说的例子，让教你的人变成你的初中老师，让你的学习成果跟他们的工资挂钩。

比如你跟师兄说我付费给你，你教我一下每个实验步骤和思路，按天算，一天200，我觉得他们应该会很乐意。

初中小学老师只是因为你的成绩好了他有绩效奖金而已，不然谁愿意管你？

第二种，明确自己就是个废物这件事，始终具有弱者的自我认知，尽可能减少师兄解决困难的成本。

总结一下，提问时




1.用意义明确且描述清晰的标题。用最简洁易懂的文字描述你在什么方面遇到了什么问题。

垃圾标题：您现在忙吗/我有个问题。

好标题：关于毕业设计中选题的确定。




2.用精确符合语法的语句描述问题。在任何场景下，没有人喜欢阅读长句子，郑渊洁除外。

垃圾句子：明明我按照实验步骤做了实验但是镁条黑了，为啥？

好句子：我尝试复现镁条燃烧实验的时候，镁条发出了彩色的光芒并发出只因你太美的声音，这跟书上的发出耀眼白光不太一样，但镁条在点燃之前有点发黑，这可能是我原材料的问题还是实验环境的问题？




3.说问题是什么，不要讲你的猜测，描述你的目标而不是你的过程。

垃圾提问：我怎样才能变成ikun？

好提问：我在刷视频的时候刷到了很多黑我家哥哥的人，我不理解他们为什么要这样？我家哥哥明明是连续二十届影帝，付出了无数努力的时间，但是他们却很片面的在嘲讽，这是因为他们不了解还是他们嫉妒？




4.去掉无意义的问句，不要催回复。不要在提问中出现类似于“我在哪里可以找到X资源？”“你能帮我吗？”的蠢问题，如果你的描述本来就不清晰，这样的句子只会让你的提问变得更加复杂。

5.如果你搞不懂对方的解答，把解答当成一个新的问题，重复上面的思考步骤。




我怕让不擅长沟通的读者误会我就是让你们当狗，所以补一条6

6.礼貌和谦卑，但不是低声下气。

跟我同级的十多个人联系了我导师，其中也有后来保研了的、加权年级前几的，也照样没逃掉被拒绝的命运，这就是没搞明白，在沟通中没有尽可能描述：我会怎样减少您的麻烦。

你保研了，你加权前几，你竞赛得奖，很厉害，然后呢？

我带你学院又不给我发补助，你那些本事在我看来还不如你多学点我的课题内容，少给我添堵。

还有些细节我没在邮件里体现，当时发了联系双选的邮件后导师就问我什么时候方便去办公室聊一下（注意这是个陷阱，人家跟你客气客气你不能真就不客气，要服从对方的时间，不能大喇喇真的说一个时间），去之前我搜了一下导师发过的论文和成果，大概了解了这个课题组需要干什么。

我查到要做网页相关，聊天的时候我就说了我会node、JS、html(都是网页相关的编程技术)，我做过的项目有什么，用到的技术栈是什么blabla。

最后我们再对照题目描述里的问题

「做好实验后的数据不太确定是不是够好」

「一些实验操作并不是很清晰」

你这问的...数据好不好，怎么做操作...觉不觉得这些问题都太大了？

按照上面的思路给一个参考答案：

什么数据叫好，你查过没有，有没有数据源可以查的？

实验步骤是一代一代相传传男不传女的吗？

你能不能用辅助资料查了自己尝试一下，然后提问：师兄我查了步骤，我做了xx操作，但是却出现了xx现象，这跟原本估计的xx结果不一样，请问我哪里出了问题？

小问题尚且会觉得你添堵，更不要说把一个那么大的问题直接丢给别人，没有利益关系的情况下，别人没有义务为你解答。

**2023.04.03 13:18 二更 答评论问：搞这么复杂有必要吗？**

兑现一下更新的承诺，补两个方面。

一是跟导师联系的。

点名表扬评论里的高赞评论哈哈，补充非常精确到位，让对方做选择题而不是开放性命题。

后面熟络以后，是可以了解导师的喜好的，如果他喜欢繁文缛节，你就老老实实按照我上面说的跟他交流；如果比较随和可以后面慢慢把称呼改成X哥X导，这是另一个问题了，扯远了不展开。

我导师后来跟我说，当时我们年级有人给他发消息：我来你课题组做毕设了，我的联系方式是xxxx。

“怎么，你还要老师亲自加你？还有的同学啥都不说甩一串数字过来的，给我搞蒙了。”

当然，你要是手握数项专利，脚踩柳叶刀手摘SCI，你这样发导师也会夸你少年意气挥斥方遒，但要是没到这个程度咱们还是老老实实比较好。

二是有人说这么复杂没必要。

这个东西咋说呢...能上知乎的读者都是比较随和好说话的，大多数人（包括导师）是不喜欢繁文缛节的，导师也知道我的繁文缛节只是客气一下，但是为什么它依然是必要的？

还是那句话：跟你客气客气，你不能真不客气。

后面熟了我们勾肩搭背走，有事了直接wx嘀嘀不发邮件，这都没问题。

但是你一开始就这样，只会让人觉得你没礼貌——性格直率和口无遮拦不是一回事。

**2023.04.06 14:52 三更 答读者问：怎么跟实验室的人打交道？**

有读者私信我说，怎么跟实验室的人打交道？

一句话，明确你会长期出现。

不要拿那些虚头巴脑的来套近乎...我感觉这是很多人的误区。

咱们是老乡？

我一个市区几百上千万人，你哪位？

以后有机会一起出去玩？

大家都那么忙而且山高路远的，一起玩的机会有几分？

我以前写代码的时候遇到bug请教过一个大佬，解决好我是发了68.88的红包（问题其实很简单给68肯定够诚意的），大佬不收，我的原话是：

“可能对你来说这个bug非常简单，但确实减少了很多我的时间损耗，而且你的讲解真的蛮清晰的，我对这个知识点的理解深入了很多，你给我带来东西的远远远远超过这68.88，你别不好意思，反而是我觉得给少了。”

“而且我比较菜哈哈哈，以后可能还会有问题请教你，你不收的话我就不好意思找你了，给人感觉我在白嫖你的劳动力。”

“如果能花这点小钱让你觉得我是个值得结交的朋友的话，那我觉得这个红包的使命就超额完成啦。”

这就是明确想要建立长期联系的表达，再高冷的大佬和人物回到老家应该都会跟邻居客套几句吧？

长期博弈是纳什均衡的良方。

**2023.04.08 08:57 四更 答评论问：什么是建立长期关系？**

再更已经违背了我对于答主的回答要精简实际不水文的定义了，最后更新一次后此回答不再更新。

补一下上次讲的建立长期联系，但这个讲开了所有人际关系都通用。

最普遍的朋友关系一定是价值互换，比如我给你提供劳动，你给我付费，这是最简单的，所有能保持良好沟通的友谊一定是建立了长期稳定的价值互换。

别说什么你跟你有的朋友什么互换都没有啊？陪伴也是一种价值属性。

但是当你想要建立关系的那个人社会价值远高于你（也就是俗称的大佬），你没有能够对等交换的东西应该怎么办？

拿清北的计算机专业举例，假如你想认识学科负责人应该怎么办？

你说你积极上进？你端茶递水？

可是能积极上进端茶递水的成千上万，会缺你一个？

但假如你已经明确了保研到他名下读研呢？

假如你已经拿到了学院的教职呢？

我非常确定他再忙也不介意跟你寒暄两句，这就是确定建立长期联系。
