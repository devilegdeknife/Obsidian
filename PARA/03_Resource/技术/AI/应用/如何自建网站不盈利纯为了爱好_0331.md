# 如何自建网站（不盈利，纯为了爱好）?
[内容链接](https://www.zhihu.com/question/514217971/answer/2999225251)

域名、服务器、mysql、nginx、网络环境等！主要是安装数据库、前后端环境，还需要配置域名、https等等，设计的点还是蛮多的！建议学习下docker

（1）提供聊天大厅，即时群聊、畅所欲言

（2）提供协同编辑的云文档，支持6大常用文档（思维笔记、流程图、markdown、word、ppt、excel）
