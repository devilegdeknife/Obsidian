# 怎么看待B站舞蹈区和某些风格比较暴露的up？
[内容链接](https://www.zhihu.com/question/312810781/answer/3546850345)

给大家稍微讲一下这些擦边up主的一些发展路径




**一，创业初期**

这个阶段擦边up主都是在校大学生或者家里蹲的状态。每天拿着父母或者男友花钱买的手机，在宿舍/家/出租屋里拍拍大尺度的生活服装照，例如把衣服掀到露出南半球的地方，再摆个啊黑颜。再把这些大尺度的照片发到推特这种监管特别松散的平台，靠着平台自身的推送机制来慢慢涨粉。

当然，也有聪明的擦边UP主会自己开个小号跑到各种男人扎堆的游戏交流群里假装成好色的男人刚发现一个擦边的up，借机发自己的大尺度写真。感兴趣的小处男就会顺着写真上的水印找到擦边UP主的账号，从而实现给账号引流。

这一阶段属于资本原始积累的阶段，照片尺度非常的大，但又不露点。当然，不排除某些烧鸡开两个号搞大号擦边，小号露点这种双面人生的，例如前段时间翻车的小仓千代，被人扒出曾用网名笔芯优优米在推特上卖自己露点的裸照和视频。




**二，转型阶段**

这个阶段也就开始慢慢转型了，也叫上岸期，绝大多数的烧鸡都是转型去做游戏主播/虚拟主播，或者是做跟二次元相关的内容。游戏区男性用户的比例极高，不但可以保留绝大部分的粉丝，还可以继续吸引新的粉丝。

主要做法包括：

（一）删除自己曾经在各个平台上发布的大尺度擦边照片。

（二）删除自己曾经在各个社交平台上与政治意识形态有关的言论

（三）开始提纯粉丝，塑造**”我只是一个普普通通，喜欢游戏，热爱二次元的女孩子，并不是什么专职擦边卖肉的up主“**的正确记忆。




第二条里的与政治意识型态有关的言论主要就是支持女权主义与恨国言论这两大类。

跟各位讲一个笑话。张三一分钟前对着自己花钱向某个擦边up主买的大尺度写真撸过，一分钟后就在某条时政热点视频的评论区里面与自己刚撸过的那位开着小号的up主怒喷三百回合。




**三，上岸成功**

到这个阶段，全网粉丝量一般都是几十万起步。已经转型成功，可以在各大漫展的宣传海报上冠上知名coser的头衔了，出一次漫展一天就能拿1000块钱。开始学会好好穿衣服，cos的角色也从二次元媚宅烧鸡变成穿着保守的游戏女角色，即使是cos像艾达王这种也不会撩起旗袍露出大半个屁股了。

这一阶段收入已经非常的稳定，收入主要来源就是开直播、跟厂家接广告，去商业漫展赚出场费。收入相当可观，绝不是什么月入3000，还要996的苦命人。




**最后，我想强调一点就是，中文互联网上99%的擦边女coser，只有”明晃晃地支持女权主义“和”暗戳戳地支持女权主义“这两种状态。前者已经上岸成功，站稳脚跟，不需要靠男人的流量赚钱，可以肆无忌惮地公布自己的意识形态，后者还处于前两个阶段，得靠男人的关注来获取流量，增加自己自媒体账号的曝光度，因此还得忍气吞声的扮乖。**

**不做流量工具人的最好方法就是不要给擦边烧鸡打赏，不要点关注，不要点赞。杜绝一切让烧鸡靠性别走捷径的可能。**

那么该怎么做呢？

上策：掌握寻找荤菜的方法，彻底戒掉素菜成瘾的毛病，彻底摆脱对素菜的精神依赖

下策：在实在无法掌握寻找荤菜方法的情况下，把所有你感兴趣的素菜up主的主页链接收集起来，方法可以是放到浏览器标签页、收藏夹，抑或是创建个txt文档挨个标注、复制粘贴保存好，等哪天忍不住了就打开收藏夹，挑个感兴趣的，点进去看看有没有更新。

很多人可能认为“我只是点个关注，不给擦边up主打赏，不买她的写真就没事儿了。”

这种想法是错误的。

正文里我也提到了，擦边up主做擦边是实现资本原始积累的手段而不是目的，擦边up主的最终目的都是成为知名up主或者是上岸，而知名up主的必要条件就是要有庞大的、活跃的粉丝数，在圈子里具有影响力。这样才能扩展自己的收入渠道，才能去商业漫展站台，吸引商家上门洽谈广告合作等等，最终到达第三阶段——成功洗白上岸。每一个你忍不住而点的关注，都是在助力擦边up主走向她人生的小目标。同时，**你点的每一个关注，都会成为日后擦边博主洗白上岸后调转枪头攻击男人的底气**

最后强调一点，要**摒弃“她不拍了我看谁去？”的错误观念，树立”她不拍有的是人拍“的正确思想。**

有句话说得好——没有永远十八岁的姑娘，但永远有十八岁的姑娘

**是擦边up需要靠做擦边来生活，而不是男人需要看擦边来活下去。**小仓千代翻车后退网了，但很快又会有小仓百代，小仓十代出现，填补小仓千代空出来的位置。

**擦边这条捷径，真的挤满了人**
