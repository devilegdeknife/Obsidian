# 零基础玩转AI小车Jetbot—基于Jetson Nano
[内容链接](https://zhuanlan.zhihu.com/p/62501901)

## 前言

前段时间英伟达发布了一款开发板 Jetson Nano, 价格只需$99，相当于3个树莓派或者一个RK3399, 性能却完爆它们。

相对于树莓派，RK3399, NanoPi, OrangePi等开发板，Jetson Nano 有以下优点：

- 强大的GPU用于机器学习。这是最重要的功能，其他开发板所不能比的！
- 兼容树莓派周边设备。
- 支持TensorFlow, Pytorch等众多机器学习框架。
- 预装Ubuntu18, 内置Cuda, Opencv等，做机器视觉很方便。也可以作为ROS开发平台。
- 便宜。和TX1, TX2相比，Nano便宜太多，学生也能负担得起。
- 超小。核心板非常小，加上底板也只比树莓派大一圈而已。嵌入机器人更方便。
- CPU是A57（树莓派A53）
- 内存是4G（树莓派1G），并且Nano是外置内存，说不定可以升级为8G, 16G？
- 外置无线网卡M2接口，可接千兆无线网卡(树莓派的板载无线，那个PCB天线速度慢的一笔，怎么传4K视频？)
- 4*USB3.0(树莓派4*USB2.0)

总的来说，这款开发板物超所值，性价比非常高，树莓派能干的 Jetson Nano 也能干，树莓派不能干的事情，Jetson Nano 反而更擅长(GPU加速)，还有什么理由不入坑。

英伟达为了让更多开发者轻松入门 Jetson Nano, 开发Jetbot机器人，并在发布会上进行了演示。

## Jetbot 机器学习入门项目

Jetbot 是一款入门级人工智能小车，能够通过摄像头自主识别障碍物并避开，还可以进行人脸识别，目标检测等等。目前 Jetbot 项目在 Github 上有将近 600 个 start, 该项目也成为了Jetson Nano 入门的首选实战项目。

该方案和官方方案的区别：

- 该方案是一体式设计，开箱即用。官方是到处购买零散配件进行组装拼接。
- 该方案基于官方的是65mm轮子，官方推荐的是60mm轮子（不是通用的轮子，淘宝买不到，因此官方更新了65mm的模型）
- 该方案的电机驱动板集成了电源管理器，所以既能用充电宝给Jetbot供电也可以通过锂电池给Jetbot供电，官方只能通过充电宝供电。多说一句，电源供电重中之重，一定要选优质电源，Nano最低电压是4.75V, 并且在启动的时候消耗非常大的电流，而且对电源的纹波要求也非常高，市面上的5V2A电源不一定能满足。有些充电宝也带不起Nano, 我的20000毫安罗马仕就不行，垃圾。该方案的用3支18650锂电池串联12V供电（电压输入范围6-24V），在CPU满负荷下可以跑2个小时。
- 该方案带I2C转接板，预留各种接口，以后可以添加其他设备，官方不留接口。
- 该方案只在硬件层面改动，软件方面完全和官方一样。

## 组装一台 Jetbot 需要什么配件？

**必须的配件有：**

- 基础套件。包含了4个小车3D模型打印件，2个电机和轮子，电机驱动板，双弯头USB数据线，万向轮（球），OLED屏幕，I2C转接板，以及各种配套的线，螺丝等配件。有了基础套件，那么就能够组装出一台小车了。
- Jetson Nano 开发板。
- Raspberry Pi v2 摄像头外加一个160°广角镜头模组。Jetbot 目前只支持该款，其他摄像头和镜头一律不支持，购买的时候注意了。官方采集摄像头数据的时候使用的就是这款摄像头，训练出来的模型对摄像头和镜头有依赖，如果你更换了摄像头和镜头，那么跑避障程序的时候可能会出问题。除非你用自己的摄像头采集数据并训练自己避障模型，那么你可以使用USB摄像头和其他广角镜头。对新手来说，买官方推荐的吧，可以省去不必要的麻烦。
- 无线网卡。官方推荐intel 8265AC无线网卡，双频1200M，支持蓝牙。当然也可以用USB无线网卡代替。intel 8265AC需要拆开nano才能安装，并且天线需要固定，有点麻烦，好处是信号强网速高。USB无线网卡网速慢信号弱，但是使用方便，通常树莓派支持的网卡到可以免驱，但是有一些是垃圾网卡，千万不要买。比如EDUP EP-N8508GS，可以用但是网速只有几百kb，而且使用jetbot的时候经常断网。因此USB网卡可以作为备用。
- TF卡。Jetbot镜像非常大，需要使用64G以上内存卡才行。推荐闪迪128G高速卡，写入速度要90M以上。有客户开机一直黑屏，以为板子坏了，实际上是用了低速卡导致的，等了半小时屏幕才出图像。
- 18650锂电池或者充电宝。3节18650锂电池同时给电机驱动板和Jetson Nano开发板供电，可以运行2个小时。充电宝的厚度不要超过20mm, 不然塞不进jetbot。

**安装风扇**

注意：

由于Nano的散热器是不带螺纹，因此配备的螺丝要比散热器的孔大一点，并且要大力才能拧进去。如果你发现拧不进去不是螺丝不够长，而是力气还不够大（给金属攻牙能不用力气嘛）。有客户觉得拧螺丝麻烦，就用扎带扎。(英伟达估计是忘记给散热器攻牙)

## 软件：启动与基本控制

1.烧录网盘的镜像

需要准备一张64G以上的TF卡，推荐使用闪迪128G高速TF卡(黑色卡或者黄色卡)，镜像烧录方法见上一篇文章。

可以选择烧录官方63G镜像，也可以选择烧录钛创客的59G镜像。钛创客的镜像优化了一些Ubuntu系统的配置，Jetson Nano连接好网络之后，oled屏幕会显示IP地址，在你的PC电脑的浏览器上输入"**IP地址:8888**"(例如**192.168.1.110:8888**)就能看到官方提供的例子程序，点击执行按钮就能直接跑起来。
