# 神兔左皇，现在网络上谁占上风？
[内容链接](https://www.zhihu.com/question/1888324470759989370/answer/1892315029350621540)

这个问题充分说明了什么叫时代的局限性，什么叫人无法理解自己没见过的事物。

也直观解释了，为什么马克思和亚当斯密这几位能区别于别的学术大师




中文互联网上所有键政派别，讨论起一战的历史时都会觉得欧洲各国无比愚蠢，难道欧罗巴各国看不出战争结果会是两败俱伤吗？难道他们没有意识到打起来各国皇冠会掉落一地吗？

答案是，很少有人能理解自己没见过的事物

因为站在1914年各国王室的视角来看，你会发现局面完全是可控的。

俄国总动员了，但对面的奥匈也总动员了，法国动员了百万大军，可德皇麾下的军队也在快速聚集。

所有力量都在互相牵制，所有国家都有对应战场，所有可能的风险，参谋部都做出了详细的预案。敌人会从哪来，进攻方向会是哪里，德国的施里芬方案会有几个师，英法要在烈日要塞放多少防御部队，哪里会是反击的起点，翻阅地图，你会发现所有的危险因素，都被考虑到了。

但没有一个人看到，红色的幽灵出现了，在那之前没有人想过，鲁尔区的工人和西伯利亚的矿工能是一个共同体，但阶级叙事打破了一切。

欧洲的王室们习惯性的用民族叙事国家叙事去看待这场战争，俄皇想当然的认为，在东线崩溃之前，不会有力量能威胁到自己。

1914的时候你去和沙皇说，有那么一天，阿芙乐尔号会炮击冬宫，他会觉得你是世界上最可笑的疯子。

太可笑了对不对，

你说前线战事失利，有可能

你说战争会输，也有可能

但阿芙乐尔号上都是坚定不移的俄国士兵，是牢不可破的基本盘，怎么可能会背叛国家炮击冬宫呢？

但历史的答案大家已经知道了，

阶级叙事时代，前线的工人拿到枪支后，是真有可能调转枪口杀进东宫的。

你说背叛国家，首先问题就来了，你说的这个国家，是哪个国家？是国王那个，还是工人的那个？

欧罗巴的王冠由此碎了一地。




而现在，新的叙事再次出现了。不同于阶级叙事和民族叙事的新叙事体系，性别叙事。
