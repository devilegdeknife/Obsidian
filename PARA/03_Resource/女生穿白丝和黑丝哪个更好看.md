# 女生穿白丝和黑丝哪个更好看?
[内容链接](https://www.zhihu.com/question/1945274486619346867/answer/1982206627747746328)

我应该有资格回答这个问题，因为我以纯学术的态度研究过，前后历时两年，付出了很多代价，骄傲一点讲，研究得相当透彻。

我是个写小说的，大家也许不知道，写小说会遇到世上可能存在的任何问题。从核弹发射井选址的抗压超压到天基动能武器轨道部署，“白丝和黑丝何者好看”，是涉及到叙事美学核心的问题，和前二者同等重要。

只谈白丝和黑丝，笼统了。纤维粗细、织法、面料，失之毫厘差之千里，感受上完全不同，白丝黑丝不能一概而论。

光说D度，D度即丹尼尔系数，相当于透肤度。低丹尼尔丝袜极致通透，近乎透明，宛如第二层肌肤；30D-60D的中薄丝袜是最经典的区间，在能够均匀覆盖腿部同时，保留肌肤的生动感；而厚丝袜主打一个材质感，完全看不到肌肤颜色，从而更加凸显腿部线条。

D度之外，还有织法纹理，有平纹、包芯丝、网纹、竖线纹，面料有尼龙、天鹅绒、棉质、马油……

以上这些要素两两配对，再加上不同的自然光线：强光、柔光、人造暖光、霓虹动态光、黑暗中、黄金的时刻……会形成截然不同的视觉效果。

除此之外，还有勾丝问题。勾丝的位置，不同D度的勾丝表现，撕口，破损度，还有浸湿后的视觉表现……同样也是各具不同风味。

总而言之，这里边儿全是学问。

我当时是越研究越震撼，越思考越心惊，这可不是上网查查资料就能研究透的学问，最后干脆买了几条，自己试着穿了一下。

这一穿，才深深体会到文字资料给不了的冲击。我发现，姿势不同时，二者的感受也截然不同。

蹲坐时，膝盖处会产生褶皱，黑丝如深渊波纹，神秘感中带有诱惑力，白丝则如石膏像衣纹，有种不真实的僵硬感；脚背绷直时，白丝清纯中有种好欺负的娇弱感，而黑丝则显得有些廉价；行走中，黑丝和白丝在膝盖后方都会形成不同程度的面料堆积，黑丝比较难以发现，白丝若是超薄的，会特别反差。

遗憾的是，我是个男的，小腿较粗，完全无法发挥白丝的实力。

这时候我只能求助于万能的读者了，当时在读者群里嗷了一嗓子，说能不能帮我找找丝袜照片，我写作要当素材。

读者自然不理解我的艺术追求，对我一顿嘲讽，但是也有很多群友踊跃出手，我当场收了一堆丝袜图，其中绝大多数，是足控大喜的福利图。

在观赏了2个小时后，我得出结论：看这玩意儿嘛用没有。

我不知道为什么，网图的丝袜照片一大堆，个个都精致得像艺术品，但很少具有我在路上看到那种有活人感的，甚至还不如穿我自己腿上的呢。

那时候我就明白了一个道理：艺术的冲击力，雕琢不如真实。所谓清水出芙蓉，天然去雕饰。把丝袜当成一种饰品，和把丝袜当成一种袜子，这是两个概念，后者才具有拳拳到肉的生活感，直击灵魂。

丝袜穿在腿上，需要被行走，被摩擦，被挤压和拉伸，堆积、滑落、浸湿汗液，这样才履行了它原初的使命。

就好像一个人，上班当牛马，工作，赚钱，被老板呼来喝去，回家被老婆提款，他是会逐渐被异化的，他会感觉到丧失了自我。人的本质是动物，丝袜的本质是袜子，动物需要面向自我与世界狂奔，袜子需要被真实地践踏，这样才能找回初心。

而且重要的不是被使用后的状态，而是**作为袜子被使用本身**。一个行动能够为物品赋予意义，看似别无二致，实则天差地别。

就好像写小说本身，我究竟是在享受写这个过程，还是在享受写完它之后赚到的钱？可能兼而有之，但其实写小说根本不用这么累的，用这种句句推敲的态度写小说，属于自己给自己找罪受，读者也感知不到。

穿袜子也不用这么累的。穿丝袜把穿袜子这个举动仪式化了，超越了保护足部本身的意义。

但正是仪式化，才让连穿丝袜本身也有了温度。写小说和穿丝袜一样，都是通过折腾自己来取悦别人，你充满仪式感地去做，充满真实感地去经历，文字带不带温度，差距是很大的。

这一点连上班当牛马也同理。工作让人痛苦的原因，不是因为工资低，也不是因为太累，而是因为没有价值。

如果自己的工作有价值，能够切切实实地看到世界被改变，看到受益者的笑容，我想，哪怕是当牛马，累一点也会很快乐。这是马克思主义预想的未来世界的理论根基。

丝袜，写作，共同理想，大家都需要自我实现。

可惜的是，当时我只是个小作者，读者群人数不过两百，又是男频，女读者一个都无，我对于丝袜的研究只得告一段落。

两年后，我的读者群人数已经两千，某次冒泡，看到群友在聊丝袜，我不无感慨地说了一句，其实我曾经研究过丝袜，只可惜没有机会去观看实际作为袜子的丝袜。

没想到过了两秒，有个群友私聊我，说，亮亮，我有很多丝袜腿照，你要不要我拍给你看看？

我说，啊，这，不好吧？

仗义群友说，你不是要研究吗？发给你发给你

我说，这不好不好。

其实我超想看的。但这个事不能这么干，这不是显得我借着研究之名，实际勾搭读者？传出去了成啥了？况且看了群友的腿照以后在群里如何相见？这根本不纯粹！

仗义群友说，别犹豫了，我是想发给你看才给你发的，没有这次，我找到别的机会也会给你发的。

说完，人就发了十几张照片过来。

有白丝有黑丝，没修图，没美颜，腿很细，背景很简陋，全是朋友圈级别都够不上的私拍，一看就知是拍着玩的。

但是满满的活人感。

我一时灵感轰然涌现如雪崩，这就是我想要的真实感。

丝袜是一种袜子，它不应该被当做赏玩道具，用观赏的目光去看，实际上是对丝袜的一种暗贬。丝袜虽然并不伟大，但它应该作为袜子，骄傲地存在于世界上，堂堂正正，不媚流俗。

在这种强有力的真实感冲击下，不管白丝还是黑丝，都很可爱，超可爱。

我跟群友说，有条件再拍几张吗？

群友说，为了你，可以。

我说，为了小说。

群友说，为了你的小说，可以。

我说，我想看看雨水从不同D度丝袜上，顺流而下的痕迹。

仗义群友说，呵呵，你有点过分。

我说，可以用莲蓬头模拟雨水。

群友尝试了好几条，花了半个小时，忙坏了。但是很值得。

除此之外，我还提了不少过分的要求，比如躺床上勾起一只脚，穿过膝袜坐凳子上用大腿摊大饼，等等。

看完了这些之后，我说，今天的感激之情无以言表。

群友说，都哥们儿，小事。

我说，不过，结论可能会让人有些失望。

群友说，你说，失望也想听。

我说，虽然白丝和黑丝各有优劣，但最重要的是，有个女生愿意不出于任何功利目的为你穿黑白丝，这才是清凉世界最温暖人心的疗伤圣药，超越性欲意义。

我又说，这个结论是不是很无聊，有种虎头蛇尾的感觉？

群友说，没有，我觉得很有道理。

群友又说，但是，哥，我是个男的。
