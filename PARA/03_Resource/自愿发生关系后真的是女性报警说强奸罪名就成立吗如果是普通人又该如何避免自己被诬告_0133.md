# 自愿发生关系后，真的是女性报警说强奸，罪名就成立吗？如果是普通人又该如何避免自己被诬告？
[内容链接](https://www.zhihu.com/question/1896278314173317963/answer/1960786977482863480)

鲍毓明案

鲍毓明是在中国上的本科，学的工科，后来是管理、工程硕士，然后工作的时候自学通过中国法考，拿到律师执照，然后以法律专家的身份留学美国拿的计算机硕士，同时通过美国法考，拿到美国的律师执照，那个时候还是九几年，归国后是知名律所合伙人，上市公司的高管、独立董事，最不服的知乎er我想都得承认人家是精英中的精英。

身份从回国后就是引进的外籍专家，中兴聘请他的时候公告名字是用的他的汉语拼音名，括号里才是汉字名，懂得一看就知道这人是外籍，这也说明他从来没隐瞒外籍身份。

![图片描述](https://picx.zhimg.com/v2-86a9a422b7c084f6d0071ab08ba0dfc6_r.jpg?source=2c26e567)

与前妻离婚后，发帖想要收养一个孩子，在韩婷婷找上他的时候，他说对方年龄太大了，他想收养一个幼儿，且并没有要求性别

![图片描述](https://picx.zhimg.com/v2-ef48d44d38bea5aeaa43e6bec874470c_r.jpg?source=2c26e567)

在韩婷婷一方诉说自己的可怜之后，双方见过一面，并未发生关系，但鲍毓明选择资助其“上学”跟生活

之后在韩婷婷的多次主动，且在她的“母亲”明确支持下，双方发生了关系，并带其回家见过父母，在老家亲友见证下举办了订婚，承诺女方年龄一到就办手续结婚

此后双方总共共同生活时间只有几个月，且按照女方虚报的年龄，也已经过了16岁了，这里也要补充说明一下，韩婷婷本人从一开始就没有“少女”的样子，她都不符合小红书幼态审美的标椎，且本人样貌、身材普通，以我个人的审美标椎，说鲍毓明是恋童或见色起意，我是觉得有些亏心的

一个精英高管，妥妥的高净值人士，我认为不至于

韩婷婷在事情闹大前，就已经有多次在多地报假警的记录，甚至还有一次鲍毓明在警察的见证下写过一份保证书，对，你没看错，报假警的是韩婷婷，鲍毓明是去领人的，但保证书是他写的，这份保证书就是后来攻击他的“罪证”之一

在放出来的录音中，可以发现，韩婷婷在异地拒绝鲍毓明去看望，而鲍毓明除了嘱咐学习，还嘱咐其注意饮食营养，并在多次表示“还有事，要挂电话”的情况下并未主动挂断电话

鲍毓明有开启通话录音的习惯，留存了大量通话录音、聊天软件的聊天记录，即使如此，从调查到最后通告也持续了半年多

在整个过程中，未发现其有任何同事、朋友传播过对其不利的言论，大家经常见到的“某某某我知道，就是我们公司，为人xxxx”之类的传言都没有

韩婷婷所说的谎言极其拙劣，有大量证据证明其撒谎，比如她谎称被强奸的日期实际与鲍毓明分处两地

最后的结果是什么呢，鲍毓明因美籍持有中国的律师执照，而被遣返，韩婷婷自此再无消息

这里面还有两个有意思的点，中国当时并没有明确规定美国国籍不允许拿律师证，但律师证需要通过法考才能拿，而法考不允许外籍参加，但在这个规定之前，鲍毓明就已经通过法考了，严格来说，其并非隐瞒国籍才获得的律师证。

第二个点是韩婷婷最后一次报警是在南京，而南京警方电话质问烟台警方为何不立案，为何不给回执，在烟台警方说给过回执以后仍然不信，说“她说你们没给”，那么我的合理猜测就是，南京警方见过或者说相信同行能做出来违规不给回执的事，至于他们自己做没做过，就不知道了。

那么回到问题，如何避免被诬告，这里不仅仅限于强奸，我能想到的就只能是闹大+对方有明显破绽+自己有无懈可击的证据证明自己无罪。鲍毓明要是失去了其中任何一条，结局能不能这么“好”，都是说不定的事
