# 你的低成本爱好是什么？
[内容链接](https://www.zhihu.com/question/1897959255778231029/answer/1949489013611271199)

当身边的人都在讨论，周末要去哪里露营、要去哪家网红餐厅打卡时，我常常会觉得有点插不上话。

我的爱好，说出来，好像都挺不花钱的。

但也正是这些几乎零成本的爱好，却在不知不觉中，治愈了我大部分的精神内耗。

**1. 公园里的捞鱼捞虾**

我有一个小小的折叠渔网和一个透明的塑料罐子。周末天气好的时候，我就会带着孩子，去公园里的小河边，捞鱼捞虾。

我们不求捞到什么大鱼，只是享受那个安安静静地，坐在岸边看着浮漂，等待着惊喜的过程。捞上来的小鱼小虾，我们会带回家，一部分，给家里养的小乌龟当“大餐”；另一部分，就养在我那个，小小的鱼缸里。

**2. 大自然的珠宝匠人**

秋天，是我最喜欢的季节，因为对我来说那是丰收季。我会提着一个布袋子去公园里，把那些掉落的无患子、棕榈籽、榆叶梅核，都当成宝贝一样，捡回来。

回到家清洗打孔，再用便宜的绳子，一颗一颗地把它们串起来。这个过程非常治愈。看着这些来自大自然的、免费的原材料，在自己手里变成一串串，独一无二的、充满了时间温度的手串，那种满足感，是任何珠宝都给不了的。

**3. 阳台上的菜园**

我们家厨房里，吃完的小葱头从来不会扔掉。我会把它们带着根须的部分，插在一个装了水的杯子里，放在阳台上。没过几天，又能长出，新的嫩绿的葱叶来。

还有那些，不小心发了芽的红薯。我也会把它一半浸在水里。它会长出非常茂盛的、可以吃的红薯叶。那就是我们家餐桌上，最常见也最健康的一道青菜。

**4. 窗台上的迷你森林**

夏天吃完的桂圆、荔枝，秋天吃完的枇杷，它们的果核，我都舍不得丢。

我会把它们，仔细地，清洗干净，然后，放在一个浅浅的盘子里，用水，把它们养起来。

过上一段时间，你就会看到，一颗颗小小的、嫩绿的芽，从那坚硬的果核里，倔强地，钻出来。看着它们，慢慢地，在我的窗台上，长成一片，小小的、迷你的森林，我觉得，特别美好。

**5. 河边的收藏家**

下雨过后，我喜欢去河边散步。

我会在那里，寻找那些，被雨水，冲刷得干干净净的、形状奇特的鹅卵石。每一块，在我眼里，都是独一无二的艺术品。

我还会，从水边，捞一些，绿油油的、不知名的水生植物。把它们，和捡回来的石头，一起，养在我那个，大大的玻璃缸里。

**6. 公园里的植物学家**

我不仅捡果实，也学着去认识它们。

秋天在公园里，可以捡到很多发芽的小苗，银杏树，红枫树，鹅掌揪 ，都是小苗，拿个小铲子，零元购，不要太爽了哈哈哈

在楼下小区转一圈 发现紫荆花也有小苗 棕榈树也有小苗 甚至是枇杷树都有小苗 还去啥网上买植物啊，都是免费的，不然被物业当做杂草拔掉了，直接放在花盆里面养，惬意...

这些爱好，几乎都不需要花钱。

它们需要的只是一点点，愿意弯腰的耐心，和一双愿意看见的眼睛。

它们让我从对物质的追求中解脱了出来。 也让我在这个浮躁的世界里，找到了能让我的内心真正，安宁下来的东西。
