# 你和你老婆是怎么认识的？
[内容链接](https://www.zhihu.com/question/68455714/answer/1902369871574041582)

研究生的课程百无聊赖，坐我旁边一妹子肩披长发，头上飘来洗发水清香，身板挺直端坐着，课间，我正赶着其他课程的作业，有一些参考书，她顺手拿起来看了看。我心中一惊，莫非此人我认识但我不记得名字，不然她怎么这么自来熟。便问她作业做了吗？她眼神明亮，自带笑意，看着我说没有，就是看一下。我说“哦”。下一节课，我心思便不在课上了，此女孩越看越喜欢，不做作，落落大方。突然，有个很奇怪的想法，此人要是我老婆，多好。从来没有过的结婚的念头的我，自己被自己吓一大跳。再一次课间，忍不住问了下她的名字。名字很怪，但是让人记忆深刻。她问了我，我在草稿纸上写下了我的名字，她说，你，就是上周在舞台上唱歌的那个人吗？我没想到，她居然先知道了我的名字。我心中满是高兴，但是仍不露神色。说是的，你咋知道的，她说她听她室友讲的。好吧，挺好的开端。没想到这厮回去就和寝室的人吹去了，说今天见到了表演者本人，是活的本人。

没想到有人看，本来就是记录下，还有人催更，还说我连生产队的驴都不如，是可忍孰不可忍，我再继续写。

分割线，接上上段。————————————————————————

为啥我知道她和她室友吹去了呢，第二天在另外一门课上，她和她室友坐一起，坐在我前面，课间转过头来，和我打招呼，她室友也跟着回头和我打招呼，她介绍说这是她室友，我有点措手不及，她室友补了一句，你就是那个谁谁谁，在台上表演那个。我脸皮带动着肉，挤了个不失礼貌的笑容，向她室友点了点头说，我就是，心里却是乐开了花。看来，我还是有一点名气的，虽然自封是十九线小艺人，但是起码粉丝终于有两个了。

课后，问她，你昨天晚上回去给你室友讲了我啊？今天终于指认了现场？她说，是啊，我好不容易见到真活人，不得回去宣传一番。我说，你只是看到了我一方面而已，唱歌只是我的不务正业的爱好，我真正好的是学习，你本科不在我们学校你是不知道的（她从外校考过来的），我成绩可好了，年年都有奖学金。

其实我说这话虽然不假，但是还有点心虚，本科班上24个人，有6个人有奖学金，只有第一名是一等奖学金，而我从来没拿过。结果她翻了翻白眼，你咋不把这天花板吹到天上去。我一时噎住了，从来没见过如此怼人之人，刚刚想建起我的光辉形象来，她就一个态度，我不信，让我无语凝噎内心抓狂。当时突然就很理解鲁豫有约里那些受访的明星，讲了一大堆，鲁豫只一句“真的吗？我不信”，就能把这些明星气得脸色一会白一会红，而且还得保持风度。

2025.5.5 13:48 突然这么多人催更，啥情况。搞得我上午开高速的时候，看到这么点赞的信息弹出来，亢奋得一匹，大家别催了啊，这不一下高速就赶紧给各位更新来了。在各位的回复中，有人问，你老婆最后嫁你了吗？我哑然失笑，最有意思的是，知乎还特地单独在这条回复下，给了我几个选项，让我选择此条回复是不满意、一般还是满意？我当然是满意的，你这气不到我的。毕竟我和这妮子刚认识的时候，我给她起了N多外号，其中一个就叫噎人大王，她不止对我，对其他人也是，就是你说着说着，她听不下去时给你来一句，让你没法往下接，这么多年，我已经练的刀枪不入了。还有人说，这像是哪里的小说，回想一下，是有点哈，只能说，艺术来源于生活。

接上上段，我顿时有点抓狂，本来想乘着氛围都烘托到了，多给自己按几个闪光点，先把你晃得睁不开眼再说，咋你怎么就不信我？我长的那么不可信吗？还是你觉得一个会唱歌的人，不会好好学习？你个假粉丝！我得回去拍一下我那些奖学金证书，把照片甩给她，让她看看，这是吹牛吗！？但是突然冷静下来，我这样做是不是太幼稚了，她就这么这一句话，我就表现的急于解释，会不会显得我太在意她的看法，还显得我很不成熟。我得沉住气，现在才第二次见面，让她多了解了解我，她就知道我是不是真学霸了。

我给她说，你不信我算了，你后面就会知道的。她说了个字，切。我问她，明天张老师的课，你去不去蹭？她说，我去啊，大家都去，为啥不去。我说，那你去了，帮我占第一排的座位。她说，座位这么难占，不一定占的到。我说，那我给你占好了。她说，好啊。我内心沾沾自喜，此乃一箭双雕之计。坐第一排说明我是好学生，你看哪个不爱学习的人会去老师眼皮底下听课。第二就是，嘿嘿我又可以和你同桌了。

话说我们想去蹭课的这位张老师，是位男老师，实际上并不是我和她那个教学班的教学老师，我们的教学老师，也姓张，是位女老师，他们两同时担任我们这门课程的主讲老师。为描述方便，后面就称男张老师和女张老师。昨天就是在女张老师的课上，我遇见了让我突然有结婚念头的她。而男张老师是我们学院的教学副院长，教课有两把刷子，从不带教材，所有课程内容都记在脑子里了，讲的通俗易懂，深入浅出，女张老师上课风格和他相反，上课习惯拿着教材抄一大堆公式，还解释不清公式之间的来龙去脉，经常听得我们一头雾水，这门课是一个4学分的必修课，大家都很着急。于是便出现了这么一个壮观奇景，两人的课安排到同一个时间段的话，女张老师这边到上课点了，门可罗雀，然后上课铃响后，陆陆续续会有从男张老师那边过来的学生，因为去的晚的，没占上座，只能来女张老师这边上课来了。后面男张老师自己教的学生强烈抗议，因为他们也上不到男张老师课了，去的晚没座了，这件事还在学院闹得沸沸扬扬，后面学院只好单独搞了间更大的教室给男张老师上课...这是后话，暂时不表。

说了这么多男张老师上课的火爆情况，相信有人猜到了第二天的结局。你们猜的没错，由于我之前没蹭过男张老师的课，加上我过于自信，上课前半个小时才到教室，进门着实让我震惊了一把，还有半个小时才上课，就已经黑压压一大片人。而第一排有些座位虽然没坐人，但是每个座位上都有书或书包，有人甚至把电脑扔在桌上，人不在，电脑屏幕上写着“此座已占”。我向相熟的同学一打听，说第一排的座位，就甭想了。昨天就占完了。这电脑昨天下午这个教室别的老师上完课后，就有人拿过来占着，刚才有人过来把电脑打开，上面敲了这几个字就走了。还有过分的，还有个座位就一张白纸，上面打印了几个大字“占座专用纸”，用透明胶贴在课桌上。

我用我的钛合金眼快速搜寻我的长发大美女粉丝，看到她在倒数第二排坐着，正低着头在赶作业，我一声不吭，赶紧默默退出教室...出师未捷...

看评论也是醉了，“匹夫，速更！”，“驴，更！”，各位大哥大姐，别催了，催的我心慌啊，白天做牛马，晚上还得做驴，真的是。有人建议设置为追更模式，这样更新了也能及时知道，说实话，本驴混迹知乎十多年，目前唯点赞较为擅长，这个追更研究了下，暂搞不定，不几道是否是因为级别不够不让设置，实在是抱歉。还有人建议弄成付费的，算了，付费了，你们不得更加理直气壮的拿鞭子天天抽驴。

接上上段，退出教室的我，站在门口，一时间有点愣神，牛吹大发了，在证明自己是学霸的路上才抬起脚，又被狠狠地摁了下去。我的大美女粉丝肯定是以为我是给她开玩笑的，或者会不会以为我已经给她占了第一排的座，只是人还没来？

我想了想，掏出手机发信息给她，“第一排那张贴着‘占座专用纸’的座位，看到了不，是我给你留的”。她：“真的假的，你这惹众怒了”，我：“逗你的，那不知道是哪位大神干的，我刚才来过，来前不知道现场情况，没想到现场这么火爆，到的时候已经没座了，不好意思啊，没给你占到座位”。发完老脸通红，人还是要诚实的，不然撒一个谎，得用十个去圆。

她：“没事，我已经有座位啦”，我：“我是第一次来蹭课，还好你占到了座位”，她：“我也是我室友帮我占的，那你下次加油咯”。还好我的长发大美女粉丝对我的境况表示深刻理解，而这句加油，让我出师未捷的悲壮感消失得无影无踪。

我：“晚上你在哪自习？”，她：“图书馆”，我：“你吃完饭就去吗”，她：”我吃完饭，回寝室后，再出发“，我：“好的，去的时候给我说一声哈”。她：“我约好了和我室友一起呢”，我：“没关系，喊我就行”。学霸人设要想立住，不一定只靠占座嘛，哪有学渣爱自习的，除非，自习室里有喜欢的人？

等吃完饭，过了饭点，她发信息来，说她们出发了，我说好嘞，她宿舍楼就在我宿舍楼的隔壁，我坐电梯下来，未看见人，估摸着是已经走到前面去了，便一路狂奔，远远看到她和五六个人在一起，正在我们学校的济海湾边，在时光雕琢的鹅卵石路上嬉笑着踱步缓行。我放慢了脚步，凝望着她，济海湾里的摇曳的秋荷、岸边郁郁葱葱的松柏、小路上来来往往的行人，仿佛都是为了陪衬她而存在。一路跟随，慢慢靠近，和她汇合在图书馆前的枫树林中，和她打了招呼，也和她的室友们打了招呼，她的室友们带着惊奇的眼神看着我，刚入秋，枫叶边沿已出现一圈红晕，在夕阳的照射下泛起光的涟漪，恰如她的颜，我眼中其他的事物在慢慢消失，我的世界里只有她...
