# 为什么国内的大学越来越高中化了?
[内容链接](https://www.zhihu.com/question/653788710/answer/1955576952690816191)

宝宝张嘴乖乖吃饭——电影不分级，色情业成为禁忌，全年龄十八禁。

宝宝有事情就举手——警察得干居委会的活，领导还得管你家长里短。

宝宝间要好好相处——同事间私交越来越少，团建文化玩小朋友游戏。

宝宝们要小心坏人——那么谁是坏人了？老师说了算，是为了大家好。

逐渐的，社会就是个幼儿园，何况大学呢？什么高中化？多是小宝宝。

在我幼稚园的时候，殴打捉弄我的男同学到嘴破皮流血，家长都不用叫。我们不会被分辨谁对谁错，谁该道歉谁得被批，只叫小孩子们闹着玩没分寸，不能握手言和那也不强求，于是小男孩们不再敢揪我头发。我也就没学会哭着找老师告状，找大人做主。那是90年代初。

在我小学的时候，不存在家长们扒着栏杆等放学。一年级时，老师已经把住的近的同学都编好组，让大家带着小黄帽，列好队伍回家。过马路要先举手，不能跑，不能并行走，叮嘱队伍头尾最高大或懂事的孩子管好小伙伴。我那很繁华，车很多。每天一群小朋友涌出学校，就和小鸭子一样排着队唱着歌，列队回家，可可爱爱的，可彪悍本质已经内化了。小学高年级的？早自己骑单车通勤了。那是90年代中。

在我初中的时候，遍地混混。学霸们豪气值也不是盖的，个个敢拍桌子。学校专门开辟了个僻静小楼放个班，装另一种比较刻苦，正常点的学霸，给他们安静单纯的环境，连路都尽量和我们分开。而我们这批天马行空的，总搞出乱子，也不会被扣处分，自由度给满。我们这批，上课热闹似菜场，一激动来个集体逃学，但学校就是默许了。乖巧秩序的，和叛逆个性的都能开心过，学力值最终差不多。这才叫做因材施教，素质教育。那是00年代处。

在我高中的时候。教师们已经偏向希望学生是只听指令的机器人了，有很多小事上的服从性测试，确认自身权威性，学习上更要求全部跟着自己的计划和思路来。但我校领导层，又还放不下素质教育，主观想开发学生的个性和自由性。于是很分裂。一边是上课学生喝个饮料都能被骂几分钟，死抠吃饭时间，搞出傻子一样的大清早边跑步边喊话背书模式；一边是鼓励建立各个兴趣社团，街舞都可以，早恋公开遍地走，两男的操场拥吻也不是个事。我是很叛逆的，都觉得太癫了，很空洞无物的癫，眼前尽是魔幻现实主义。管制和自由的负面，全占了。那是00年代中。

大学，没啥好说了，“做题家”的天下。“做题家”释放后，哪怕觉得自己很疯狂了，也还是一股子乖巧摸样，尽做为了证明自己疯而疯的事，活在模式里。

我从初中起，就混迹大学城，兜售二手手机的。00年代初，大学生还和我砍个精明的价，要求拆机壳看，要个质量保障，不懂就问不会觉得没面子，也不会觉得我是个小孩子就怎么了，生意是生意，很利落；到00年代中后期，我开卖二手电脑及硬件了，男孩子们大多害怕自己动手会搞坏了；或者要鼓气质出一副胸有成竹的样子，然后瞎搞，涂个硅脂都能出事。真是卖个白菜的钱操着白fen的心。探索冒险精神也越来越微弱，我见了鬼了总得给男生装系统，互相都挺不好意思的。而女生？想走个拉近感情不给钱，一次购买后续免费服务，烦的要命，我见了鬼了总被拖手起哄撒娇，撒娇能抵钱吗？被撒娇浪费的时间也是钱。都是得哄的宝宝。学校什么都教了，却没教契约精神才是最高效的，也没教自己动手丰衣足食。

问题是从00年代中开始出现的。

人本来就能分两种，秩序的，叛逆的。秩序的是大多数。

给秩序的提供守序环境，给叛逆的提供自由环境，从性格本质区分，而不单纯从学习成绩区分，和我初中那样，就顺畅了。我初中有点反倒天罡，学霸都叛逆占多，校园风气就是癫，和占半的混混群体交相辉映，互相加持后更炸裂，要不是都爱我校给学校面子，且脑子聪明的能预测后果会克制大家，燃烧瓶都快玩起了。所以秩序的那批被小心翼翼圈地安置了。如果把那群秩序型的学霸放我们楼里，边上就是抽象疯子和大群混混，他们肯定五味杂陈，没安全感，读书会分心，压力太大时也容易学坏；如果把我们这群脱缰野马拉到僻静小楼，没的造，也得憋疯了，逆反心上来就得作妖不搞学习了。我校真做到了，让两种学生都出学霸。可这批叛逆学生，从某个层面看，难道不是危险分子？还有兵有将成系统，仿佛模拟动荡里的战争与和平。

真正的素质教育时代结束了，全面分数至上时代到来。可也许觉得那样太呆板了，后来逐渐走成了：希望秩序的开发个性面；又希望大棒压制叛逆的。这不是没事找事发疯吗？如同我的高中。于是秩序的假装有个性，也以为自己有个性，乱找邪门榜样，越远离本性就越空虚，越空虚就越想释放。而叛逆的读个书和坐牢似的，不如不读算了。

完全的分数至上后，我们的升学制度，本来就是个服从性测试，注定越到高处，叛逆的越少。那么自然，大学里装的人，本就是秩序者为主流，而不是大家理解成的独立个性遍地走的自由学府。

本就秩序的，被要求宣扬“个性”，也是一个被抹去自我的过程。明明秩序本身也是一种美好，结果搞成秩序学生为了叛逆而逃学，为了标榜而早恋。每个生态位都会有人占据，叛逆学生没有了，就得分出一部分秩序学生去扮演叛逆。有看似叛逆的，大棒就不会停。而秩序学生本就不像天生叛逆的那样爱掀桌，没人反抗，于是大棒越来越沉。

秩序学生本身并不是乖宝宝，如果给个好环境，就是亦步亦趋，逐渐成长的务实类型。按照道理，最后会比本性跳脱的叛逆学生更稳重，更成熟，更有担当，是越走越开阔的。

可为啥都逐渐宝宝化了？

宁可错杀一个，不可放过一千。只要全部压成幼儿园，反贼含量就为零。自以为一身反骨的，也不过是“扮演叛逆”的，小孩子过家家而不自知。

真正的叛逆者，要么被剥夺了继续升学的权力；要么在高压模式卷学习的时候就已经傻掉和抑郁了。剩下零星的，更是难以大面积聚头，且也是叛逆者里最聪明的，看得懂大势，能适应新模式隐藏自己。如果高校里，一大堆叛逆的聚头了，各种奇异思潮会铺天盖地，而不像现在的那么王八念经般的，钻进三瓜两枣里，车轱辘无意义。

为什么原子化了？其中一个原因，是叛逆者最有号召力，本身该作为领引者开创者的角色。凝聚剂全没了，可能性也被提前抹杀。

另一方面，谁又会不觉得我们很团结呢？最伟大的时代。

原子化是公认的，团结也是公认的。这才是最魔幻的。

让原子化又团结的方法，就是每一点一滴里，都给套上正确模板，多放几种模板，就看起来丰富了。看起来很丰富，难道不是自由？最巧妙的思想钢印，是人不会去想象，也不会去模仿，完全没见过的东西，比如真正的叛逆者。
