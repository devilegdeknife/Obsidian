# 如何评价谷歌正式发布的 Gemini 3？哪些信息值得关注？
[内容链接](https://www.zhihu.com/question/1974260216498500253/answer/1974433378829875126)

我之前做自动驾驶的时候，总结了4道关于点云网络PointNet的题目用于面试候选人，PointNet最经典所以做点云的人都看过，它的结构也最简单所以具备一些有趣的数学性质可以用来进行面试。面试时能正确答对全部4问的，在我这一关算法面试都通过了，而且会在面试评价里强烈推荐。

这四道题一开始Chatgpt 3.5只能答对第一问，第2、4问需要再给一定的提示才可以答对。后来渐渐的各家AI都不再需要提示，但第3问一直答不对，给了提示也不行。

终于，Gemini 3能够答对全部4道题目了，所以我的这几道面试题也没必要再藏着掖着：

> 1. PointNet是如何实现置换不变的，即随机打乱点云顺序，输出保持不变？
> 2. 我有一个训练好的PointNet和一张桌子的点云，现在我想找到哪些点是比较重要的。换成比较数学的描述就是我想求得桌子点云的一个子集，使得这个子集经过这个PointNet之后得到的特征跟全部点云得到的特征要一模一样，请问该如何获得这个子集？
> 3. 我想知道PointNet提取的特征有什么物理意义，比如池化后的特征的第一个维度的特征代表了什么物理特征？我这里有4个选项：A. 代表了某个局部的形状 B. 代表了全局的某种形状 C. 代表了某个局部有没有点 D. 代表某个局部点的密度。请做出选择并说明原因。
> 4. 我现在想找到空间中的一个点，这个点的坐标xyz输入PointNet后，能够使池化后的第一维特征达到它能达到的最大值，请问该如何找到这个点？

第三问确实比较困难，所以我一般都会给4个选项，而且跟候选人说可以使用排除法来做这道题。其实如果仔细看过PointNet论文，见过这张图，应该能意识到答案是代表某个局部有没有点：

![图片描述](https://pic1.zhimg.com/v2-493417286448cc3397080891e69b2de3_r.jpg?source=2c26e567)

按理说这些AI都是读过人类所有论文的，但可能之前的AI真的都只是“读”过，是将论文转化为了文字来进行训练，而没有看图的能力。

看Gemini 3的回答：

![图片描述](https://pica.zhimg.com/v2-b32be61fd95ebc64dce42f25e22f060d_r.jpg?source=2c26e567)

它提到了“在论文的可视化部分也有证实”，说明Gemini不只是“读”了论文，它跟人类一样“看”过这篇论文，所以能够答对这道题。

原生多模态，而且我猜是从训练之初就是多模态，才会有如此强的图像理解能力，这可能是Gemini 3有别于其他AI模型的关键。
