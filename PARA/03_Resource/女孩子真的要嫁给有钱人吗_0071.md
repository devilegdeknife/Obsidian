# 女孩子真的要嫁给有钱人吗？
[内容链接](https://www.zhihu.com/question/292422462/answer/1898441750844014653)

之前有个小兄弟，他家流动资产有个2000万吧，然后自己有个上海双学区电梯三房，大概1500万的样子，没贷款。父母的退休金应该有个2万左右，在三线城市还有个4,5套房子/商铺，父母早年似乎在上海还有1,2套房子。自己是C9的土木本，然后跑到澳洲读了个CS的硕，入职不累的中厂（踩到的就别评论了，我不会回的）。

然后处了一个江西在券商的妹子，这妹子学历一般吧，全靠熟人关系进来的，家里农村的，收入在30-40W的样子。这个妹子快领证前，突然拖沓了一下，关键婚礼延期不影响领证啊，然后突然和周围的人借了500多万，外加自己攒的100多万，凑了700多万，买了个1800万的房子，只写自己的名字。

然后这个小兄弟的父母知道了以后呢，就以老家要修管道重新装修为由，搬到了这个小兄弟这里住3,4个月，然后做饭天天是甜的，然后还有各种恶婆婆的作怪...这妹子有一次大发雷霆，公公第二天租了个更高档的小区，说你要不先出去住....给你住更好的，还可以走路上班，balabala.

然后就是换锁，删人脸识别....等这个妹子提出要彩礼200万，用于还债降低婚后压力的时候。公公婆婆直接拒绝。

然后小兄弟肯定还是有感情的嘛，问题是经不住这种内耗，另外就是他爸妈也天天在他面前吹耳边风，还介绍各种大学副教授啊，什么围棋大师的女儿啊，之类之类的。

后来结果就是这小兄弟手机换号，房子也租出去了，然后搬到了父母早年在上海买的一套老破小里...

再后来就是发现这妹子到处相亲，然后小兄弟搞了几个小号加了下，发现这妹子只要月入多的，能帮她顶房贷的。什么初中毕业的40多岁的离异无娃暴发户（伪造的身份）也无所谓

再后来这两年券商也不行了，发现这妹子的房子在法拍房里....我无法想象发生了什么。
