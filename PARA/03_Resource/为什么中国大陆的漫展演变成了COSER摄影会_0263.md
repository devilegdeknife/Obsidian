# 为什么中国大陆的漫展演变成了COSER摄影会？
[内容链接](https://www.zhihu.com/question/661537655/answer/92341308681)

要相信市场看不见的手发威，筛选下来性价比最高可复制性最强的就是这个模式：

主办方租个差不多的场馆，请几个当地小有名气coser嘉宾，开放自由行，搞点集赞转发活动，现场就是coser逛街+摄影拍照+嘉宾排队+搞个舞台+弄点随舞+搞点小游戏互动+少许社团/pdd摊位。

coser就是出cos，拍场照，攒人脉，拿到自由行，或者组剧组游场，追星的找找嘉宾，到处集邮，拍点低脂小视频，重复上述过程。

简化版的only啥的，就直接保留市场筛选的漫展核心：coser逛街+嘉宾+随舞+场照（当然还有夜场，玩法要多一点）。各地二次元商场就更简单了，就是提供日常的coser逛街+随舞+场照场所，用来给谷店引流。

这一套已经被证明是最简单可行易复制模式了，是主办方和游客的最大公约数。从主办角度，只需要研究一下请哪个嘉宾其他固定流程重复即可；从游客角度，普通人cos又好看又不复杂（只要别出气质差别太大的，那浓妆+修图+正常的服道想丑都难），全套都能租/用完闲鱼，妆娘摄影啥的出点小钱全都有，还能社交能玩高兴，还有成就感和模糊的上升路径（coser-剧组核心-自由行-嘉宾）；摄影妆娘角度这也有得赚，所以大家可以达成共识，就这么办吧。

很多人说的这样那样，市场会打脸，就这么简单粗暴的结论。这个模式的漫展本来就是给宅充玩的，社交是漫展的主属性（你都不想见人来这么多人的地方干啥？），你真是死宅本来就不是人家想要的人。抱怨这种流水线漫展的人很多，到处都有人吐槽有人挂，但主办方有话说，人家说你们都不要嘉宾但只要不请，追星的人不来票就卖不出去就纯亏，你让我搞新花样成本不可控不说效果还未必好包被人挂。很多人试图自己办only然后发现不知不觉又绕回了这条路。主办-嘉宾-自由行/剧组核心-普通coser-妆娘摄影的链条目前来看运转良好，也是24年二次元大爆发成时尚单品后市场筛选出来的共识。

特殊的展也有，比如cp。但全国只有一个cp证明了，那种以同人和本子为核心的漫展全国只能养得起一家，其他同人展也大多没什么本子，也得加一大堆普通漫展要素。

再说另一个新兴的东西，谷子。很多人觉得这小纸片小铁片棉花娃娃算啥周边，高低得是手办这级别才是吧？这也是市场筛选的结果。24年二次元大扩圈，消费能力不高的学生是绝对主流。什么人均富哥富姐都是扯，大多数学生都是得省吃饭钱玩二次元，几块钱都要精打细算那种，当然只能玩这种便宜的。至于同人制品，数量类型有要求，肯定这些做起来便宜划算，本子没那么好出，前后需要大量时间精力。至于手办不是个人玩得起的同人产品。这也是市场筛选的结果。

最后，市场经济就要用市场规律说话，念马经复读康米没任何用处，自诩铁血老二次元更像一个笑话。想要其他样子的漫展，简单，有人出钱，有人参与即可。就像2024年几次大事件后，市场终于意识到男性有很大的消费潜力所以开始关注男性市场。看不见的手总被嘲讽但在市场经济下，看得见的手再狠也没空管天管地而且是这种不怎么重要的领域，还是看市场。
