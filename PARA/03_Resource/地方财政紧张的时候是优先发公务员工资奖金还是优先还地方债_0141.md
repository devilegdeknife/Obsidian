# 地方财政紧张的时候是优先发公务员工资奖金还是优先还地方债？
[内容链接](https://www.zhihu.com/question/609771926/answer/90227857849)

财政系统的人来给你解答一下：

预算第一优先安排三保：保民生、保工资、保运转。这个永远是第一位的，不要扯发不出工资，那是地方政府挪用人家这块中央和省里给你保三保的钱；顺便说一句：绩效也属于三保里面的范畴（为了严谨，修改下表述：没有奖金，只有十三个月工资），这个一般是省级和市县级财政自己安排进来，如果不安排，那是各个地方财政对预算的理解不同。所谓地方财政：指的是省、市、县三级。

预算第二安排偿还债务：这个债务说的是专项债、一般债这些，隐性债务自己地方看着办；

所以，知道保哪个了吧？

2月2日更新：

评论里大家对没钱地方政府还上基建项目，工资都发不出去了还搞工程，这个大家比较困惑，我简单说一下：

首先，财政是有四本预算：一般公共预算、政府性基金预算、国有资本预算、社保基金预算。一般公共预算就是税收和非税收入组成的，政府性基金预算主要是卖地的钱和城市基础设施配套费，国有资本预算是和国有企业运营挂钩，社保基金就是我们的养老医疗这块。4本预算里面，最重要的是前面2本预算，也比较复杂。

1.一般公共预算就是和上面说的三保有关系，税务部门预计今年可以收入10亿，加上各部门预计的非税收入预计5亿元，中央和省级的转移支付等等共20亿，一般公共预算收入就是35亿元（这里不考虑税收、非税和中央、省级分成的事，仅说市县一级留存部分），这35个亿收入，要考虑三保多少、法定增长教育支出安排多少，法定增长农林水支出安排多少，要偿还一般债务多少等等，还有各项财政政策补贴、各项工作经费安排多少，这么一算，钱就没剩下多少了。

2.政府性基金，原则就是以收定支，收入多少支出多少，如果支出大于收入，到时候帐是搞不平的，一查就出问题。这块主要是用来搞征地拆迁和工程项目的支出较多一些；

划重点：

预算安排上预算股/预算科的事，根据年初的收入盘子来定整个大盘，这是财政的龙头；预算执行支出，有没有库款（简单理解说人话就说是银行存款吧），是国库股/国库科的事，这是两条不同的线。预算股安排给各单位的是指标，比如，长期聘用人员经费100万，给你这个指标，系统里面就会显示你可以申请使用100万的指标。但是，单位在用钱的时候，不是说给了你这个指标就可以马上有钱可以花出去，你要在系统里面申请用款计划，类似于我们去银行要取钱，要写申请单，但是银行说我今天没有现金了，给不了你，所以国库股要是没有了库款，他可以不批。问题来了，库款的来源是哪里？

库款的构成：税收收入+非税收入+卖地钱+中央和省的转移支付收入+……

库款在调度的时候，一般是省财政厅国库处根据年初各业务处给市县财政安排的指标，按序时进度，比如一季度就是25%，按各指标给你25%的库款，也可以根据各财政局的需求，我需要1个亿就调拨一个亿的节奏慢慢来。看到没，这里国库处给你钱是根据年初各业务处安排的指标给你钱，不是预算处给的钱。

然后库款的构成主要是上面这4项，这里就不一一列举了。现在，我税收收不上来，非税收入又没有，地又卖不出去，转移支付收入又那么一点，领导又想搞项目，怎办？

现在很多地方是没有三保子账户的，就是在地方人行国库的账户里面下设一个子账户，把涉及到三保的指标都纳入进来，到时候申请三保指标的库款再把钱转入到这个子账户里面，这样就保证了人员工资、单位运转和基本民生支出等等。所以，如果没有这个子账户，所有的钱都在这个母账户里，虽然帐上可以分得出是卖地的钱还是税收的钱，但是这些钱都可以统筹使用。因此，当出现上面说的问题时，税收、非税上不来，地卖不出去，只能靠转移支付的收入，但转移支付收入大部分是有固定用途的，如果在指标里面没体现出你花了这个指标的钱，虽然我给了你库款，这样就会变成我们说的挪用挤占。比如，营养餐经费这个指标有100万，我给你这个100万的库款，但是你拿这100万去支付工程款，然后营养餐经费没有任何支出，这时候就变成挪用挤占了，因为我给你钱了，你把我的钱花到别的地方去了，本来要给学生买牛奶蛋糕的，现在欠供应商的钱了，这个指标没显示支出，但是库款那边显示这个指标国库处已经给钱了，这就成为挪用挤占。所以，本来要发工资的钱，因为地卖不出去，只能拿中央和省里给的钱去支付工程款，就变成拖欠工资问题。

办法就是上面提到的三保子账户要尽快设立，钱都进入里面去，虽然不能完全杜绝但至少大范围减少。

2月15日更新：

评论提到说，说了那么多篇幅没切入主题，就是为什么没钱发工资了还搞基础设施项目。现在和大家解释下：

我们知道，以前一个省、市县的考核，都是唯gdp论，我们先不说这个考核的对错，毕竟每个时代都有局限性，不能拿现在的眼光去套以前的东西，这样永远都有矛盾，都有问题，大环境下，不是每个人都有第三眼。后面，经济发展到一定程度，发现唯gdp论是不行的，毕竟会出现生态环境、腐败、数据造假等诸多问题，这里就 不展开了。但是gdp太重要了，有文章研究过gdp增长多少个百分点，就拉动就业多少，这里就不再论述。后来，北京发现，再这么干下去不行，虽然gdp很重要，但是它不是唯一的，这里先占个坑。2月16日补上，尽量用说人话的语言给大家解释，少用专业术语，希望大家不要讲用词不严肃 的问题，毕竟受众者不是都懂专业知识。

2月16日更新：

2023年3月13日，十四届全国人大一次会议闭幕后李总理在出席记者会并回答中外记者中指出：客观地说，绝大部分老百姓不会天天盯牢GDP增加了多少，大家更在乎的、更关心的一定是自己身边的具体事，像住房、就业、收入、教育、医疗、生态环境等等，政府工作要贴近老百姓实际感受，要真正做到民有所盼、政有所为。

我们看里面的内容拆分开来看，住房(商品房、公租房、廉租房、安居房)、教育(幼儿园、中小学)、医疗(疾控中心、地方市县级医院，还拼命争创三甲医院)、生态环境(污水、蓝天、土壤、空气)，从上面来看，任何一项的投资随随便便都得500万及以上，从发改的固定资产投资项目来讲，这些都是要纳入固投的，你不搞这些东西，你怎么谈发展、谈更贴近老百姓的实际感受。而且，固定资产投资是拉动建筑行业税收的，间接是提高我们农名工这个群体的收入、解决各上下游企业的库存问题，推进企业生产，员工收入有保障、员工有了钱就去消费，然后再扩大生产。这么不断的形成良性循环，gdp自然而然就上去了。

但是，搞基建也好，固投增长也好，有时候的统计口径不是以你支付出去多少钱来算作为完成数的，而是以形象进度、工程进度这种方式来统计。意思就是说，只要我的项目达到这个进度(形象进度也好、工程进度也罢)，就可以当成是固投任务的完成。当然不排除有些必须要开税票、付款之后才算完成固投任务。因此，没钱发工资和搞基础设施项目并不冲突，因为搞基础设施不是一定要付钱才算，政府部门很多项目基本上是拖欠工程款的。这个就涉及到另外一个问题了，这里不展开了，大家有兴趣再更新。

我没在发改和统计任职过，也是业务上和他们打听了解这块的内容，不对之处多批评指正。
