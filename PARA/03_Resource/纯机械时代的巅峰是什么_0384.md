# 纯机械时代的巅峰是什么？
[内容链接](https://www.zhihu.com/question/270090908/answer/2659049212)

如果从技术角度来看，应该是气动管道。

如果从精密加工来看，应该是陀螺仪。

如果从国际影响力来看，应该是AK47。

如果从信息安全来看的，应该是恩尼格玛乃至KL-7和HX-63等密码机。

如果从国家级较量来看，应该是无畏G3等一票战列舰。

所以我会做一个综合性打分，从社会影响力，产品价值，设计难度，成本控制这四个角度来进行综合评分，评分标准如下：

S：同类产品顶尖的存在

A：同类产品中的优秀产品

B：还算优秀

C：及格水平

D：凑活而已

E：这是啥？/金属垃圾/金字塔级别/造不出来

PS：我对"纯机械"的定义是指“在无效化产品内部的电子管等其他精密电路之后，该产品任然能正常发挥出预想的功能”.......

所以按照这个标准大和鸭滑南达黎姐VV前卫不幸的被剔出了榜单........以及，东方1号和Bagger293也被开除了.......

1.为什么G11没进第一档？

G11是一款典型“一顿操作猛如虎，一看只有二百五”的典范，具体参考[@沛青](https://www.zhihu.com/people/1158466149f08c509f85595e904134f5)的文章

2.为什么你不看好差分机？

我认为差分机最大的缺点是巴贝奇没能有效控制技术风险从而导致差分机没能建成……总之，适当的管控技术风险和创新一样重要。

当然，论若要论创新，KL-7和HX-63密码机的“回注”设计，也是足以完爆差分机的……可以说HX-63的“回注”设计确保了即使在面对现代电子计算机的暴力破解时，HX-63仍有一定的安全性!!!

[最后的转轮密码机](https://link.zhihu.com/?target=https%3A//mp.weixin.qq.com/s/dkGmxN4kG58pmpGTnXnh7A)

3.为什么你对HX-63密码机的评价会如此的高,以致于要将评分重新调整为S？

我个人评价一款机械产品的一个重要原则就虽然看上去是这款产品能否应对近20年后的技术迭代？从这个角度上看无畏级，Enigma在20年之后的技术面前几乎等同于金属垃圾，AK47虽然看上去还行，但是在小口径弹面前已经被拉开了代差，而G3虽然被认为几乎可以和嘴炮狮甚至鸭滑一战，但是仍然受到高速轻弹的制约，虽然PKM重量几乎和M249相当，但是7.62mm*54mm是凸缘弹……但是在HX-63密码机面前，它要面对的是摩尔定律……但即使这样，暴力破解HX-63的口令所需的时间起码相当于暴力破解Enigma的口令所需的时间的10^584倍，这几乎相当于1993年的美国海军航空兵花了6个多月才勉强击沉大和！！！！！！
