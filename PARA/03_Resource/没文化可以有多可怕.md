# 没文化可以有多可怕？
[内容链接](https://www.zhihu.com/question/26192324/answer/2998865070)

我小升初那阵子曾经流行过一本特别学生腔的网文，叫《卡徒》。那个小说里卡片基本就跟现实世界的电路一样，区别在于越是高等级的产品就越不可能量产，只能手工搓。简单来说就是科技形态的魔法卷轴。

主角是一个社会底层的、好学坚韧的制卡师，是孤儿，连续多年只能制作一级能量卡。相当于买散装材料自己手工做七号电池，并且完完全全以此为生。




那不是一个读起来多有意思的小说，而且那个年代大家普遍水平也都不怎么样，所以后面写了什么、乃至主角叫什么我都已经全部不记得了……只不过，十好几年了，有一个段落我至今记忆犹新：




主角遇到麻烦跑路了以后，几个顶级学校的天之骄子聚集在他住的贫民窟廉租房里，计算一级的能量卡的市面价格，还有他凭个人身份购买那些制卡的物料的成本，来推敲他是一个什么样的人。

结论是他的手非常稳，损耗很低，制卡的成功率非常高，比他们这些天才制卡师都高。他每天只要多做坏一张能量卡，就会没赚头，多做坏两张就会入不敷出，但这样的生活他居然坚持了三年。

而这些原料对他们来说，都是免费无限供应的，他们从来没有想过世界上居然有人需要尽量节约每一滴……类似电路上的焊膏那样的东西。而且居然真的靠这种节约活了下来。

然后他们心悦诚服地表示这个人很可敬，自己不如他。




那个故事里的学院就是门派，顶级学校同时也是顶级企业，这些人就相当于最核心的嫡传弟子什么的。作者在这里应该是想要刻画一种天才+贵族的群像，似乎是觉得，天才应该是爽朗和开阔的，贵族应该有赏识别人的眼界跟气度，而且有一种对于事实的纯粹的喜悦。

这个段落让我读到了某种堪称“高尚”的东西，即心平气和地辨认别人身上那些努力和闪光的地方——而且是在完全陌生的领域、用近乎纯推理的方式，也完全不介意对方在某些很重要的方面超过了自己，更不会吝惜对其他人的赞美。

在看到那一段之前，我从来没有在现实里见到过类似的东西……当时我最大的感觉就是这个作者应该是一个很善良的人，很难得地见过一些美好的东西。




后来我在好学校里也没能见到这种东西，

在好公司里也没能见到这种东西……

那些地方更多的都是……反正我写过很多次了，自我物化得很严重的那种人——急不可耐地寻找别人身上可以攻击跟值得利用的部分，也不在乎自己攻击或者利用的动作是不是符合事实的，就好像快要淹死的人攥紧了足以安慰自己的稻草，因此没有能力对事实感兴趣。




我在b站很喜欢关注那些花半天时间给未来一星期认真备菜的up主，

或者是十块钱八块钱做一顿饭，

或者十分钟八分钟快手菜，

还有做饭糊弄学、拿剪刀剪菜、就着锅吃饭的人……

我真心实意地觉得这种……怎么说呢，也许没有意义，但是很有意思。




还有很多年轻人的创业项目，没有资金也没有技术，也没有所谓的行业视野，就是拿公开资料和免费资源再加上一些忽悠来做的，我也觉得蛮让人感动的……就像《卡徒》的主角做一级能量卡可以做三年。

当然这是很大的天文数字的浪费了。我的意思是导致这种巨大浪费不断发生的原因一直都是存在的，但这阻挠不了那些有天赋、热爱跟恒心的人一直进步，就像草籽突破岩石，这个真的蛮让人感动的。

（当然，坑蒙拐骗还是很不好的。想挣快钱会摧毁一个人认真的能力。如果不是特别有必要的话，建议大家不要坑蒙拐骗。）




题目强调了“文化”和“知识”是两回事，我就想起了我在小学读的质量不怎么样的网文里面见到了一种让我心生向往、自惭形秽的东西，然后这一点一直影响我到现在。我真的希望自己是这样的人，同时我希望周围我重视的人也都能做到这样。

但我不是很理解，这算“知识”吗？还是算“文化”？

《卡徒》居然能算是“文化”吗？还是说“高于现实的东西”都能算“文化”？




那么，没有文化（或者说没有向往）的人大概可怕就可怕在，无论拥有多少、多好的东西，眼睛里面还是只能去寻找可以攻击的对象，和值得利用的对象——换句话说，对“事实”漠不关心，永远只能看到自己那一点点私心跟心结——好让他心里面的感觉变得稍微好一点。
