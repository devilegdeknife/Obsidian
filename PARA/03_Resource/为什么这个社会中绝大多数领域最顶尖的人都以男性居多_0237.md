# 为什么这个社会中绝大多数领域最顶尖的人都以男性居多？
[内容链接](https://www.zhihu.com/question/429140806/answer/1950739347109290285)

oracle的总裁拉里.埃里森2016年在南加州大学的演讲，很能说明一部分男人和一部分女人的区别。

只是一部分男人和一部分女人，并不是所有的。




拉里.埃里森说，他年轻的时候，就喜欢各种自然景观，各种山，太平洋。

还喜欢玩帆船。

他搞IT相关的东西，完全不是什么兴趣，而是就是想赚点生活费。




他老婆看不起他，说他整天在山里面瞎转，应该去读一个大学的计算机课程，然后找一份高薪的工作。

他去报课程了，结果报了一个帆船的课程，课程结束后，借钱加分期付款买了一艘帆船。




拉里.埃里森自己对生活非常满意，满意得不得了，如果可以，一直这样过下去。

但是在他老婆看来，他是太失败了，不上进，loser。

听说他还借钱买了帆船，忍无可忍，把他赶出了家门，还和他离了婚。

拉里.埃里森的家里人，对于他放弃医学院，极为愤怒。

拉里.埃里森没有读完任何一个大学，是高中毕业。




他就这么晃荡着，接一些工程赚点生活费，30多岁还一事无成。

其实他是一个天生的销售员，而对于计算机并没有天赋。

直到33岁那年，那时，他在做一个40万美元的中情局的项目。




这个时候，IBM的研究员发布了划时代的关于关系数据库的论文。

拉里.埃里森从他在IBM的朋友得到了一些介绍关系数据库的期刊，他马上敏锐的意识到这是一个机会，而IBM自己都没有意识到。




于是拉里.埃里森组建了一家公司，找了2个人，总投资才2000美元，开发数据库系统。

他极力劝说中情局的人，采用这种新型的数据库。

这就是后面著名的oracle数据库。




如果按照拉里.埃里森老婆的话去做，拉里.埃里森绝无可能创立这样的公司。

因为机会来临的时候，他都33岁了，在大公司打工，绝无可能放弃当时的职位出去，也没有关系和人脉。




还玩帆船吗？

还玩，拉里.埃里森5次赞助船队参加大名鼎鼎的美洲杯帆船赛(America’s Cup)，其中两次夺冠。




就是因为他啥也没有，到处晃荡，反而可以抓住这种开创性的机会。




我并不是说一个逆袭的故事，而是说，有些男人，女人认为他很失败。

其实他对自己的生活很满意。

按照自己满意的方式过一生，也未尝不可，蛰伏起来，真有机会还是能奋发向上的。

没机会，没风口，你瞎折腾什么?




我认识一个人，玩房子的，30多少岁的时候，也是整天和倒腾房子的，搞信用卡的混一起，每年工作2月，虽然赚钱不少，但是看起来要多失败有多失败。

其实他对生活已经很满意了。




他老婆看不起他，他就出来做房圈大V，曾经很有名，现在楼市这样，他就更失败了。

这不是重点，重点是过去那种满意的生活，再也回不来了。
