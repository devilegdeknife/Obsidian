# 如何评价国产丝袜品牌「绫」?
[内容链接](https://www.zhihu.com/question/363511595/answer/4769379991)

这几年冒出了好多类似的网红品牌，什么绫、涞觅、绯隐、安幸，感觉本质都差不多，都比较擅长品牌包装和营销。绫算是这个赛道的鼻祖？当年第一次看到绫的产品页面确实给人的感觉很不一样，视觉冲击很强烈。网上卖货视觉很关键，再弄个有设计感的包装、喷点香水，加上打**的推广，就这么火起来了。

国内丝袜行业也确实一直没什么太大变化，除了早年打出了一个家喻户晓的浪莎，这么多年了也没出几个能让人记得住的品牌。所以有一些这样的新品牌也算是精准地挖掘和抓住了消费者的需求，至少在某些方面还是有些新意的。

当然就产品本身而言并没有颠覆性的变化，东西还是那些东西。国内丝袜大部分都是出自浙江，主要集中在金绍嘉地区，核心产地就是义乌、诸暨、海宁等几个地方。绫跟涞觅也不例外，产品基本上也是出自这几个地方。他们本身也不是制造业出身，连公司名字都叫什么文化公司、网络科技公司。像绫的代工厂有一家就是义乌的，叫浙江曼姿袜业，这家算是行业头部大厂了，代工的客户都是包括厚木、郡是、福助在内的知名品牌。毕竟绫卖的不算便宜，质量太拉的话也说不过去，工厂还是要找好一点的。

义乌这个地方作为小商品之都，丝袜产量可以说占据了行业的半壁江山，但大部分都是低价走量的，质量参差不齐。当然优质大厂也有，除了曼姿还有不少，比如宝娜斯、梦娜、芬莉（芬那丝）这些。像宝娜斯在义乌和江苏宿迁都有工厂，也给优衣库等一些品牌做过代工（涞觅也找过宝娜斯代工），不过现在好像它自己的产品有些也是贴牌的。

![图片描述](https://pic1.zhimg.com/v2-c017f0858134ee2ab9c7e492a1d82a84_r.jpg?source=2c26e567)

梦娜的话曾经也辉煌过，代工和自主品牌都做得不错，08年还是北京奥运会的赞助商。芬莉主要做出口，给很多国际大牌代工，它自己的品牌叫芬那丝。最出名的某莎其实不太推荐，它现在跟南某人一样大搞授权贴牌了。

诸暨、海宁的话也有一些做丝袜的大厂，像耐尔就是海宁的，它家的丝袜口碑也不错。也有一些专门做代工的，比如亚润、威尔斯什么的。诸暨的大厂有丹吉娅（好像经营出了问题，老板限高，另一个情怡袜业也差不多）、东方缘等。还有一个热销的网红品牌叫SK的，他们倒是制造出身，工厂叫浙江博羿针织有限公司，就在义乌隔壁的东阳，就是横店影视城所在的那个县级市。

![图片描述](https://pica.zhimg.com/v2-ec0173ec7a044f1abfddefc07bb405f1_r.jpg?source=2c26e567)

浙江台州的天台也有不少做丝袜的，不过工厂规模都比较小，作坊小厂为主，其中还有一些做情趣丝袜的。至于江苏连云港灌云县的丝袜，那纯粹就是情趣用品了， 那地儿可是情趣内衣之都。

山东半岛因为沿海又靠近日韩，对外贸易也比较发达，像日本袜业巨头厚木就在烟台建了厂，最早是和央企华润旗下的华润纺织合资的，后面华润退出了，现在烟台三家工厂（厚木针织、厚木袜业、阿姿谊）都是厚木全资所有。说起来，华润自己也有丝袜品牌，叫rime（韵魅）、华姿伊（CREASUGI），只是现在哑火了，当年进驻的都是高端商场。

![图片描述](https://picx.zhimg.com/v2-5af5f277f3a933aa03281006dbed7d46_r.jpg?source=2c26e567)

![图片描述](https://picx.zhimg.com/v2-20f1e5b792555cccd07fd3e8a8dbce09_r.jpg?source=2c26e567)

当然溢价最高的还得是欧洲品牌，什么Wolford（奥地利品牌，号称丝袜中的劳斯莱斯，现已被郭广昌的复星公司收购）、FALKE、FOGAL、GERBE、La Perla、Cervin，毕竟是行业祖师爷，历史悠久，无论是品牌还是设计工艺暂时都比不过人家。就是它们的价格对于丝袜这种消耗品来说不是大众所能承受的，适合不差钱的。退而求其次，日本的Tabio、厚木、郡是这些也不错。
