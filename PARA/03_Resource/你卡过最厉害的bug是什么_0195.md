# 你卡过最厉害的bug是什么？
[内容链接](https://www.zhihu.com/question/316703576/answer/1954865172578559934)

08年，想创业了。拉上3个亲朋好友一共4个人，凑了100W，搞了一个汽配厂。

天天起早贪黑，09年没赚钱，10年没赚钱，11年添了几台设备也没赚钱，12年还是没赚钱。分析原因后，是因为厂房租的太小，产能上不去，各项开支又很大。

想了半天，租大厂房，资金压力太大，还要添设备。刚好当时隔壁县市来招商引资，心一横，干脆买块地自己建。

和股东一商量，把他们都吓了一激灵，这还没赚到钱呢？哪有钱去搞啊？商量了半天，退了两股东。借了钱，很好的安置好他们，就准备去拼一把啦！

其实我是做了很多精算。基本上是有把握的。因为当时政策比较好，土地是有优惠的，15万一亩。我们必须拿下20亩才够支棱起来。

具体情况是这样的，20亩土地买下300万，但是市场评估是按35万一亩评估，价值700万，抵押贷款后能到手490万，还掉买土地借款300万后剩下190万，就找施工队入场，然后施工方按进度给我开发票，我靠发票的在建工程贷款，贷到钱给施工方，以此类推把厂房建好。

土地太少，那么初始贷款少，支持不了施工方入场。土地要是多了，我心虚啊！那时候招商引资办公室主任问我，你们投产后能有多少销量，我心一横说有4000万一年。他沉默了半天，说有点少啊！其实那时候我们最多不超过1500万。

具体操作就这样按部就班起来了。厂房20亩建了18000多平方，总投资1680万，总贷款1780万。厂房建成保证金还有80万是按年支付的，所以账面上还有几十万的钱。

第一步，找所有供应商来庆祝开会，吃吃喝喝之前致辞，说花钱建厂房了，以后规模和采购量都会增加，但是没钱了，所以接下来三个月，只拿货，不结账，三个月后再结账。能支持的，表个态，不能支持的，也感谢以前的帮助，只是现在要找能支持的合作伙伴了。因为以前诚信好，特别是给钱从不抹零头，所以整个行业口碑很好，大家都给了支持。

第二步，拿账上的钱添设备，抓紧招人，工资可以压50天，就是这个月工资，下个月20号发，所以马上增产。

第三步，因为是汽配的易损件，市场容量很大，所以促销降价搞活动，现款销售。因为以前注重品质和细节，市场口碑也不错，就把产量拉上来了！

现在年产值1.5亿，员工近200个。场地不够了，又买了30亩地，没有任何银行贷款。

赌赢了不是！！！

更新1

评论区有人猜是汽车三滤，猜中了！三滤需要啥设备，冲床，拉伸机，封边机，折纸机，其它全是人工。二手冲床几万一台大把都是，量大还能赊一部分每月付。所以设备没啥问题。

至于顺利，那是一篇说完说得轻松，其实企业发展3000万是个坎，6000万是个坎，再往上1.5亿是个坎。6000万的时候清退了所有亲戚，花了几百万找咨询公司做正规化管理，度过最痛苦的两年。当然也有过账上没钱，到处借钱的窘迫。这几年发展的好，也确实有很多运气成分。23、24、25三年我们都持续增长，搞得商务局都打电话给我，询问数据是否真实？我说，有少报偷税的，难道还有多报就想多缴税吗？

其实我真正想说的，如果靠原始积累，100万起步，每年每个周期赚10-15%，一年最多4-5个周期，你想滚到一个亿，就真的猴年马月啦。而且大企业看不上你，中等规模同行打压你，真的没啥空间和机会。其实这个bug还能卡，去年我就教会一个，他跑珠海买了地，也慢慢起来了。

更新2

不信的朋友，你们就当故事看吧。

为啥简介上是创业指导师啥啥的，因为在3000万这个坎的时候，我明白企业的天花板其实就是老板自己的天花板。所以大概16，17年那时候去培训自己了，学过教练技术，学过上海行动教育的赢利模式（现在改名叫浓缩EMBA），学过湖北群艺的积分制管理，学过深圳华一的股权激励。然后结交了许多企业家朋友，加入当地各种企业家协会，后TA学会，盛和塾等，定期学习和分享。那时候分享表达欲很强，所以被朋友戏称你可以当创业指导师了，然后真有一些创业初期的协会成员找我，那时候沾沾自喜，就写上。

卖课？那你等着哈，看我什么时候收钱！

风险？亏了咋办？耽误别人咋办？让其他人亏了咋办？有这些清奇思路的人，你是不是也不该找对象不该结婚啊，万一你出事走了，不也是让你对象难过一辈子？所以别扯有的没的，干就完了！懦夫有一万个理由，勇者只管前进！

另外，比起做企业，这几年我更在意做一个好爸爸，提倡所有家长接受教育，先把自己教育成合格家长。有兴趣的可以看看，孩子才是未来。

还有，没有那些什么挤兑赶出门啊，处长干部来担保护航啊。我纯纯外地人合法合规，甚至因为体质原因，滴酒不沾，也不请客吃饭，江浙沪和珠三角的创业环境都挺好的。有些朋友搬到内地去了，听说有些吃拿卡要的，但也不严重。

更新3

有些小伙伴喜欢抓虫，这里就统一回复一下。

17年18年就是我们引入管理咨询公司的两年，19年公司基本都是制度化管理后，我觉得传统制造业的瓶颈期到了，所以就出来去搞自动化设备去了，开始新的创业。

但是，22年我们汽配厂迎来了一波机遇，吃到了俄乌战争的红利，突然破亿，并且稳定增长，这是我没有预料到的。

新项目自动化设备目前发展也还可以，谢谢大家关心。

所以现在我说论收入，我老婆是我三倍以上了。当然，夫妻没分家，这也是纯粹的玩梗而已。我个人开销很省，不抽烟不喝酒，不爱豪车不买奢饰品，爱吃点红烧肉实在不费钱。

更新四

对于土地买是15万，银行能评估35万的事。这不是银行宽松，很多地方的工业区招商都有类似的政策，甚至内地一些地方还有替你担保的政策。以前苏北某地方招商，地方政府还能拿出1000万无息无担保的贷款给你当基建资金，那个是真的好多人去骗钱了。

我那时候，政府向农户买地就要20多万了，然后还要三通一平，所以评估价35万很正常。多跑几家银行多谈谈。

前几年比亚迪拿了200亩，土地白送，政府还出了40亿基建资金盖好厂房买好设备。完了，比亚迪还说不签任何税收条件的合同，唯一保障就是2万人的就业。这才是真牛批！

最后谢谢大家抬爱！我也就是普通人而已，谢谢，谢谢！
