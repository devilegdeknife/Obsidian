# 如何看待 QQ 扫描读取所有浏览器的历史记录？
[内容链接](https://www.zhihu.com/question/439768601/answer/1682759708)

如何防止所有软件（包括但不限腾讯系、阿里系、百度系）在后台**偷偷摸摸**读取、修改你的**谷歌系**浏览器历史记录，至于edge与火狐的保护请复制本回答最下面的三行代码（分别为chrome、edge、火狐）依样画葫芦即可，或者直接使用我在本回答中提供的json导入即可，已经全部包括。

下载火绒安全软件，建议搜索“火绒”或者点击官网：[火绒安全](https://link.zhihu.com/?target=https%3A//www.huorong.cn/)下载最新版火绒安全软件5.0。

1.下载安装后打开火绒安全软件，点击“防护中心”

![图片描述](https://pic1.zhimg.com/v2-320123d90ad2d71a0ef0673ddd6501bf_r.jpg?source=2c26e567)

2.点击左下角的“高级防护”

![图片描述](https://picx.zhimg.com/v2-922b8e6b9102ec5f74d4756499b35c1f_r.jpg?source=2c26e567)

3.点击“自定义防护”五个字

![图片描述](https://picx.zhimg.com/v2-03331f97e9b48fbb0287e798146c658c_r.jpg?source=2c26e567)

4.点击右下角的“添加规则”

![图片描述](https://pic1.zhimg.com/v2-eb3cc77d4b9d0063787977b5030a639f_r.jpg?source=2c26e567)

5.“发起程序”一栏中建议填写默认的“*****”即为所有软件的访问都先过一遍火绒，然后点击右下角的“添加防护对象”。

![图片描述](https://picx.zhimg.com/v2-6601b020634e9159e0fce5daf7863894_r.jpg?source=2c26e567)

6.在“文件规则”下填写冒号以后的黑体字符：***\User Data\Default\History**

在“保护的动作（必选）”中勾上“读取”与“修改”，接着在“规则名”中填写一个自己喜欢的名字，点击“保存”。

![图片描述](https://pic1.zhimg.com/v2-c59cb5f9b3caab96ec8a9a38be73ded8_r.jpg?source=2c26e567)

7.检查发起程序与规则是否正确，全部正确后点击右下角的“保存”。

![图片描述](https://picx.zhimg.com/v2-680fab98c2f1a782f63e7d77c79a5dc7_r.jpg?source=2c26e567)

8.在“自定义规则”中即可找到刚才所添加的规则，此时退出QQ/TIM再重新打开，等十几分钟来验证规则是否生效。

![图片描述](https://pic1.zhimg.com/v2-9dcd5ffcf11df8f3af4adba3db7d146b_r.jpg?source=2c26e567)

9.规则生效

![图片描述](https://picx.zhimg.com/50/v2-af14719ad4117aaa9150e27aa3332c37_720w.jpg?source=2c26e567)
