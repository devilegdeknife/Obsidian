# 在科幻小说里塞教科书般的详细设计会咋样？
[内容链接](https://www.zhihu.com/question/493758111/answer/2237277529)

一个善于说明的作者：

“这名殖民地属地卫队的士兵使用红石激光枪瞄准了眼前的独立主义叛乱分子，这种AA-7红石激光枪原本是一种对空武器，在经过电池小型化后配发至军队，因此有着千米距离0.1米散布的精度和能击穿200mm等效装甲的穿甲能力，叛乱军使用的是前俄罗斯制造的AKM250实弹自动步枪，根本无法与卫队的火力抗衡，况且，卫队还能用中继卫星遥控UAV进行远火支援。这是一场屠杀。”

一个善于叙事的作者：

“这名殖民地属地卫队的士兵使用激光枪瞄准了眼前的叛乱分子。马歇尔殖民地有着非常严重的贫富差距，在这贫富差距下，富者夜夜笙歌，贫者却无立足之地。而因为饥饿和压抑，流民组成了土匪，土匪变成了革命分子。但是，流民终究是流民，没有钱，也没有武器。现在第七步兵营撞上的这支游击队，只有少量的实弹武器和一些劣质的RPG，恐怕只需要一轮齐射，这些乌合之众就会如鸟兽散。这是一场屠杀。”

一个善于描述的作者：

“这名带着绿色头盔的殖民地属地士兵使用方盒子状的激光枪瞄准了眼前的披着一身废铜烂铁的叛乱分子。眼下是一片沙漠，方圆数十里看不到任何绿茵，除了沙丘以外没有什么好的掩体。叛乱军似乎完全没有感受到敌意。他们将破旧的飞船拆解，将里面泛黄的电子器材搬到长着兔子耳朵、已经不堪负重的本地驴上。其中的一个人抱出来了一堆罐头，引得叛乱军士兵们一阵欢呼。而殖民地士兵趴在沙丘背面，一百只枪如毒蛇吐信般伸出。这是一场屠杀。”

一个善于抒情的作者：

“在长官的命令下，玛吉纳瞄准了这些‘叛乱’分子，这是她想做的事情吗？显然不是，她出生在这片殖民地，曾祖父是传统的玛娜教徒，然而她的父亲，和家庭决裂，在地球学习，最后成为了技工。这是对宗族的背叛，她接下来唯一的出路，就是融入地球人社会。但是无论如何，她都是土著，地球人不信任土著，把士官学校优秀毕业生的她挤到了步枪兵团。甚至恶趣味地逼迫她对着自己的同胞开枪？他们做错了什么？他们仅仅只是想活下去。

人类不值得效忠——玛吉纳把枪口抬高，虽然这是一场屠杀，但是自己不会参与。”




因此即使是塞教科书一样的说明，也不会影响事情的发生——“一个强者将要殴打一个弱者”。只是叙事者的关注点不同，这与叙事者的性格、生平和专长有关。以上的四位作者，可能分别是工科生、社科生、编剧、爱看知乎的家庭主妇。

少年坐在花坛的边缘，弹奏着来自地球的尤里克吉他。这个沙漠星球特有的焚风吹拂着他柔顺的半长发。他大概十七八岁的样子，斜带着帆船帽，折领大衣像披风一样，显得放荡不羁。

“玛吉纳，这个家伙是谁啊？”

塞西莉亚询问着自己的好友玛吉纳，玛吉纳是这颗星球的本地人，棕色卷发，深色皮肤，红宝石一样的瞳孔。

在人类抵达之前，这颗星球有着名为玛娜的原住民。他们和人类一样，都是大圣灵的造物，只不过玛娜人更加地重视与自然的关系，他们的科技等同于魔法，可以使用名为“魂力”的法术操纵森林、大地，与生物对话。不过随着伤膝谷战役以人类的胜利告终，玛娜开始放弃自己的“魂力”，学习人类的科技。

“是步兵学院的龙正明，”玛吉纳说，“来自西边的龙城。”

“大花坛里不能卖艺”，塞西莉亚说，“我得让他回去。”

塞西莉亚是地球最尊贵的血脉之一——朱沃洛夫家的继承者，也是殖民地防卫学校最年轻的“勋爵骑士”。

而殖民地防卫学校也是她们家的产业，这个学校培养着殖民地防卫军未来的军官——分为步兵学校、炮兵学校、装甲骑兵学校、轨道骑兵学校、舰队学校、情报学校和后勤学校，有三个年级共计四千名学生。塞西莉亚是轨道骑兵学校的学生，而轨道骑兵被誉为深空之花，是最适合贵族的学校。

她走下台阶，高帮皮鞋发出“沓沓”的声音。而坐在花坛上的龙似乎听到了这声音，收起琴站起来，向她鞠了个躬。

“勋爵骑士，请问有何贵干？”

“她是装出来的，”玛吉纳在塞西莉亚耳边小声低语，“龙城是这个星球最贫穷的城市，那里来的人粗鲁且缺乏礼教。”

然而塞西莉亚一句话也没听进去，在下午不算炽烈的阳光下，少年显得是那么柔和温暖，就像是鹿特丹的向日葵一样。

“身为学生会的执行者，我要警告你——公共场合不能……不能……”

塞西莉亚斟词酌句，想要把“卖艺”这个词用符合自己身份的贵族语言说出，不过玛吉纳帮她解决了这个问题。

“公共场合不得卖艺，这里是学院不是戏院。马上要上课了快点回到步校去，你们的苏明教官可是体罚人毫不手软的。”
