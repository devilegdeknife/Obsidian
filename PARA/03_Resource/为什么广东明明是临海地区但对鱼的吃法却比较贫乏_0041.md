# 为什么广东明明是临海地区，但对鱼的吃法却比较贫乏?
[内容链接](https://www.zhihu.com/question/1923095535369298449/answer/1968867000668357438)

我从侧面讲个例子，你们理解看看——

沿海地区盛产牡蛎，也就是大家常说的“生蚝”。

而牡蛎最完美的烹饪方式，不是煎蛋、不是煮汤、不是清蒸、不是蒜蓉…

## 而是微波！

对于非沿海地区生活，接触不到活牡蛎的人，一定会觉得牡蛎虽然好吃，但总是会有一股淡淡的腥腐气味，必须要加香料高温烹饪才会好吃。比如蒜蓉生蚝…

但其实，如果你往蒜蓉酱底下煨个煎蛋，到你嘴里其实和生蚝也差不多…这就是典型的没有控制变量做出的美食测评，酱味代替了食材味，你酱个抹布也是好吃的。

长期的复杂烹饪，麻痹了你对“鲜味”的认识。

同时你又缺乏鲜活食材的接触经验，这让你对牡蛎料理的理解，宛若坐井观天。

作为一个沿海城市生活的人，作为一个在海滨吃过活鲜大餐的人，我必须要带你们跳出井口见见世面！

**微波牡蛎，绝对是生蚝料理中的无上王者！**

选购的牡蛎，一定要是大个子的，带壳的，鲜活的牡蛎。这是海鲜之中的极品美味！因为大个牡蛎生命力通常比较顽强，活体不易死，它的免疫系统可以一直压制有害微生物，不易出现腥腐气味。大个体的表面积率比较小，与微生物的接触面积也小，即便死亡，腐败速率也低。

我家时常会去海边渔村采购新鲜的牡蛎，带回来之后，洗去外壳的泥沙，冲刷干净后就放冰箱冻着。这是采购大量牡蛎时候的保鲜技巧，可以确保牡蛎本体不会受到任何微生物的腐化。

想吃的时候，拿几个牡蛎出来，整个大碗儿，加水淹过牡蛎壳约七八成，确保壳缝在水面以下即可。然后直接放微波炉，高火4~5分钟。（有条件的话尽量用深一些的碗，这样水加得少也可以没过牡蛎壳，微波炉火力会更加集中，加热速率更快）

这玩意叮出来之后，绝无半点腐腥气味！接近沸点的汤水里透着海藻的清香！

闭壳肌松动，牡蛎壳微微张开，用金属汤匙插进缝中一撬，再用汤匙的边刃铲断牡蛎的闭壳肌，然后将牡蛎壳打开。

映入眼帘的是又大又白又肥的一块蛋白质聚合体，上面还不断冒着高温白气。扑面而来的热浪裹着海藻的清香，四周弥漫的海鲜汤独有的咸香味逐渐浓郁。

用嘴轻吹几下放凉，将洁白圆润的蛋白质聚合体放入口中，用舌头勾卷到牙刃之间，鲜甜之味溢满口腔！

轻轻咬下，圆滚饱满的蛋白质聚合体仿佛爆裂开来，带着几分咸甜味的汁水瞬间涌入，在唇齿间回荡。

那味道，是蛋白质与天然海盐的完美组合！

有些烫，有些鲜，有些甜，然后便是无穷的回味……

高功率微波带来的超速烹饪，微生物根本来不及侵蚀，环境温度就已经达到消毒水平，并且仍然快速持续加热。堪称炸裂的加热速度确保了绝对鲜度，同时也蒸腾起了贝壳中各种营养物质轻微挥发时的诱人气味。

什么蒜蓉、什么十三香……都不如这糅杂着海藻清香、带着蛋白质鲜甜的元初之味！没有任何侵染、没有任何做作、没有任何瑕疵…

蛋白质的鲜、海水的咸、海藻的香、腹肉的软、鳃页的韧、闭壳肌的弹…咀嚼一颗完美烹饪的牡蛎是一种多维体验，就像是在奇幻的味觉世界里发现了一枚珍珠，那在阳光的映照下折射出七彩虹光是何等耀眼夺目…

啊……Perfect！嘴里，不仅有热气，更似有光芒。

没错！

水生无脊椎动物最崇高的轮回典礼，

就是速蒸！

吃鱼亦是如此。

并非吃法贫乏，而是最高典礼真的只有一种！

只有这一种典礼，才能让你切实体会到盘碗里的动物数亿年进化的艰辛！

补充回答评论区常见的几个问题：

1、生吃不会更好吃吗？

生吃嘴里不会发光。至少生吃的那几次，我是写不出这么妙的词句的。主要差别在于，高温可以激发香味；高温可以造成蛋白质变性，不同部位的收缩程度不同，口感会形成差异化的层次感；高温可以让闭壳肌失活，可以整根铲下来，不浪费；高温可以杀灭有害微生物，保障健康。

2、高压锅做法有什么区别？

味道区别不大，但我更推荐微波炉，因为口感更出色。微波是直接加热水体，而不需要慢慢进行热传导，加热效率高，蛋白质变性速度快，停止加热后食材暴露在空气中会迅速失温，不会焖在锅里，口感会因遇冷收缩而明显变得更加紧致。当然前提是你的水量加得刚好，水多热得慢，失温也慢，会更像高压锅；水少热得快，失温也快，最容易出现复合口感；水太少淹不过生蚝则容易烤干，那又是另一种偏干有嚼劲口感。

3、为什么新鲜的生蚝也有腥味？

大概率是内陆人不经意间养成的嗅觉习惯。举个例子，有人觉得榴莲很香，有人觉得榴莲很臭，而榴莲成分太过丰富，它并非只有一种气味，而是很多种气味混合在一起的。有的人对香味敏感，闻着就香；有的人对臭味敏感，闻着就臭；有的人对香臭都敏感，那闻着就难以形容…没接触过鲜活生蚝的人，闻到的往往就是鲜香味和臭味混合在一起的气味，吃到的往往也是并非绝对新鲜的生蚝，因此对鲜臭混合的气味与不够新鲜的生蚝进行了认知绑定，成为一种味觉保护的习惯。这样的群体，闻到海鲜的香，也会“宁枉勿纵”地判定为“腥”。

4、蒜蓉酱烤生蚝不好吗？

简单来说，新鲜生蚝本身气味单一，而且并不浓郁，蒜蓉酱气味太大，除非生蚝已经发生腐败，否则气味上根本打不过蒜蓉。蒜蓉的问题不是好不好吃，而是喧宾夺主。

5、生蚝买回来要先冻吗？

不用。那只是一种保鲜手法。

马上吃不冰、明天吃浅冰、下周吃深冰。

6、微波炉加热不会炸吗？

加水没过壳缝，确保闭壳肌松动后水可以涌进壳里，那就不会炸。牡蛎本身含水量也比较高，只要食材本身周围有水，就不会炸。

7、加水会让壳内原有的海水流走，味道不会损失吗？

这个看个人理解了。壳内是海水，本身盐度就比常规菜品要高，咸味远大于鲜味。而且海水离子成分复杂，有时会有涩味。即便是要喝，我自己也会另外掺饮用水，不会直接喝。
