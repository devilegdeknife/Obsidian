# 为啥小孩会被叫做"吞金巨兽"，"行走的碎钞机"?
[内容链接](https://www.zhihu.com/question/495647821/answer/1890423647065314166)

以北京为例。

父母都是高学历尻一代，有了北京户口，孩子身份证110开头的。

刚结婚，四个老人掏空了钱包，买了大兴的豪宅，月供一万七。

婚礼五十万，彩礼二十万，买车二十万。都是男方出大头。

婚后男方精子活性低，才20%是活的。去北医三院做全套检查，包括男方脑袋上插电线检查生殖系统反映状况，女方输卵管造影，花了好几万，都不走医保。

开了生精胶囊和换少胶囊，一个月一千五百元的药费，吃了俩月，嗓子眼冒烟了，终于怀孕成功。

怀孕十二周了，想起来领叶酸，去社区医院，人家说已经怀孕了不能领，要备孕前三个月领才免费给。不得已去药店买叶酸，结果花了好几百被营养品割了韭菜，其实管用的十块钱一瓶。

过几天，出血了，去医院检查，先兆流产，打保胎针，孩子保住了。当然，医院的部分可以报销，就不说金额了。

早产了，出生时，父亲和奶奶给产科大夫包了五千的红包，这回大夫收了，本来床位没了，结果给临时抬来一张床。

母亲大喊雅蠛蝶，尻二代出生了。父亲在门外焦急地上淘宝找大仙，花了五百块买基础算命套餐，为孩子起名字，推算生辰八字。大师顺带给父亲算了命，说他中年大发，然后他龙颜大悦，打赏了大师五百块。大师背后骂他是傻13。

出生当晚，医院领导小舅子的老乡来推销护工服务，一晚上五百，结果护工睡得比猪还死沉，五百块白花。

第二天直接入住高档月子中心，怀孕六个月的时候就货比三家，花了十万订的高档别墅式。

进去后，各种另外收费。母亲没奶，要单独花钱请月子中心的催乳师，催乳师在A罩杯附近捏了捏，就要六百，多捏几次算加钟。

没想到别墅是临街的，晚上有鬼火少年摘了消音器玩摩托，说的一对一看护，其实是一对多。催乳师还说漏嘴了，楼上可以给男人喂奶。

父亲请了陪产假两周，结果第一周结束就被叫回去通宵加班，然后当月绩效被扣了一半，到手工资少了好几万。

煎熬的二十多天过去了，尻二代终于回家了。母亲经常回奶，孩子姥姥农村的比较相信传统文化，从山姆买最贵的有机红花生，从同仁堂门脸买几百块一斤的通草，熬五红汤和通乳汤。

还是没奶，只好买进口配方奶粉，几百块一小罐，只能喝两个星期。

母亲在月子中心和回奶的双重打击下，产后抑郁了。最开始是打砸家具，到后来是暴打奶奶和姥姥，遥控板被摔碎了好几个，电视也被砸了，姥姥也去医院缝针了。好几万又没了。

父亲觉得尻二代要有好身体，所以各种疫苗注射都把自费的给注射了，每次自费注射至少二百元。

尿不湿和奶粉买的进口的，奶粉四百元一罐。尻二代乳糖不耐受，喝了奶粉不停拉稀，尿不湿质量太好，看不出来是拉稀。直到有一天，尻二代便血了，尿不湿保不住了，这才紧急去医院。住进了icu，又花了三万。出院后，开始购买低乳糖奶粉，还是四百元一罐。

终于有母乳了，母亲怀疑乳糖不耐受是父亲遗传的，挤了点母乳让父亲喝掉。母亲喜欢吃辣条，吃了一包辣条后，挤出来一小瓶母乳。父亲喝掉之后，腹泻了一周，去医院打点滴，花了一千。这下石锤是父亲遗传的乳糖不耐受，属于隐瞒身体异常，罚跪搓板。

尻二代好不容易两岁了，开始用进口的松茸粉，纯有机酱油，进口无污染水果。其中有机酱油五百元一瓶。

开始学走路和说话了，父母工作忙没时间，又觉得爷爷奶奶姥姥姥爷都是老农民不懂现代教育，于是开始了找教培这个不归路。花了一万二，找了个教培机构，教尻二代走路和说话。教培的教学方式是，让尻二代看着小动物走路，并同时放音乐，美其名曰情景浸入教学。

尻二代三岁了，报名了国际幼儿园，上的是vvvip班，由金发碧眼的英国人授课。保教费一万一个月，餐费一百一天。听院长讲，他们的食材都是空运过来的，和富二代们吃一样的东西。

教委不让幼儿园提前教授小学知识，国际幼儿园只好偷摸教，为了应对教委检查，幼儿园准备了一套反侦察体系。一旦听见风声，教师们和幼儿们都会第一时间将教学资料藏起来，幼儿们统一说没学过数学和英语。

父母觉得光是幼儿园还不够，又在课外花了一万八报了个幼小衔接班，学的是一百以内加减法、汉语拼音、一千个汉字、26个英文字母。

尻二代上幼儿园大班了，该买学区房了，把大兴的豪宅卖了，再添了一百万，去西城买三十万一平的老破小，还没电梯。就这样，父母刚处理了大兴豪宅的贷款，又背上了西城老破小的贷款。

上幼儿园大班之后，该上兴趣爱好课了。少儿游泳二百元一课时，买了二十课时；少儿钢琴二百元一课时，买了二十课时，需要自备钢琴在家练习，物理没地方放钢琴，又买了一间地下室，花了四十万，用来放一万块买来的钢琴；少儿美术课一百五十元一课时，买了二十课时，同样高档画板、颜料和画笔都存在地下室里；少儿轮滑三百元一课时，含意外保险，买了二十课时；少儿平衡车一百元一课时，含意外保险，买了二十课时。。。还有格斗课、跑酷课、思维课在排队中。。。

课外活动也得丰富多彩，为了拓宽孩子眼界，每周听交响乐一次，每次门票二百元；每个节假日要坐飞机去旅行，每次五千至一万不等。还没上小学呢，已经踏足过好几个大洲了。

太啰嗦了，简略点说吧。假设这次教培没跑路，妈妈也没去吃酸黄瓜。

幼升小那个暑期报了幼小体验营，这不是幼小衔接，这是体验小学生活的夏令营，模拟上小学，美其名曰让尻二代提前适应小学的氛围，费用一万三。

每年开始，尻一代都花一万五左右上各种父母训练营，训练营内容就是教怎么当合格的父母。训练形式类似传销，训练营上安利各种产品，每次跟着训练营喊口号做拓展运动完了，再花几千块买产品。

上小学了，奥数课五百元一节课，英语外教课八百元一节课。这都是团购的课程，单独一对一价格更高。

钢琴、美术都要考级，没有停下学习或者说交钱的步伐。平时为了当班干部，贿赂全班同学，每次家长会，挨个家长发礼品，每次的礼品金额不低于二百元，一个班四十人。本来也想给老师送礼品，但是老师坚决不收。

老师吃教培回扣了，鼓励全民学编程，最开始学scratch，花了一万五千八。慢慢发现，这玩意纯智商税。逐步了解到班里有大神在搞信息奥赛，于是乎，想到了程序员的高收入，狠下心给尻二代买了一台外星人笔记本，然后花了两万报名信息奥赛课程。结果尻二代连C艹语法都学不利落，看见递归就蒙圈，信息奥赛这条路告吹。不过外星人笔记本上，多了个STEAM。

对了，上学要接送，尻一代开S60送孩子上学，但看到同学们的父亲都开A6，觉得很自卑。尻一代发现自己孩子每天不让自己把车停在校门口，要停远一些别被同学看见，觉得不能让孩子留下心理问题，咬咬牙贷款买了辆All Roads，每天风风光光地接送。

尻一代只是能送孩子，老人都去西天了，只好雇人去接孩子。雇的是家政公司的人，每次五十元，如果需要提供晚餐，每个月包月价格一千五。

小学三年级开始，要买学习机了，买了台旗舰机一万二。同时购买题库，和各类网课，一次三千元左右，每个学期买三到五个套餐。

班里的A娃们需要陪读，家长都有工作，那就得请影子老师，影子老师一个月15800元，嗯，这个我觉得本例的尻二代还是不要陪读了，他大体是好孩子。

上初中了，教培升级了，数理化全要补课，一节课一千六一课时。兴趣班停了，但游学出现了，一学期游学一次，一次五千元。

基本每年在尻二代教育身上的开支在二十万元左右。

中考前，尻二代体测不合格，一千米要跑四分零三才是满分（老子当年班里倒数第一跑四分零三，真不知道尻二代怎么体质这么差），半个班都跑不进四分钟。请教练来教跑一千米，一万二一期课程，就是带着到免费的体育场里跑圈。

中考完了，尻二代有惊无险地考上个中游高中。嗯，假期没闲着，花了一万五去学初高衔接课程，就是把不等式、化学方程式、受力分析讲讲，防止他们上高中受不了。

。。。

上高中了，教培继续升级迭代，这个场景过于惨烈，暂且不提。

开始攀比了，以高中家远为名义要智能手表，apple watch两千多块钱一个。校服还要穿，只是鞋帽眼镜书包都要潮牌的，否则会被同学瞧不起。于是乎，书包要古驰的，眼镜要巴宝莉的，鞋要鬼冢虎的，不给买就不是亲爸妈，不给买就离家出走割手腕。

高中参加模拟联合国社团，尻一代觉得当初自己的村校只会做题不利于全面发展，于是欣然同意。只不过每学期出国考察一次，一次两万元的费用交给模联公司，然后做红眼航班去美利坚的联合国总部转一圈，在个没人的会议室里，自己跟自己讲讲话。

高中有各种舞会，要买西装，纠结是阿玛尼还是范思哲，最后选了个荷兰品牌，八千块一套在班里不算丢人了。看着尻二代在舞会上，搂着穿礼服的女同学，村里长大的尻一代无必欣慰，北京真是好啊，孩子成长就是快，就是有见识。

尻二代开始不满学校的食堂了，于是对尻一代谎称学校食堂是预制菜，尻一代瞬间暴怒，本想去拉横幅，但是公司耐心地敲打尻一代，告诉他们不要去，否则，嗯，大家都懂。尻一代手里没资源，已到中年，失去平台等于只能去当保安。没办法，只好每个月给尻二代一千五百元，让他点外卖。每天中午，学校的大门里，无数的手紧紧握住铁栅栏，学生们带着狰狞的表情，仿佛便秘一样青筋暴起要捏碎这一根根铁柱子。等着穿黄袍的御风者打开保温箱，一盒盒大鱼大肉拎出来时，尻二代满意地笑了。这幅场景，像是生化危机里的尸潮，也像是监狱里的犯人渴望明媚的阳光。

尻二代继续以各种理由要钱，比如要买必读物，要买学习用平板，要参加学校的社会实践课。尻一代已经招架不住了，已经给了那么多了。结果发现尻二代偶尔身上有烟味，甚至书包里发现了杜蕾斯。

不想往下说了。。。

这是本人和小区里一个高知邻居交流得知的，本人是将他们的口述内容改造了一下，把他们孩子同学的经历加上了，去掉了他们去维权的过程。
