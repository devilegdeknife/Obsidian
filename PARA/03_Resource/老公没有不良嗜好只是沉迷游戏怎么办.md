# 老公没有不良嗜好，只是沉迷游戏怎么办？
[内容链接](https://www.zhihu.com/question/35877248/answer/112319949206)

作为曾经的游戏寡妇，想答一下。

谈恋爱时我非常反感老公沉迷游戏，他玩一局中途不能停不能退出，说话不理叫不答应，这个是我最讨厌的。因为他打游戏这个问题我们大吵小吵无数次。

他一直试图让我陪他一起玩，给我讲什么星际虫族人族，带我看游戏相关的电影，我的态度一直都是拒绝拒绝拒绝。我对游戏完全没兴趣，最多偶尔玩一玩平板打发下时间，他也会陪我玩，三国杀阿数独什么的。

有一次聊起来他说他小时候最喜欢的游戏是仙剑奇侠传，我小时候也玩过，还爱看电视剧，就这样我们下了一起玩，开启童年美好回忆。还有一个游戏叫饥荒，它在我平板里就是偶尔打开逛逛地图捡花捡草吃吃野果的休闲小游戏，我从来没有活过冬天。后来看老公玩，简直打开了新世界的大门，原来这个游戏这么丰富有意思。平板只能一个人玩，见我起了兴趣，他开始在我的电脑上下steam安装联机版，这样可以我们两一起玩，他负责外出采集打怪，我在家种地做饭，他带着我一个人打过了四季大boss从地上打到地下，我就穿最厚的甲跟在后边捡捡东西。就这样在老公的熏陶下，我慢慢放下敌意和偏见，虽然也只对特定的游戏有兴趣，他玩的大部分我还是无感。

后来又有一次我无意中刷到了星际争霸李培楠夺冠的新闻，看完也觉得很热血很感动，就主动和老公聊起这件事。他大概做梦也不会想到有一天能我嘴里听到这些，那一天我能明显感觉到他的激动，看我的眼睛里都闪着光。

结婚后买了PS，到这里我觉得已经好多了，起码游戏碟可以随时暂停，不会出现叫不答应的情况。我也时不时给他买新的游戏碟让他玩，有意思的我就边刷手机边跟着看看。

最后的转变是在我怀孕生娃期间。出门不方便，他开始发掘一些有意思的双人游戏，鼓动我在家娱乐放松一下。我们一起玩双人成行，在每个哄睡完孩子的晚上。每晚一关，一个多两个小时，没有白天现实里孩子家人乱糟糟的问题，只有完全属于我们两个人在虚拟世界里的大冒险。我过不去的地方他耐心等我教我，有时候也嘲笑我，我们拌着嘴，在游戏里互相攻击，追逐打闹。最后打通关的时候我哭得一塌糊涂，我重来没想过有一天会有一个游戏让我这么投入。它陪我度过了产后最难的那段时间，也完成了一次从虚拟世界到现实世界的情感修复。

想一想我真的是不能接受游戏吗，我只是看不惯我老公那些与我无关的快乐，特别是在我不怎么快乐的时候。反过来他因为我的“纵容”变得更沉迷吗，我尊重他的私人时间，相信他能把握好放松和贪玩这个度，他也会事事以我和孩子，以家庭为优先。现在游戏并不是一个横在我们之间的问题，只是可以带给我们共同语言和提供快乐的娱乐工具。

哦对了，我老公还爱抽烟，偶尔也喝酒打牌打麻将和狐朋狗友混在一起。这些也不是什么大不了的滔天大罪吧，只要他自己能掌握好度。我爱旅游，他懒得出奇不爱动也愿意陪我上山下水，我沉迷过露营，他也会买贵贵的装备支持我，现在一年去不了两次东西丢在家里吃灰，比起他我的爱好更败家，难道女的花大几万美容旅游是天经地义，男的打牌输一两千这个家天会塌？难道你不爱逛淘宝没有爱好喜欢做苦行僧就要要求他人也放弃一切陪你清修吗？还有一点，我老公教会我的，快乐就是快乐没有什么高低贵贱。滑雪打高尔夫的快乐就高级一点，打游戏玩牌的快乐要低人一等？抽烟喝酒打麻将是臭屌丝，抽雪茄喝威士忌打德州是成功人士是吧。人生已经这么难了，想做什么就去做，哪来那么多双标偏见画地为牢。容错率高一点，家庭氛围就会好一点，人活得也会轻松一点。

与其围追堵截，一味要求他人为自己做出退让和改变，不如自己先抱有积极的心态去尝试接纳和转变，你使出的正向的力大概率也能收获正向的反馈。

———————————

看的人很多，想追更几句。

老公没有知乎，把评论讲给他听，给他笑成了耐克嘴。

我态度的转变是我们长达几年磨合下来的结果，即使在最开始我完全排斥抵触的情绪下，老公也依然不放弃对我洗脑，找每一个机会激发我的兴趣，试图让我融入。

比如那种可以自定义角色名字的，他就会把主角设置成我的名字，把神兽设置成我家狗子的名字。

比如玩2077那种可以自定义角色的，他就会让我来捏脸选衣服。说我在玩赛博暖暖。我搞半天结果一进游戏一看，呵呵是第一视角。

他会找我感兴趣的东西和游戏联动。比如我喜欢露营他就带我玩饥荒，电子荒野求生，然后我们去野外的时候还真的找了牛粪来烧。

他大部分那些打来打去的我都没兴趣，星际我从来没看明白过，聊李培楠是想讽刺他这么爱玩怎么没打出人家这个水平（u1s1能把热爱发挥到极致的人真的很厉害）玩巫师3，边玩边跟我搞解说，遇上个什么怪，现在要去哪做个什么任务，有对话故意问我要回答什么，问我要不要帮助他或者要不要接这个任务，我再无感也知道了里面有个随叫随到的马叫萝卜，有时候也会阴阳他找你的萝卜去。

题主其实可以主动去了解一下，起码知道他玩的是什么，他喜欢这个游戏的点在哪里。

另外题主最关注的是老公的身体健康，他11点下班玩到凌晨1点，工作8小时以上，游戏2小时，恕我直言真正影响你老公身体健康的是要上到11点的班。但是赚钱养家是必须，娱乐时间可以舍弃，那他的心理健康怎么办？

我老公从小父母离异，游戏是他童年时光唯一的陪伴。有了儿子以后，常常在他身上看到童年老公的身影，如果我不能穿越时光去抱一抱那个独自坐着打游戏小小的他，起码现在我可以坐在他的身边。支持和陪伴不就是家人最大的意义吗，要不然结婚干嘛。
