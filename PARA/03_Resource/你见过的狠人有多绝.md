# 你见过的狠人有多绝?
[内容链接](https://www.zhihu.com/question/538690869/answer/1961264910878095351)

讲一个17岁赚了至少90w，然后无缝衔接回去参加高考的真人真事，狠人中的神人，拍成电影保证一票难求。

这家伙没有任何一点机制，全是爆表的数值，也是我唯一佩服的同龄人。

这家伙不怕死，有两句经典口头禅：上策赚钱回家、中策蹲两年、下策大不了去世…

以及“这么简单还叫事？？”

先声明一下，不要轻易关注我，本人成分复杂！！！！

## 梦开始的地方

时间拨回2020年，我上高2，当时成绩还可以，历次考试平均540-550＋，本打算考个211（985没机会），后来不知怎么脑抽，觉得就算考进211，这辈子也不会好到哪去，索性高三直接开摆，请假一年，2021年上半年只在学校参加了两次考试，一次一模统考，一次高考。

和他是2020年底闲鱼上认识的，他比我小一岁，05的，名字里有个“天”字，我后来一直喊他“天哥”，因为他能力太强，我慕强，姑且尊称为哥。

20年底的闲鱼，恰是疯狂发展的时代，尤其数码圈乱成一锅粥，当时iPhone12刚出，标志性的三镜头引得无数小年轻趋之若鹜，在那年，手揣一部12pro max就是潮。

我自然买不起，那部手机顶我妈半年工资，于是我百无聊赖刷闲鱼，刷到一个标价1599的12pro max，他标题就是“智能手机”，没说是苹果，但界面和外观都和iPhone一模一样，我瞬间来了兴趣。

其实五年前的闲鱼，华强北部分商铺小作坊已经开始批量出货这种伪苹果。小年轻没钱，又虚荣，总想给自己脸上贴金，还真有不少人购买。

而他的产品，是闲鱼最便宜的，我本想着买一部，研究一段时间，出售给我们班装x哥，结果在闲鱼和对方深入交流后，发现我俩年龄几乎一致，三观也差不多，家里都没多少钱，后来又连着聊了好几天，越聊越爽，相见恨晚，打算见面。

我向他订了一部伪12pro max，本来也就发个快递的事，为了和我面谈，他居然坐大巴从老家河南来到我家门口的人民广场给我当面交货。

那时候是口罩元年，也是整个社会最死气沉沉的时候，于我而言，能出我们市区都难，他硬是用了各种办法坐了一整天大巴来找我了。

天哥老家是河南一个GDP很低的市，其父在村里骑摩托车和施工队三轮车在路口相撞，骨折较重，大几万治疗费花光全家积蓄，后期医药费还问亲戚借的，其母在村子西边的县道上经营一个二元百货店，倒闭了，家里还一个妹妹要养。

他一直觉得自己是天才，奈何家里不让他试试，天天把他闷学校里，直到20年秋，他上初三的时候（他上学比同龄人晚一年），父亲骨折卧床，家里才允许他休学去赚钱。

他把自己书桌改成了工作台，从他老家电脑城收零件，纯手工组装高仿iPhone，再用安卓的主板，再一通捣鼓把界面替换成iOS的。

不依靠任何人。

天哥不知是刚踏入青春期还是营养略微不良，身高是硬伤，头发短短的，整个人看起来不太聪明。

话不多，照现在的说法应该是个i人，我也是i人，但两i相遇必有一e，所以当天一直是我在找话题。

接过他手中的机子，我捣鼓半天，发现几个不得不讲的问题：1.做工对比华强北的货有明显差距2.镜头模组是硬伤，长期用pro max的人一眼能看出不对劲（通透度不好）3.屏幕黑边太明显4.流畅度比较一般。

天哥给出的解释是：1.华强北那边一些人做这种组装机已经是老手了，他刚起步暂时没法比2.哪怕2500价位的组装机，背面三摄模组也是这个水平，想要做到正版太难3.屏幕采用安卓屏里的低端产品，这个没办法4.流畅度这一块，就连华强北的组装机都无法避免。

我当时对他说：“我说这些的目的，不是想给你砍价。如果咱俩要合作，就必须把机子质量做的非常好，你只要出货合格，销售这一块都交给我，闲鱼收款账户绑你的。”

那时候我还没成年，没有自己的银行卡账户，他更没成年，索性用他爹的支付宝收款。

他也觉得我言之有理，而且我向他保证，只要质量过关，出多少台我帮忙卖多少台，卖不出去的我自己收。

我之所以敢这么保证，是因为当时组装pro max这个机型在整个闲鱼刚开始走上坡路，虽有几个商铺持续出货，但没有一家独大，而且客户越来越多，当时闲鱼商品界面下面就已经有“我想要”三个字，上面会显示有多少人点击这个界面并进入商家对话。天哥昨日发布的帖子，今天已经有七八个人咨询。

对于天哥来讲，他唯一的优势就是便宜，整个平台最低价。因为质量问题，也不得不便宜，单机利润始终不如华强北一些专业户。

而目标客户第一追求却是装x，说白了不能让路人或者身边人一眼看出机子不对劲。

所以天哥那时候的优势其实并不大，我们总结了一下平均每5个咨询者里，有2个已读不回，有3个会详细聊，最后只有1个会进入砍价阶段，至于最后要不要，概率又要打50%。

我们需要快速走量，需要利润，哪怕以后我们想干其他更有趣的事，总归要有钱再说。

机子花我1599，这笔钱还是我爹妈忘记收取的压岁钱，把机子拿在手里仔细端详，它带给我的思考确实比之前在学校上晚自习的思考要深。

（最初的机子，可以明显看出质量不太好，屏幕黑边明显）⬇️

## 开挂的道路

我对20年底的唯一印象只有俩：灰蒙蒙的天、满街的白大褂。环保导致许多工厂关停，或者突然被拉闸断电，路上没几个人，沿街商户没几个开门的，还要不断给房东交房租。许多人工作都停了，一出家门，天上挂着的不是太阳，而是危机二字。

我爹那时候步入半失业状态。

我时常想，如果我也是富二代，只需要安心学习，根本不用担心往后考研考公找工作，不需要为人生许多道路而焦虑。

奈何我不是。

天哥真就只身一人去福建和浙江转了一圈，找福建一家工厂替他生产了第一批100个镜头模组，这个模组唯一作用就是让机子质感跃升，和原装几乎没区别，至于摄像，还是用的安卓。

我很震惊，那时候口罩盛行，他不知坐了几趟黑车，高速国道县道走了个遍，从河南一路到温州，甚至不知用啥方法，说服一位大巴司机让他藏在大巴车侧面放旅客行李的侧备箱里。

他惯用组合拳，那两年为了四处跑，一般先展示绿码，再对司机出高价，再软磨硬泡，一个不成再找一个。

至于他那个破绿码，我都怀疑是假的。他反复咳嗽发烧了几个月，直接五毒不侵，不打一根针，阳了还活蹦乱跳。

做样品的厂子虽然生意不好，但最低量也是50或者100，少了不做。

怎么让模组和后置摄像匹配是天哥的事，我只需等他弄出来第一个机子，上架闲鱼卖即可。

过了几天，成品拿到手，起码看背面的质感确实提升了，但壳子还是透露一股廉价。我们在闲鱼上架了二十几部机子，姑且称为第二代，稍微提了点价格，半个月卖光，还余下几十套镜头模组。

天哥坚决暂停，他要一鼓作气把机壳也改进到完美，21年初，他千方百计搭上一辆从苏北往上海运蔬菜的货车，坐人家副驾，从苏州下车，我当时给他发消息，结果他又搭车去了之前订镜头模组的福建厂家，下单了300个，还给人家提改进意见。然后一路跑到深圳华强北附近。

那俩月我们一直手机联系，他去华强北附近的下布庙小区拜一个专门做组装机的大爷为师，愣是装小白交学徒费搁那帮人家干了一个月的活，他说那年华强北附近挺乱，虽然人流比平常少了很多，但也无法阻挡大家向钱看的脚步。

师傅夸他学得快，悟性高，他把师傅的手艺、常用的工具、小机器等全学到脑子里了，然后不辞而别。

说难听点就是把师傅踹了。

而后又在深圳和佛山乱转，收到一些港版iPhone拆机壳，质感果然不错，稍微抛光打磨一下就焕然如新。

一番奔波，他于21年4月回到河南老家，那段时间他天天跟我发消息吐槽说家里让他回去上学，但他把自己关屋里硬抗。

她妹从县里上初一，和他一学校，学习很上进，学校也是公办的，自然没多少书本费，可他妹妹坚持去城里的一中，而且也确实有那个成绩，他老家那地方补课很盛行，家庭负担就重些，这就是他坚持要多赚点钱的另一层原因。

而且他爹虽然能站起来走路了，但腿上有块钢板还没取，到时候估计又是一笔花销。

在此期间，我虽然帮他又卖出去十几台机子，赚了2万，但也没啥成就感，我非常期待平均一天至少两部机子的效率。

他从华强北还带回来一部当地最高水平的pro max组装机，闭门研究，反复尝试如何组装的更严丝合缝，偷师学来的一些小机器很好用，他直接从淘宝下单相关机器。

我本身不太懂手机的许多零件，也不懂组装，都是他在钻研，他负责批量生产，我负责卖掉变现。

5月，他说神功大成，给我展示手上三十多台亮丽如新的机子，这便是我们口中的第三代。

售价2000，卖出一部机子我俩能有650的利润，而且平均保底一天卖一台，好评如潮。

天哥只身一人去南京秦淮河那边搞了办公室，现在租房都要身份证，签合同，那年线下租房市场还是比较魔幻，无需身份证，而且口罩影响，中介揭不开锅，租客明显减少，中介连身份证都不看，明知他未成年，还帮他租房子。而且天哥深知有钱能使鬼推磨的道理，南京老城区部分地带中介费当时是半个月房租，也就是说，如果整租2500/月的房子，至少要给中介交1200，天哥不按套路出排，给中介大妈单独加了好几百，因为他是未成年。

我笑他有家不回吃这苦干啥。

他说，这样方便来回去福建甚至深圳。

果然，他还是比我想得多。

“那你为啥选南京那边，还选个老城区窝着？”

天哥：“风景好看。”

我也是醉了。

## 一路干到底

我们如何定位自己的“产品”？闲鱼当时几家竞争比较大的组装机店铺都直接打iPhone标签，没人管，卖的很火。我们为了对得住顾客，决定采用“苹安手机”这个名，配图都是我们自己拍的，顾客一看就懂，对我们来讲也更合规，起码不会标榜iPhone来诱骗，价格也是很公道。

屏幕的黑边让我俩在21年上半年很头疼，天哥甚至要去深圳那边批量采购港版iPhone拆机屏幕，但我们一算，成本拉爆了。

我试探问：“要不屏幕咱不改了？”

天哥：“不行还是要改，我试试。”

最终的解决方案是，多花点钱费尽心思找了完美替代的一款国产屏，可折腾死我俩了。

至此，我们机子的质感压过华强北几个卖家一头，价格还更低一点，我们终于有底气在闲鱼里加上那句“保证图货一致。”

这句话是我们除价格外，相较对手的一大明显优势，因为不少卖家不敢说，他们一般用真iPhone pro max照片，然后卖低质感机子，虽然成交量不少，但退货的也有一群。

宜将乘勇追穷寇。我建议他要做就做到位，把顾客体验感拉满，包装、充电器等一切向原装看齐，每个机子让顾客任选三个不同款式手机壳。

天哥觉得有理，还真就从广东找熟人订购了很多套包装盒、国产苹果充电器，他还把耳机给搞定了，可笑的是，耳机是从华强北采购的。

他还专门去淘宝买了个封塑机，把包装外面做好塑封，给每位顾客奉上开箱体验，这便是我们口中第四代机子。

他招了好几个员工，那几个员工也是人才，都发自内心钦佩天哥极强的工作能力和拼搏精神，我记得第一批员工好像是3个人，一个是南京林业大学快毕业的本科学生，一个是研究生（哪个学校的我早忘了），一个是网吧辞职的网管。

还真别说，南京大学生太多了，许多大学生白天上完课晚上就出来兼职。

我俩当时有分歧，我认为很难和那位快三十岁的网管共事，天哥坚持认为，网管熟悉本地网瘾群体，甚至有部分人联系方式，如果能打通网管这条线，可以借对方朋友圈宣传机子。

我只能鼓掌。

他们负责出货，我负责在平台上卖，订单信息发到他们手机里，他们发完货再把单号给我，我填在闲鱼上。

仍记得南京林业大学那哥们对天哥非常崇拜，天哥当时16岁，那位学长21岁，跟在后面像小迷弟，笑死我了。

（多次改进后的机子，已经非常完美）⬇️

## 疯狂的闲鱼、激烈的斗争

我确实起到了非常关键的作用。天哥和其他人擅长出货，我负责变现，各有所长。

当时的顾客基本都是爱慕虚荣的小年轻，我们会明确告知他们，这是“苹安手机”，顾名思义，苹果的外在，安卓的主板，系统表面是ios，底层还是安卓。哪怕我们每次发货前都给买家实拍图，但还有一部分傻冒觉得我们没法100%复刻iPhone就是我们的错，拿到手装几天x就想退货。

这群人可比现在拼多多只退货不退款那帮人高明多了，常年混迹闲鱼，闲鱼水深得很。

如果把闲鱼比作海洋，这群人起码躺在大西洋海底，但我可是躺在马里亚纳海沟的水平，闲鱼被我玩透了。

对于这种恶意坑机子的人，我早有妙招。

当时闲鱼退货有两种，只退款不退货和退货退款，有些人看前者走不通，就选择糟蹋完机子后退货退款，他们先申请，我需要在xx小时内同意或拒绝，我直接点拒绝，然后对方会第二次发起申请，我会重复上述动作。

这时对面买家会直接申请闲鱼小助手或者人工介入，买卖双方开始磋商、举证，争论白热化。

对方总会说，我们的机子无法100%做到和iPhone一模一样，我也是无语，明明下单前都给你说了是苹安手机，安卓主板。

但对方会执意退货。

面对这种焦灼情况，平台会启用“闲鱼小法庭”，买卖双方需要在一定时间内（好像是几小时）提交有利于自己的文字和图片去举证。

时间一到，双方提交通道关闭，闲鱼会随机抽取平台上n位用户来参与评审，认为我有理的给我投票，认为买家有理的给买家投票。

我总会在提交通道关闭前最后两三秒用一系列举证和实拍图封贴，把自己的发言始终挂在最上面，让参与评审的人点进来第一眼就能看到我丰富的论述。

对手被击溃，大部分人都选的是我正确，闲鱼小法庭直接宣判我胜利，不用等对方确认收货，货款直接秒到账。

最后闲鱼默认给我好评。

（下图是后期修改过的闲鱼法庭，之前的流程是，只要判定卖家胜出，则不会给买家申诉机会，直接打款到账⬇️）

有段时间，竞争对手派人来恶搞我们，我最多一天打了4场闲鱼小法庭，全胜，成功狙击对手，未尝一败。

南京林业大学那位哥们认为自己学的是经济，想试试，结果差点打输，最后是我接过手机费了好大劲扭转局势。

后来他也成了我小跟班，我当时17岁半，他大四。

面对不同顾客，我可以做到只要进入议价阶段，就有十足把握让对方下单，一旦对方下单了，不论对方想不想收货，有没有拆封，都必须收货了。

我的流程是这样，完全遵守平台规定：对方下单，我秒填一个单号，先把货搞成“已发货”，第二天再去驿站实际发货，把单号修改回来，这样哪怕对方突然反悔拒收也必须打闲鱼小法庭，而一旦进入闲鱼小法庭，我未尝一败。

所以只要下单，就必须收货了。

没办法，一些人太虚荣，我这是好心给他们上课，没收学费就大发慈悲了，而且平台判定不违规。

有段时间我们发了货就期待打闲鱼小法庭，某天一个小法庭没打，瞬间一点成就感都没了。

22年天哥又招了几个新员工，最早跟我俩的南京林业大学的那哥们因为进步飞快，我俩单独给他开辟了卡西欧业务，让他打入各种中学校园群内部，低价抛售各种款式卡西欧，只要走量即可。
