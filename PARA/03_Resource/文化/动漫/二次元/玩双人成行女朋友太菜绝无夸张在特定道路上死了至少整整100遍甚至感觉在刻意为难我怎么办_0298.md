# 玩《双人成行》，女朋友太菜，绝无夸张，在特定道路上死了至少整整100遍，甚至感觉在刻意为难我，怎么办？
[内容链接](https://www.zhihu.com/question/452785716/answer/99171902801)

兄弟，你这个问题我太懂了。

去年我和对象玩《双人成行》，她能把小梅玩出《掘地求升》的难度。

当时那个蜜蜂关，她愣是卡了半小时，我在旁边憋笑憋到内伤。

后来我俩最后反正成了朋友圈里令人羡慕的的游戏CP，

这年头找个一起玩游戏的CP可太难了。

-

这游戏不是《只狼》，你也不是在带徒弟打竞技场。

你女朋友可能连手柄的ABXY都得分神找位置，

你让她在锯条钉子雨里精准跳跃？

这不是强人锁男嘛？

相当于让刚学自行车的人直接骑独轮车过梅花桩。

我对象第一次玩3D游戏时，连推摇杆往前走都会撞墙。

-

所以——心态要切换成“双人散步模式”。

你见过谁散步的时候掐表计时吗？

她死100次你就当看100次专属表情包：龇牙咧嘴、瞳孔地震、疯狂甩手柄…

我后来专门给她做了个《死亡鬼畜集锦》，现在每次看都笑出眼泪。

-

**甜蜜技巧“假装自己也很菜”。**

在简单路段故意掉下去几次，喊“这设计太反人类了”。

当她开始教你“这里要二段跳接冲刺”时，成就感直接拉满。

有次我演过头，对象怀疑我老年痴呆，反过来给我递六个核桃补脑。

让她也体会一把大神的感觉。

-

**甜蜜技巧2“偷换概念战术”。**

钉子雨过不去？就别急着过去：

比谁死得姿势更妖娆，赌下一局BGM播到哪句歌词，预测锯条第几次转会被卡住…

后来发现这游戏里80%的机关都能当游乐设施玩。

有次在冰面滑了二十分钟，就为看小梅各种劈叉摔跤。

别那么快通关，游戏是过程不是结果。

-

**甜蜜技巧“场外援助”。**

准备点零食饮料，每次团灭就投喂一波。

死亡次数超过阈值，立刻切出去看猫片缓一缓。

记住：胃和心离得近，吃饱了怒气值自动减半。

-

**终极甜蜜“反向教学法”。**

手把手教反而容易炸毛，不如让她教你。

“宝宝刚才那波操作为什么如此风骚？”

“这个BOSS换你指挥的话，准备怎么安排我？”

现在我家那位已经进化到能分析关卡设计缺陷了。

-

扎心一击。

你以为她在为难你？其实她更怕拖你后腿。

有次我对象凌晨三点偷偷练跳跃，被我逮个正着。

后来才坦白：“怕你觉得我笨就不一起玩了。”

现在她安利这游戏的金句是：“连我这种手残都能通关，你怕啥？”

-

通关那天别急着卸载游戏。

回放那些死亡片段，比通关动画珍贵十倍。

去年万圣节我俩COS成科迪和小梅，她非要给自己衣服上缝一百个补丁。

理由是：“这都是朕江山上摔出来的功勋章。”

-

记住，这游戏真正的隐藏成就不是“灵魂伴侣”，而是“没分手”。

Steam成就里应该加个“承受伴侣100次死亡微笑依旧”的徽章。

等你们熬过钉子雨，往后现实里的坎就跟踩蘑菇似的轻松。
