# 真正的泰国是一个什么样的国家？
[内容链接](https://www.zhihu.com/question/47366128/answer/112933401414)

讲个故事。

2014年，泰国的玛哈国王，给宠物狗举办生日宴。

期间，他要求年轻漂亮的西拉米王妃，身穿性感比基尼，半祼跪地，喂食宠物狗，并且和宠物狗同吃地上的一块蛋糕。

参加生日宴的众多政界和军方大佬，全程目睹，嬉笑着拍照。

西拉米王妃出身平民家庭，幼年家境贫寒，长大后却因为‘巅峰颜值’而幸运的成为玛哈国王的第三任妻子。

顺利生下王储之后，她更是成了泰国所有平民心中最圣洁的女神。

在那场生日宴上，她全程没有表现出任何不悦，当着一众大佬的面，半祼匍匐在地上，面带微笑配合表演。

因为她知道，这条狗比她金贵一万倍。

![图片描述](https://pic1.zhimg.com/v2-e309699cf03e20d3c4879c360a751189_r.jpg?source=2c26e567)

此前，这条深受玛哈国王喜爱的宠物狗，就已经被授予空军元帅的军衔，可谓一人之下，万人之上，妥妥的狗生巅峰。

全国的军人，包括将军，见了它都要行军礼。

后来，西拉米王妃的半裸匍匐喂狗照片曝光。

西拉米王妃成了整个泰国怒骂的对象，曾经当过舞女的经历也被人扒出来，所有人都骂她肮脏，玷污了皇室的圣洁，要求废黜她。

她失去了王妃的头衔。

被囚禁在曼玫行宫，为国家祈福的名义，剃光头发，出家为尼，过上了孤身一人，洗衣种菜，与外界隔绝的生活。

就连她的娘家人，也牵连进一桩贪腐案，集体入狱。

![图片描述](https://picx.zhimg.com/v2-8126d7f8f0756a3e74c8132578e6a8b0_r.jpg?source=2c26e567)

玛哈国王被评为全世界最富裕的国王，他的私人财产比当初的英国女王还多。

如今，玛哈国王早已经有了新的王妃，以及一位贵妃，三人其乐融融，歌舞升平。

世事就是这样，只管新人笑，哪管旧人哭。

西拉米王妃从2017年至今，已经被囚禁第8年了，她如果想翻身，唯一希望就是玛哈国王突然逝世。

玛哈国王生于1952年，今年已经70多岁高龄。

他和第一任妻子只生了一个女儿，早已经出嫁，脱离皇室。

他和第二任妻子总共生下五个孩子，有儿有女，然后，这个妻子被他以感情不忠诚的名义驱逐出泰国，就连五个孩子也都一起被驱逐。

西拉米是他的第三任妻子，育有一子，是皇室公认的储君。

玛哈国王年事已高，可能是力不从心了，反正现在的新王妃和贵妃，膝下都没孩子。

因此，西拉米王妃当尼姑之前，为皇室生下的那个儿子，至今仍是泰国的皇储，是唯一合法继承人。

国王死了，她儿子继位，她就能重获自由。

但是如果玛哈国王多活几年，碰巧再弄出一个新的王储，那就悬了。
