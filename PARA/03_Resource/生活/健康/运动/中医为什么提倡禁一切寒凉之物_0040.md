# 中医为什么提倡禁一切寒凉之物？
[内容链接](https://www.zhihu.com/question/620680634/answer/1964359622950360821)

刚刚发现知乎上有个问题，为什么人类的体温越来越低？参考链接[为什么人类的体温下降了？](https://www.zhihu.com/question/396083940/answer/1911429690910050263?share_code=gIjEBM1YwlCn&utm_psn=1967184347787429085)

以下是原答案。

人体是人和细菌的共生物。

我们对食物进行抑菌储藏基本上有五个方法：高氧，低温，加热，盐渍和风干。

高氧就是自然发酵的食物，如大酱，酱油，醋，酒等。

低温就是我们现在的主要储藏手段，靠冰箱冰柜冷藏冷冻。

加热不用说了，煎炒烹炸煮炖烧都是这种方法。

盐渍的代表是各种泡菜咸菜。

风干的代表是腊肠，风干肉等。

抑菌过后，不是说一点菌都没有了，而是留下了特定的菌群。高氧过程留下了耐高氧的菌群，低温留下了耐低温的菌群，加热留下了耐高温的菌群，盐渍留下了耐湿的菌群，风干留下了耐干燥的菌群。

这些特定的菌群和人体结合之后就形成了五大体质：耐高氧体质，耐低温体质，耐高温体质，耐潮湿体质，耐干燥体质。

这五种体质，在黄帝内经里称之为阳寒暑湿气五种阳气类型。

从氧气的获得效率来讲，阳寒暑湿气五种体质呈逐渐下降趋势。耐高氧体质氧气获得率最高，耐干燥体质的氧气获得率最低。大家如果在大风天走过路就知道，在大风下你根本不敢大口呼吸的，因为风会灌进你身体里，吸进去的是气，灌进去的则是邪气，是会生病的。

氧气的获得率决定了你的心肺功能，而你的心肺功能决定了你的健康状况。所以黄帝内经里说寒暑湿气四种体质折寿而不彰。

这五种菌群实际上我们都需要，因为我们确实有的时候需要耐寒耐暑耐湿耐气。只不过分不同的地方和气候而比例不同。冷的地方或者冬天我们体内的耐寒菌就多，热的地方或者夏天耐暑菌就多一些。每到换季的时候我们身体可能会不舒服，就是身体内的菌群配比在进行调整。吃应季的蔬菜水果有助于这个过程，因为它们上面就有适应这个季节的菌群。这五种菌群的和谐就叫“五气和”。

虽然五气和，五气我们都需要，但老大是谁？古人认为，只有耐高氧菌群当老大的时候，人体才是健康的，其它菌群当老大的时候就是人体呼吸功能下降的表现，这个状态被称为“阳气失其所”。

那么怎么能让耐高氧菌群当老大呢？首先就要从食物上少吃寒凉之物，因为从阳跌落下来的第一站就是寒。你把老二看住了，老三老四老五就不足虑了。

其次，要打气功固阳气。我们在冷的时候就会打哆嗦。打哆嗦的时候，说明体内已进入寒气模式了。如果你故意打哆嗦，你会促使身体进入寒气模式。反过来考虑，如果你做出阳气充足时的身体姿势，你就能帮助身体吸纳更多的氧气来保持阳气模式。五禽戏易筋经都是很好的气功。
