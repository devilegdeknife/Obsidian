# “父母都走了之后，兄弟姐妹就散了”基本不来往了，是真的吗？
[内容链接](https://www.zhihu.com/question/368765655/answer/1897219230836962205)

是的，我妹嫁到南方了，一年回不了河北一次，有两次甚至整年都没有回家。

我和我妹的沟通仅限于

“咱爹的的商业险到期了，一人800。转发给我就行，我给你发票。”

“村里的补助发了，一人400，咱三个的补助全交给咱爹的新农合和养老保险了。我给你截图。”

“咱爹体检费用1000，一人一半，给我就行。”

“咱爹老是喝散装白酒，这样对身体不行，咱们每个月分摊点，给他买稍微好点的粮食酒行不。半年一结。”

......

我是做销售的，做表格统计还是非常在行的，就把老父亲的支出罗列清晰，留存发票，然后半年和我妹妹报账一次。我妹妹也很痛快地分摊养老。

甚至我爹都责怪我：你赚钱多，你多负担点不行吗！一家人老这么算计。

我只能回复：爹你不懂，我只是一个销售，赚钱多是暂时的。我要培养我妹平均养老的习惯，不能让她觉得我养老是应该的，这样万一我不行了，爹还能去我妹那边养老。

（ps：我爹对待子女很公平，没给子女准备嫁妆和彩礼）

初次之外，任何沟通都没有。甚至我外甥出生了，我和我爹都没见过。

可能是人一老了话就多，我爹时常和我说：当我死了，你和你妹就是最亲的人了，你们要互相扶持。

这话听多了我也烦：爹，如果你死了，我妹还会回河北吗？

我爹嘴硬：为啥不回，你是她哥啊！

我：我说实话爹，你一死，保准就断亲了。估计每年就是互联网拜年。甚至我们的孩子一辈儿就不来往了。甚至孙子一辈儿都不知道有这么个亲戚。

我爹只能幽幽叹气
