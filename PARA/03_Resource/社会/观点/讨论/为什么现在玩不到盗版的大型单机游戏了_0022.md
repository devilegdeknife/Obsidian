# 为什么现在玩不到盗版的大型单机游戏了？
[内容链接](https://www.zhihu.com/question/499885119/answer/1985766792593946545)

[http://cs.rin.ru](https://link.zhihu.com/?target=http%3A//cs.rin.ru/)，你盗版资讯的第一选择。

各个组的打包者（repackers）都会把最近打包（repack）了什么的资讯放到上面。

顺着这个找各个repacker的网站，就可以几乎玩到各种破解的游戏了。

你甚至能找到很多老游戏。




看到好多评论表示不会使用[http://cs.rin.ru](https://link.zhihu.com/?target=http%3A//cs.rin.ru)，2025年12月26日特做此更新，教各位如何使用这个网站。
