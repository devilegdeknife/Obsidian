# 女生到底应不应该穿性感的衣服？
[内容链接](https://www.zhihu.com/question/608308600/answer/1961745054172426872)

该穿，多穿，趁早穿（未成年人不在此列）

穿的出去的、穿不出去的，都得试试，虽然不是每一种都适合，不合适就换，不敢出去就在家里穿，还得多拍照片多拍视频记录，只要场合合适就大胆的穿，别人爱说什么就说什么

折腾要趁早，不然以后你在想“为什么我年轻的不敢穿”的时候，就什么都晚了

35岁的身体就算补的再好，也不如22岁，错过了就是错过了

年纪大了许多事情就变了。就像小时候特别喜欢的玩具，20多岁即便随便买，也带不来小时候的感觉。
