# 有哪些关于南京的冷知识？
[内容链接](https://www.zhihu.com/question/296009140/answer/3404710514)

写一个面向历史爱好者和游客的吧

1.隋灭陈后 南京城被犁

六朝古都几乎没有留下任何六朝地面建筑

只剩一些墓葬石刻还在荒郊野岭孤独驻守着

比如陈霸先和刘裕的

东大老校区倒是有一颗六朝松 据说是梁武帝手植的 从树龄来看年代对得上

想来凭吊六朝金粉是没机会了 移步六朝博物馆吧

那里还能看到一点城墙遗址

2.打卡圣地牛首山旁边有一座南唐二帝陵

即李昪和李璟与各自妻子的陵墓

是目前发现的五代十国时期规模最大的帝陵

也是长江中下游已发掘的最古老的帝陵

更是南京唯一能进地宫的帝陵

没错 明孝陵只开放了地面的享殿 还是同治年间重修的

初始建筑在太平天国时期毁于战火了

但是我不推荐赶场子的游客花时间去看

因为真的没有啥可看的

除此之外

南京目前已经过考古发掘的帝陵有

东晋帝陵3处

分别位于南京大学北园（元、明或成帝）

南京汽轮机厂（穆帝）

富贵山（恭帝或安帝）

刘宋帝陵2处

分别位于郭家山（明帝）与西善桥宫山（孝武帝）

注意 孝武帝是刘宋第五位皇帝刘骏 刘裕是武帝

他的初宁陵位于江宁麒麟铺街道

目前只能看到一尊石刻

陈帝陵2处

分别是西善桥油坊村（宣帝）与灵山（疑似文帝）

大家比较熟知的陈武帝陈霸先的万安陵还没有进行考古挖掘

仅在江宁区万安北路附近存有神道遗址和一座石刻

萧齐帝陵不在南京 而在丹阳 目前也考古挖掘了三座

3.阅江楼是中国十大名楼 江南四大名楼之一

也有一篇阅江楼记

由明代大儒宋濂所作

听起来很唬人

但实际上 人家黄鹤楼岳阳楼滕王阁是代代重修

咱们这个从一开始就只有个地基

是本朝90年代才动工的

4.地铁五号线有一站叫神机营

所谓神机其实就是火铳

最早是制造机构 后来变为特种部队

另一个有意思的地名叫鱼雷营

这玩意是清朝正儿八经的水师驻军地

但到底是不是因为装配了我们理解中的那个鱼雷才叫这个名字的

我还真不知道

因为这地不对外开放

我知道的是这地方在抗日战争时期成为了一处乱葬坑

现在还有一座纪念碑 叫鱼雷营丛葬地纪念碑




先这样，上班了，有空继续

再更几个

5.莫愁湖是一方因长江改道冲击积水而形成的小池塘

形成时间约在六朝时期

明清时期 莫愁湖都是权贵私产园林

于是乎

在金钱和文人的推动下 莫愁烟雨一度成为金陵四十八景之首

但如今游览的朋友们可能要失望了

虽然免费 但也只有几个破亭子

光从景观上来说 跟普通公园没多大区别

因为太平天国内讧时期 莫愁湖精美的建筑和花木都毁于战火了

后来曾国藩虽然重新修缮 但也难复金陵第一名胜荣光了

当然 时间充裕的话还是可以走一走

里面有不少诗画作品

值得一提的是 莫愁湖南岸安葬着几十位牺牲于北伐战争中的粤军将士

并有追思亭与墓记

最后还有一件很离谱的事情

莫愁湖得名于莫愁女 是民间传说中楚国著名的音乐家

正史里也多有记载

代表作几乎无人不知无人不晓

《阳春白雪 》《下里巴人》

没错 这是两首歌

但这妹子其实是湖北[钟祥](https://link.zhihu.com/?target=https%3A//baike.baidu.com/item/%25E9%2592%259F%25E7%25A5%25A5/79148)人

宋人传抄时误把[钟祥](https://link.zhihu.com/?target=https%3A//baike.baidu.com/item/%25E9%2592%259F%25E7%25A5%25A5/79148)古称的石城和南京古称的石头城搞混了

所以她变成了南京人

6.中国现存最早的诗文总集《昭明文选》由南朝梁武帝的长子昭明太子萧统于南京编选

这部书的历史地位极高 是唐及后代文人的必读书籍

时至今日 文学院依旧是不读文选无以治学

昭明文选首次将经史子等思想学术文献与赋诗骚等文学作品进行了区别

改变了先秦以来“文学”与“非文学”之间模糊的界限和范畴

可以说 自此开始 中国文学走上了独立发展的道路

南京之所以成为世界文学之都 昭明文选功不可没

而昭明太子也因其卓越的学识 品德与文学领域的伟大贡献广为后世文人追念

南京目前有不少他的读书台和手植树景点供人凭吊

明孝陵景区也有一处被列为金陵四十八景的台想昭明

为清人所建

对了

江阴也有他种的红豆树
