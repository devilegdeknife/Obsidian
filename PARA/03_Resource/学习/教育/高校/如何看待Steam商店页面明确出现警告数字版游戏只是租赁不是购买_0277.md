# 如何看待Steam商店页面明确出现警告：数字版游戏只是租赁，不是购买？
[内容链接](https://www.zhihu.com/question/828901579/answer/6611602902)

**如果你今天反对盗版，那明天你手里的正版就是盗版**。

比较关注业界的玩家应该记得，之前日本厂商在国内搞过一波舆论宣传，“跨区的游戏，即使是正版也是盗版”。意思是你是中国用户，如果你买了日版的光盘，那就是盗版，不是正版。这个事情的背景是2014年游戏机禁令取消，日本游戏要大规模发行国行了，国行肯定要删改，为了避免玩家继续买港版游戏才搞了这么一波事。结果这次宣传才刚起了个头就被简中玩家问候先人，无疾而终了。

这还真不是日本人歧视中国，因为历史上所有日本游戏机都是锁区的，你根本就没有玩“盗版”的机会。任天堂一直锁到前几年。为啥现在不锁了？因为欧盟财政亏空需要小日本上缴罚款，吓得太君赶忙不锁区了。




法律问题其他答主都回答的很完整了，我这里纠正大家对实体游戏存在的一些误区。

首先，实体游戏，确实不会带有完全的权利，但是也绝对不像很多数字版鼓吹者说的那样，“跟数字版没区别”。

实体游戏，不仅允许玩家赠与和转卖，而且是允许玩家修改游戏内容的。比如90年代流行的金手指（相当于修改器），就可以大幅修改游戏的核心游戏性。很多人以为金手指只能锁血改数值，不是的哦，金手指是直接从机器码级别替换内存数据，理论上只要你的金手指容量够大（以现在的技术非常简单），你可以在任天堂FC游戏中实现MOD。当时的游戏机厂商对这种修改是不管的。

90年代末日本有一个著名的案件，某游戏出版社在杂志的光盘里附送了修改过的游戏存档，被Konami告上法庭。最后判决是你自己改了给朋友玩没问题，但是你拿出来卖不行，哪怕这个光盘本身没收钱是白送的也算商业行为。在此之前，玩家买来游戏卡改造然后再卖是不违法的，但是这以后就违法了。

至于现代游戏开头都有的“玩家不得XXXXX，点击即表示同意”，没有任何法律效力。它还可以在里面写你进入游戏就代表你同意把所有财产转移给游戏公司呢，骗小孩玩的。




其次，很多人说现在的游戏机都联网，你光有盘也玩不了。这就是根本没有游戏机的PC婆罗门瞎栽赃了。

到目前位置，不提XSS和F1这种奇葩，**所有主流游戏机，它的单机游戏必须能够在永远不联网的情况下用实体盘运行，而且不是一般的运行，最低限度要能打通关。**

你们以为游戏机厂商凭什么收30%的平台税？它要负责的啊！游戏机游戏都要求提前一个月到3个月交付最终版本，硬件商的团队会进行品质测试，不能在实机上通关的游戏要么你拿回去重做，要么你终止发售。哪怕是赛博朋克2077这样千年不遇的奇葩游戏，它的PS4版也是可以直接用光盘玩到结局的，只不过你要顶着黑梦玩罢了。这是任天堂1985年定下的“祖宗之法”，也是现代游戏不会发生ATARI大崩溃的原因，没人敢改。




我对盗版的态度一直都是，违法，但是我不反对盗版。很多玩家对我的态度感到不可思议，觉得这完全违背了玩家的基本道德。**我会有这种立场，是因为我真的从历史中吸取了教训——早年的玩家权利远比现在要多，随着游戏业逐步发展，厂家的权力越来越大，玩家的权利越来越少了**。

一个例子就是游戏业的世纪诉讼：中古诉讼。1997年开始，以索尼为代表的日本游戏厂商，起诉全日本的游戏零售店，要求禁止二手游戏交易。实际上在1997年以前，游戏厂商就尝试过限制二手游戏交易，反复沟通无果，最后对簿公堂。一审游戏店胜利，二审游戏厂商胜利，全日本开始禁止二手交易，你自己把游戏卡卖给朋友也违法。这下子日本ZF急了，直接开到高院，法官在判决书里把所有游戏公司骂了一顿，说以后不许你们再搞这种动摇国本的事了。

看到没有，连统治阶级都着急了，都急着保护玩家权利。我们的很多玩家，简直比俱乐部里的M还贱，厂商剥夺你的权利，你不仅不反抗，还说“理应如此”。你觉得理应如此是你自己的事，我们这些老玩家不觉得理应如此。

**玩家作为用户，力量天生就是不对等的，让所有玩家联合起来罢买游戏是一个非常困难的事情，盗版是相对最可行，成本最低的对抗方法**。

阀门之所以会说“正版的价值在于比盗版方便”，就是因为PC游戏在长达30年的时间里受盗版困扰，正版游戏无法存活。如果没有这种威胁，Steam是不会一统天下的，各大厂商还是会搞橘子这样的垃圾数字平台。

大家发现了没有？游戏史上想要扩张版权的，日本厂商占多数。很大一个原因就是日本游戏市场几乎不存在盗版，而日本玩家对版权的重视程度也绝对是世界第一。在美国找个没玩过盗版的人何其困难，但是绝大多数日本人真可以做到一辈子都不玩盗版。

日本玩家有多顺民呢？日本玩家拍游戏的画面，都会注意不要把图拍正了，因为自己没有二次发布游戏内容的权利。我第一次被日本网友提醒的时候被堵的说不出话，差点就跟王司徒一样吐血而亡了。后面我每次放图都有日本网友出于善意来提醒，这么好的人民才配得上日本游戏厂商啊。

所以还是那句话：**如果你今天反对盗版，那明天你手里的正版就是盗版**。
