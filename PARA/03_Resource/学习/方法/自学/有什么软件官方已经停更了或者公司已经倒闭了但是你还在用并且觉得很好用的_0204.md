# 有什么软件官方已经停更了或者公司已经倒闭了，但是你还在用并且觉得很好用的？
[内容链接](https://www.zhihu.com/question/571445355/answer/125771194094)

这就不得不提到来自O Deus Audio公司出品的ASIO Link Pro，开发者不幸离世了，随之而来公司也倒闭了，但他的侄子授权给第三方志愿开发者，提供了补丁能够使这款软件能够被免费使用。

![图片描述](https://picx.zhimg.com/v2-b8eefca89cae71d4ad44ccfc2381544d_r.jpg?source=2c26e567)

别看这软件界面粗糙，但是它CPU占用低到不可思议的同时，还能实现所有你能想象到的在Windows平台上的复杂音频路由功能。

游戏直播Up主？希望直播自己的游戏声音同时能独立控制音乐的音量？——ASIO Link Pro

音乐直播Up主？希望在副屏观看B站视频摸鱼的同时不希望声音被功放出来的同时还希望音频能被正确路由进宿主机架？——ASIO Link Pro

抽象派玩家？希望玩游戏的时候击杀敌人还能很抽象的给队友放击杀BGM的同时不把你游戏内的打击音效放出来只有BGM？——ASIO Link Pro

所有你能想象到在PC平台所需要的音频路由基本上少不了这个软件作为桥梁，因为PC平台糟糕的设计，所有的音频直播你基本绕不开WDM这个介质，那么它就是一个介于WDM与你直播平台之间的一个完美桥梁。

类似的软件还有Live Processor，但是是收费的而且仅就个人而言虽然界面更加华丽但是占用更多的同时并不直观。我会说truly engineering思想的人，会首先选择这个软件——ASIO Link Pro，Bro.....它甚至还有Audio Over Ip的功能。

总之，只要你有设计复杂音频路由通道的能力，确实，这很需要你首先有自主设计能力，它确实不适合小白同时并不愿意钻研的人——都不花钱了，你不能指望别人免费花时间手把手教你，你需要自己动手学习，那这款界面粗糙的软件就是你所需要实现功能的必备软件。

这有点符合簇拥Linux派的自由浪漫主义思想，My Machine can do the hell that I wanna do.
