# 如何评价台湾模拟考作文题目是《我的妈妈是代理孕母》？
[内容链接](https://www.zhihu.com/question/1948925409119957473/answer/1949180067121959390)

以前人家说中华五千年文明遗留下来的正统糟粕在台湾，我还觉得这话是不是说得有点极端了，现在看台湾还是太超前了。

我有时候真是觉得台湾人骨子里就对现实生活，尤其是底层人民的现实生活缺乏实感，有些台湾作家的文字偶尔就会给我一种无病呻吟伤春悲秋的感觉。本来看一眼题目火气已经够大了，再看一眼题干火气更大了，真是有点意林那一套，自由，浪漫，梦想，纯粹的自我感动。

出题人还挺能描摹的，“自愿去做代理孕母”从而“实现他人梦想”，这玩意要真是开放了那千千万万走投无路的底层女性都得“自愿”献出子宫。

记得高三做过一道阅读题，改编自柔石的《为奴隶的母亲》。

大致情节是一位皮贩，因生活贫苦又染上了烟酒赌，走投无路便自作主张将妻子以身价100元、为期三年的条件典给邻村一个五十多岁秀才，为其生子承续香火。

妻子在秀才家的三年，要忍受秀才的冷眼和侮辱，要承担秀才妻子的咒骂与发泄，既要给秀才生孩子，还要做秀才家的仆人干体力活，一到夜深，她就开始思念留在家中的孩子春宝。

后来皮贩妻子终于为秀才家生下一个孩子，她给孩子取名秋宝。稚嫩的秋宝常常能抚慰她的心灵，让她对春宝的思念和在秀才家遭受的苦难显得不那么煎熬。

秋宝两岁时，皮贩妻子结束了作为生育奴隶的身份，回到了家中。但她发现面黄肌瘦，衣衫褴褛的春宝已经认不得她了。而在继续着苦难与贫哀生活的过程中，她又无可抑制地思念着秋宝，这份牵挂几乎将她撕裂。

这篇阅读的背景就是旧社会流行于浙东一带的典妻习俗。

当时高三的我们，哪怕再神经大条，再天真无知，哪怕语文课睡得天昏地暗，做这题的时候也知道写些“吃人的旧社会”一类的话。

中国人对于传宗接代的可笑执念，底层人民不被当人的现实，女性被当做可随意典当买卖的性资源和生育资源。历史遗留下来的丑恶糟粕，早在将近百年前就被一批有思想有心气的革命青年用纸笔披露和撕碎。

然后你告诉我，2025年了，有人堂而皇之地对着一群与百年前那批青年年纪相仿的年轻人说：“你四十多岁的老妈要去当代理孕母了哦~这是为了帮助他人实现梦想哦~”

出题人还是先关心关心自己老妈吧。

二编:

我一直觉得大陆在以文学阅读为主的语文文化教育方面做得很不错的一点，就是让所谓“在蜜罐里出生的孩子们”接触到了一批相当有意义的近现代作品。

这批作品的最大特点，就是集中于社会变革，时代动荡时期的底层人民的生活。

最典型的当然就是鲁迅。哪怕你对文学一点不感兴趣，但只要你接受过完整的九年义务教育，就知道祥林嫂，阿Q和闰土。这些人物某种意义上几乎已经成为了不止一代中国人的文化基因，“在晚清至民国时期，许许多多的旧知识分子受封建科举制度毒害”这种说法太模糊，但一说“孔乙己”大家就瞬间明白了。

或许这就是文学的意义吧，比史实更直观，比数据更生动。借由文学的载体，许多珍贵的人格沉淀为文化。

除了课本的课文选择以外，教材中推荐的课外阅读书目，甚至试卷阅读题和作文题的选材也给学生接触这类作品提供了大量机会。除了鲁迅笔下耳熟能详的人物，还有萧红笔下的冯歪嘴子(如果没记错的话也是某年高考阅读题)，小团圆媳妇，老舍笔下的骆驼祥子，小福子(教育部推荐阅读书目)等等。

哪怕是新中国成立后的作品也不乏集中于底层人民的社会生活的，白鹿原，红高粱家族，许三观卖血记，平凡的世界……总之我想表达的就是，大陆人写的东西，大陆学生看的东西，是不缺乏扎根于民众的，注重于整体社会变迁和底层生态的。

但是台湾呢，我倒是不清楚那边语文教育如何，不过我看过一些台湾作家的作品。张晓风，简媜，林清玄，龙应台，邱津妙，好是好的，词藻情思非常细腻华丽，我高中就靠背张晓风作文提的分。但就是……相比起大陆作品，怎么总感觉有点小资……

就是缺那么一股土腥味和麦香味，就是少那种大河儿女骨子里的壮阔和关怀，没有长河落日的寂寥无垠，没有东北平原的砭骨寒意，也没有黄土坡河涛轰鸣的高亢锐意，但也不是江南烟柳的那份明丽婉转。

我读台湾作家的作品，偶尔感觉人读得有点丧失人形了就跑回去看看张承志的《北方的河》，还得是要大陆北方作家这份燃烧的生命力才能对冲掉那股莫名其妙的阴丽哀艳感。

所以这个作文题目一出来就让我很难绷，看起来台湾教育确实不注重扎根于民众，这种荒唐的议题都能明晃晃地写在高三考卷上。所以说能被假自由民主假大空那一套忽悠的，有一个算一个都是脱离了群众的。但凡去农村里找个老阿妈聊两句呢，你要说社恐那也简单，去医院转转，那是最能看见众生百态的地方。
