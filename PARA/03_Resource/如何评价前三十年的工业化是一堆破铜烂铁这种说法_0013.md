# 如何评价「前三十年的工业化是一堆破铜烂铁」这种说法？
[内容链接](https://www.zhihu.com/question/51700891/answer/1996712950090711360)

前三十年的工业是不是一堆破铜烂铁我不清楚，但是以计经爱好者们最喜欢拿来哭丧的“两大神器(光刻机和大飞机)”中的“大飞机运-10”为例，就能看出一些端倪来：

![图片描述](https://picx.zhimg.com/v2-a97cfb8b86cc0bd467c12db5fd666bf5_r.jpg?source=2c26e567)

运-10是作为一种旨在提供跨洋飞行任务的远程专机于1970年8月上马的(故称“708工程”)↘

![图片描述](https://picx.zhimg.com/v2-796afa893aabe640612d8fc016d2fb85_r.jpg?source=2c26e567)

最初该机的设计指标是一种最大(商载15吨)航程6000公里，飞远航程(5吨商载)8300公里的远程轰炸机轰-8(如下图所示)缩小尺寸魔改的远程专机(毕竟总师都是马凤山，但708为确保高亚音速巡航采用的三叉戟的机翼，和单纯用图-16机翼放大的轰-8还是有些不同)

![图片描述](https://picx.zhimg.com/v2-945a6cf2917447ecc4de5198c1a1c9d2_r.jpg?source=2c26e567)

但在测绘了巴基斯坦航空迫降的B707-320C残骸(仅起落架受损)后，上飞团队有了新的想法...

![图片描述](https://pica.zhimg.com/v2-338f0ffb04a1252146a3c480540b96aa_r.jpg?source=2c26e567)




![图片描述](https://picx.zhimg.com/v2-b2ca3475f9e924f1f84acd9287e7989a_r.jpg?source=2c26e567)




![图片描述](https://pic1.zhimg.com/v2-612b8b3a714efef6a4fae8ee03c77d59_r.jpg?source=2c26e567)

并在首架B707-320C客机交付CAAC(民航)的当月，在以“B720B”作为原准机的基础上，对708工程进行了指标大改(明确了最大起飞重量为110吨，将最大商载提高到了17吨，同时把15吨商载航程提高到了“不低于7000公里”且“飞远航程(5吨商载)8462公里”)，也就是后面看到的“B707风味外观飞机”——“运-10”

![图片描述](https://pica.zhimg.com/v2-b440a340723a8a8047bda8baa124b184_r.jpg?source=2c26e567)

> ↑B720是B707-120的缩短机身尺寸用于中短程航线的版本，由于是B707衍生出来的过渡型号的缘故，该系列客机的燃油经济性极差(采用四发布局用来执飞中短程航线燃油经济性能不差么)，因此很快就被B727所取代，最终720系列客机于1967年停产，在其整个生命周期里仅制造了150余架(作为对比，B727系列生产了超过1800架！)↓

![图片描述](https://picx.zhimg.com/v2-3660d63e8c465b48b8bbd0b31d5dbd78_r.jpg?source=2c26e567)

于是不可避免的...

![图片描述](https://pic1.zhimg.com/v2-5734698bed1286b56186de7c3607761c_r.jpg?source=2c26e567)

![图片描述](https://pica.zhimg.com/v2-c37fca9c31222f2ce7a86d1b5572694d_r.jpg?source=2c26e567)

![图片描述](https://picx.zhimg.com/v2-7b00722e4ee445a434d480cb3f7b52a2_r.jpg?source=2c26e567)

运-10最终在立项十年后成功首飞，并在1985年下马(同年我国的B707-320C也因不符合ICAO(国际民航组织)的噪音法规全部退出了客运航线，转飞货机)，这也是无数计经爱好者愤愤不平的，这种情况在1990年代末期达到了顶点(主要是当时争夺未来的ARJ客机“花落谁家”，上飞是使出了全力拼命为曾经夭折的运-10“造势”)，但由于信息传输不畅的缘故，在1999年时，以**《经济管理文摘》杂志的执行主编高粱先生撰文，“创造性”的将运-10和“两弹一星”关联在一起，并将运-10与同时期的A300比肩，引得无数人纷纷效仿↘**

> 最幽默的是，部分计经爱好者甚至得出了如下所示的奇葩结论↓

![图片描述](https://pic1.zhimg.com/v2-806ffd49dce8f0926fd0d8eb81efbc3d_r.jpg?source=2c26e567)

**且不说1949年首飞的初代彗星客机十倍于运-10的试飞时间中也没发生过任何事故**，但**问题是初代彗星客机在投入运营仅一年后就因为应力设计缺陷接连发生空中解体事故**(但这对主材料采用LC4(远不如初代彗星使用的7075-T6铝合金)铝合金的运-10来说，是超纲问题了，**仅就材料和安全性来说，运-10还不如1949年首飞的初代彗星客机呢，拿来对比50年代的客机属实是白日做梦**...

![图片描述](https://pic1.zhimg.com/v2-9682dbef0061336cf21c9d4c50fe6686_r.jpg?source=2c26e567)
