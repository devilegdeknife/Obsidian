# 如何看待「内卷恰恰是阶级尚未固化的明证」的观点？
[内容链接](https://www.zhihu.com/question/1961350493709238427/answer/1966112048975942186)

没毛病啊，梦幻西游早就发现了这个问题会让玩家流失，所以直接分级。所以它才能活20多年，如果让所有玩家强制去攀175，低等级玩家游玩内容过少就会直接导致弃坑。

如果一个游戏本来就肝，那么如果不分级当然会导致肝度氪度过大，所有玩家都在提升过程中就直接不玩了。

所以各种游戏都在探索如何留住玩家的方法，大致有以下思路，至于地球ol哪个服采取了哪个玩法，可以对号入座一下。
