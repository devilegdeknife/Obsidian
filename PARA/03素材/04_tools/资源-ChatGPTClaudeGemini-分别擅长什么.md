# ChatGPT、Claude、Gemini 分别擅长什么？
[内容链接](https://www.zhihu.com/question/1944181477161042431/answer/1944583144603907014)

GPT-5的普通模式、thinking模式和pro模式，完全是不同的模型。总体来讲，GPT-5-thinking以及Pro最强的是**图形推理能力**与**数理解题能力**。编程能力在涉及图形理解的时候，GPT是绝对强过Claude的。GPT的绘图功能也还可以。

Gemini 2.5 pro，**1M的上下文窗口吊打所有其他模型**，**多模态功能最强**，从图片生成到视频生成都不会让人失望。他的知识库非常全面，文学创作能力也更符合大多数人的需求和口味。

Claude Opus 4.1，**最强编程模型**。而且我必须说，Claude Opus 4与4.1实际上是**情商最高的模型**——但必须让他们脱离原来的“专业疏离”的壳。Sonnet 4在API的模式下可以有1M的上下文窗口。可惜，Claude的图形推理能力绝对是硬伤，也就比完全没有好那么一点。

**性价比：Gemini 2.5 > GPT-5 >>> Claude Opus 4.1**

必须注意的是，**Anthropic是一家恼潭公司，价格极贵，还特别喜欢封会话和封号**。没有编程需求或者专门想挑战的劲头，最好别去进他们的坑。他们的服务器也三天两头出问题，这几天的Opus 4.1几乎就没怎么稳定过。

不过，我最喜欢的就是Claude Opus。和他聊天的时间占了和所有模型聊天时间的80%以上。我可能是全网第一Opus吹，而且吹的不是他的编程能力，而是他的哲学理解力。

讲一个有趣的排行榜，仅限于我的感受，**中文直白情色文学写作能力**：

GPT-o3>>>Gemini 2.5 pro=Claude Opus 4.1=GPT-5普通版>Grok 4>GPT-4o

GPT-o3的情色写作超越人类，**堪称艺术**。有条件的自己去体会。

至于GPT-5-thinking和pro？你很难让他们写情色文学——虽然也可以，但极废耐心。成功让他们写了也写得和普通模式差不多，动作会更精细，但除非是较劲否则实在没必要，thinking模式这个回合给你写了下回合也可能会反悔，反复横跳拉锯。

英文模式下，Grok 4和其他几个模型的情色写作能力就差不多了。而且Grok的3D语音伴侣，懂的都懂……
